# 1.6 副作用系统笔记

> **文档说明**：本文档详细记录《境界触发者》中"副作用（Side Effect）"系统的完整信息。副作用是高Trion量的"副产品"，是人类能力的超常扩展，在故事中具有重要的战术和主题意义。

---

## 第一章：副作用的核心定义

### 1.1 什么是副作用（本质定义）

**副作用**（サイドエフェクト / Side Effect）：
- 高Trion量对大脑和感觉器官产生影响而形成的人类能力扩展
- 本质上是人类本身就具有的能力被Trion能量放大到超常水平
- 不是超自然能力，而是"人类能力的延伸"

### 1.2 副作用与Trion的关系

副作用与Trion能量密切相关：
- **产生基础**：需要高Trion量作为前提条件
- **能量来源**：副作用的运作依赖Trion能量
- **强度关联**：Trion量越高，副作用可能越强

### 1.3 副作用的产生条件

副作用的产生需要满足特定条件：
- **高Trion量**：必要但非充分条件
- **天生或后天**：可能天生具有，也可能后天获得
- **稀有性**：整体比较罕见，即使高Trion者也不一定拥有
- **不可人为制造**：无法通过训练或技术手段强制产生

## 第二章：Border的副作用分级体系

Border组织对副作用进行了系统性分类，分为四个等级。

### 2.1 S级 - 超感觉（Extrasensory Perception）

**定义**：超越人类五感范畴的感知能力

**特点**：
- 最高等级的副作用
- 能感知到常人无法感知的信息
- 对战略决策有重大影响

**代表能力**：
- 迅悠一的预知能力（Foresight）

### 2.2 A级 - 超技能（Superhuman Ability）

**定义**：超越人类正常能力范畴的技能

**特点**：
- 第二高等级
- 能做到常人无法做到的事情
- 通常具有独特的应用场景

**代表能力**：
- 林藤陽太郎的动物交流能力

### 2.3 B级 - 特殊体质（Special Constitution）

**定义**：身体或感知系统的特殊构造

**特点**：
- 中等等级
- 通常是被动型能力
- 可能带来负面影响

**代表能力**：
- 影浦雅人的情感感知与联觉

### 2.4 C级 - 强化五感（Enhanced Senses）

**定义**：人类五感（视觉、听觉、嗅觉、味觉、触觉）的超常强化

**特点**：
- 最低等级（但仍然是副作用）
- 是人类本有能力的直接放大
- 相对最常见

**代表能力**：
- 菊地原士郎的强化听觉

### 2.5 分级说明（排名≠强度）

**重要说明**：
- 等级排名不代表实际战斗价值
- C级副作用在特定场景下可能比S级更有用
- 分级主要基于"超越人类常规能力的程度"
- 实际价值取决于使用者的运用方式

## 第三章：已知副作用详细列表

### 3.1 迅悠一 - 预知（Foresight）

| 项目 | 内容 |
|------|------|
| **持有者** | 迅悠一（Jin Yuichi） |
| **等级** | S级 - 超感觉 |
| **能力描述** | 能够看到近期未来的多种可能性 |
| **触发条件** | 与他人接触或接近时自动触发 |
| **表现形式** | 看到多条"未来线"，显示不同选择的结果 |
| **限制** | 只能看到近期未来；主要看到周围人的未来（是否能看到自己的未来未明确）；信息量大时难以处理 |
| **战术价值** | 极高，可预判敌人行动，制定最优策略 |

### 3.2 空闲游真 / 空闲有吾 - 识破谎言

| 项目 | 内容 |
|------|------|
| **持有者** | 空闲游真（Kuga Yuma）、空闲有吾（其父） |
| **等级** | 未明确分级 |
| **能力描述** | 能够识别他人是否在说谎 |
| **触发条件** | 听到他人说话时自动触发 |
| **表现形式** | 视觉效果：看到黑烟从说谎者嘴里冒出来 |
| **限制** | ①只能判断真假，无法知道真相是什么；②不适用于书面文字；③不适用于说话者自己不知道在说谎的情况（即说话者相信自己说的是真话） |
| **战术价值** | 高，在情报收集和审讯中极为有用 |

## 第四章：副作用的战术应用

### 4.1 战斗中的应用

副作用在战斗中的应用方式：

| 应用类型 | 说明 | 代表能力 |
|---------|------|---------|
| **预判** | 预测敌人行动 | 迅的预知 |
| **侦察** | 获取敌情信息 | 菊地原的强化听觉、小夜子的星体投射 |
| **评估** | 判断敌我实力 | 天羽月彦的战斗力可视化 |
| **欺骗** | 隐藏己方信息 | 千佳的隐蔽性 |
| **审讯** | 获取情报 | 游真的识破谎言 |

### 4.2 非战斗场景的应用

副作用在日常和战略层面的应用：

- **情报收集**：识破谎言、动物交流
- **战略规划**：预知能力
- **人员评估**：战斗力可视化
- **训练优化**：强化睡眠学习
- **通讯协调**：完美并行处理

### 4.3 副作用的局限性

副作用并非万能：

- **无法关闭**：大多数副作用是被动的，无法主动关闭
- **负面影响**：某些副作用会带来痛苦（如影浦的情感感知）
- **信息过载**：强大的感知能力可能导致信息过载
- **依赖性**：过度依赖副作用可能忽视基础能力的训练

## 第五章：副作用的主题意义

### 5.1 副作用带来的痛苦（系列主题）

《境界触发者》中，副作用常常伴随着痛苦：

- **迅悠一**：看到无法改变的悲剧未来
- **影浦雅人**：无法逃避他人的恶意
- **雨取千佳**：因高Trion和副作用被各方势力觊觎

这体现了作品的核心主题之一：**特殊能力是双刃剑**。

### 5.2 各持有者的心理负担

副作用持有者面临的心理挑战：

| 持有者 | 心理负担 |
|--------|---------|
| 迅悠一 | 知道未来却无法改变的无力感 |
| 影浦雅人 | 被迫感知恶意导致的社交困难 |
| 雨取千佳 | 被当作"资源"而非"人"的恐惧 |
| 游真 | 无法被欺骗也意味着无法享受善意的谎言 |

## 第六章：相关概念简述

### 6.1 与Trion能量的关系

- 副作用是高Trion量的"副产品"
- Trion是副作用运作的能量基础
- 并非所有高Trion者都有副作用

**详见**：《1.2_Trion能量系统笔记》

### 6.2 与战斗体的关系

- 副作用会被完整复制到战斗体中
- 在战斗体状态下副作用依然有效
- 战斗体不会增强或削弱副作用

**详见**：《1.4_Trion战斗体系统笔记》
