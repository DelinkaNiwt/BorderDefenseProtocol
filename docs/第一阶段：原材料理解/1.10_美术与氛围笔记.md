# 1.10 《境界触发者》美术与氛围笔记

## 第一章：整体视觉风格定位

### 1.1 核心美术风格

**现代科幻 + 军事组织风格**：
- BORDER作为军事防卫组织，视觉设计强调功能性和制度化
- 科技感与现实感的平衡：既有未来科技元素，又保持现代都市背景
- 日系动漫风格特征：清晰的线条、明确的色彩区分、夸张的特效表现

**写实与科幻的平衡**：
- 战斗场景发生在现代城市环境中
- Trion能量和触发器带来科幻元素
- 角色外观保持相对写实的比例和服饰

### 1.2 视觉设计理念

**功能性优先**：
- 军事装备的实用性设计（触发体、制服、武器）
- 清晰的视觉识别系统（派系、等级、功能）
- 避免过度装饰，强调效率

**科技感表达**：
- 全息投影界面
- 能量场和光效
- 几何化的设计元素

**清晰的视觉识别**：
- 不同派系有明确的视觉特征
- 等级体系通过服饰和装备体现
- 功能通过颜色和形状快速识别

## 第二章：色彩体系

### 2.1 Trion能量的色彩表现

**基于1.4笔记提取的视觉描述**：

**标准Trion能量**：
- 主色调：绿色系
- 表现形式：发光效果、能量流动
- 视觉特征：半透明、流动感

**Trion泄漏状态**（战斗体受伤）：
- 颜色渐变：绿色 → 深灰色
- 视觉效果：气态Trion从伤口逸出
- 泄漏约1分钟后自动停止

**战斗体破坏效果**：
- 严重伤害：裂纹在表面扩散
- 达到临界点：爆炸成烟雾
- 最终状态：真身显现在原位置

**血管通道的视觉**（战斗体内部）：
- 5条血管状通道，呈十字形排列
- 中央通道颜色比周围四条更深
- 负责Trion能量在体内的流动和分配

### 3.3 Trion战斗体的视觉特征

**基于1.4笔记提取**：

**与真身的视觉区别**：
- 默认外观基于真身
- 可在一定范围内定制（发型、服装等）
- 极端改变（身高、体型大幅变化）会导致控制困难

**受损表现**：
- 伤口：气态Trion泄漏，无血液
- 裂纹：从伤口处开始扩散
- 严重伤害：裂纹覆盖全身

**解除效果**：
- 裂纹扩散至全身
- 爆炸成烟雾
- 真身在原位置显现

**特殊案例 - 游真的永久战斗体**：
- 外观变化：头发从黑色变为白色，眼睛变为红橙色
- 身体停止生长（保持11岁时的状态）
- 可自我修复（唯一能自我修复的战斗体）

### 3.4 Trion兵外观设计

**基于1.7和1.9笔记提取**：

**Rabbit（兔型高级捕获型）**：
- 高度：约3米
- 体型：双足行走，宽肩膀，长臂三指无拇指
- 腿部：短粗腿，两趾脚
- 头部：类似兔子
- 配色：白色身体配黑色条纹
- 眼睛：绿色，位于黑色口腔内

**Ilgar（飞鱼状轰炸型）**：
- 体型：小学校大小的巨型Trion兵
- 形态：飞鱼状，细长身体配鱼鳍
- 头部：钳状结构
- 口部：类似人类的牙齿
- 武器：背部配备多门炮

**Marmod（战斗型）**：
- 特征：关节处装有刀刃
- 身体：为战斗优化的结构
- 弱点：眼部位于口中

**Bamster（捕获型）**：
- 体型：多层楼高的巨型四足生物
- 外观：白色鳞片覆盖全身
- 头部：兔耳状结构

**Idra（人形战斗型）**：
- 外观：人形机械装甲设计
- 结构：明显的机械关节
- 武器：前臂可展开护盾，肩部和手背有刀刃伸出口

### 8.1 战斗氛围

**基于1.9笔记的"对抗体验"维度提取**：

**压迫感**：
- 来源：大量弹幕、被包围的感觉
- 视觉表现：Hyrein的动物弹群、Lamvanein的火力覆盖

- 来源：不知道敌人从哪里出现

## 第九章：RimWorld适配考虑

### 9.1 像素风格转换策略

**如何将动漫风格转换为RimWorld的像素风格**：
- 简化线条，保留核心轮廓
- 减少细节，强调识别性
- 使用有限的像素表现关键特征

**保留核心视觉特征的策略**：
- Trion能量的绿色发光效果
- 战斗体的裂纹破坏效果
- Trion兵的标志性外观（如Rabbit的白色+黑色条纹）

**简化与抽象的平衡**：
- 保留足够的细节以识别单位类型
- 简化到RimWorld的像素限制内
- 使用颜色和形状快速区分

### 9.2 色彩简化方案

**RimWorld色彩限制下的适配**：
- 使用有限的调色板
- 保持色彩的识别性
- 避免过于复杂的渐变

**关键色彩的保留**：
- Trion绿色：必须保留，作为核心识别色
- 派系识别色：BORDER、Aftokrator、Galopoula等
- 状态指示色：正常、受损、能量耗尽

### 9.3 图标设计原则

**清晰度优先**：
- 小尺寸下可识别
- 避免过于复杂的细节
