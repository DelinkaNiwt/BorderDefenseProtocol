# 1.7 Trion兵系统笔记

> **文档说明**：本文档专注于"Trion兵"这一核心概念的系统性定义和描述。与1.4 Trion战斗体系统笔记不同，本文档聚焦于作为武器的Trion兵，而非作为人类意识载体的战斗体。

---

## 第一章：Trion兵的核心定义

### 1.1 什么是Trion兵（本质定义）

**Trion兵**（日文：トリオン兵，英文：Trion Soldier）是由近界民（Neighbors）创造的武器，完全由Trion能量物质化构成的人工战斗单位。同时，Trion兵也是独立的Trion宿主实体——其体内的Trion既是构成材料也是运作能源。

**核心特征**：
- **本质**：武器，不是生物（但作为独立的Trion宿主实体，拥有自身的Trion能量池）
- **构成**：完全由Trion能量物质化而成
- **意识**：无意识的战斗单位（与战斗体的本质区别）
- **创造者**：近界民各国

**详细特征**：
- **能量驱动**：完全由Trion能量构成并由其驱动的无人机
- **主要用途**：
  - 侦察敌方领土和防御力量
  - 捕获高Trion容量的人类
  - 提取Trion腺体（用于近界民的能源需求）
- **设计多样性**：设计取决于部署国家的需求和环境，各国发展出不同的特化类型
- **大规模部署**：可同时部署数千单位，形成压倒性的数量优势

### 1.2 Trion兵与Trion战斗体的本质区别

| 对比项 | Trion兵 | Trion战斗体 |
|--------|---------|-------------|
| **本质** | 独立的Trion宿主实体（武器） | 人类意识的载体 |
| **意识** | 无意识 | 承载使用者意识 |
| **控制方式** | 预设程序或远程操控 | 使用者直接控制 |
| **创造目的** | 战斗/侦察/捕获等任务 | 保护使用者真身 |
| **自主性** | 有限（取决于类型） | 完全自主 |

**补充对比分析**：
- **行动方式**：Trion兵按预设程序或远程操控行动，战斗体由使用者直接控制
- **部署规模**：Trion兵可大规模部署（数千单位），战斗体一人一体
- **损失代价**：Trion兵被摧毁无人员损失，战斗体被摧毁需要Bail Out保护使用者
- **战术灵活性**：Trion兵行为模式相对固定，战斗体可根据情况灵活应变
- **资源消耗**：Trion兵消耗制造资源（制造时注入Trion作为自身能源池），战斗体消耗使用者的Trion能量

### 1.3 Trion兵在近界战争中的战略地位

**战略意义**：

1. **弥补人力不足**：
   - 触发器使用者数量有限，Trion兵可弥补战斗人员不足
   - 如Rhodokhroun无法派遣人类战斗员，完全依赖Trion兵作战

2. **降低人员风险**：
   - 可执行危险任务而不损失人员
   - 侦察、先锋攻击等高风险任务由Trion兵承担

3. **战术分散作用**：
   - 大规模部署可分散敌方注意力
   - 消耗敌方资源和Trion能量
   - 为人形邻居的关键任务创造机会

4. **情报收集**：
   - 侦察型Trion兵可评估敌方防御力量
   - 为后续入侵提供情报支持

## 第二章：Trion兵的分类体系

### 2.1 按功能分类

| 功能类型 | 说明 | 代表类型 |
|----------|------|----------|
| **侦察型** | 负责情报收集、设置门（Gate）、潜伏监视 | Rad, Takia (Dog) |
| **战斗型** | 直接战斗，近战或远程攻击 | Marmod, Batelier (Dog) |
| **捕获型** | 捕获目标，Trion腺体提取 | Bamster, Rabbit, Bander |
| **轰炸型** | 空中轰炸，自爆攻击 | Ilgar |
| **人形战斗型** | 群体战斗，可组合护盾，光束攻击 | Idra |
| **支援型** | 战场支援，护盾生成 | Manverd (Dog) |

**各功能类型详细说明**：

- **侦察型**：负责情报收集和战场准备，通常体型较小，具有隐蔽能力。Rad可设置门，Takia可使用隐形模式。
- **战斗型**：直接参与战斗的主力，具有近战或远程攻击能力。Marmod以刀刃近战为主，Batelier以头部刀刃攻击。
- **捕获型**：专门用于捕获目标，通常体型较大。Bamster捕获普通人，Rabbit捕获触发器使用者。
- **轰炸型**：空中打击单位，具有大范围杀伤能力。Ilgar可进行轰炸和自爆攻击。
- **人形战斗型**：具有人形外观的战斗单位，可执行复杂战术。Idra支持手动操控模式。
- **支援型**：为其他单位提供战场支援，如护盾生成。Manverd是典型代表。

### 2.2 按操控方式分类

| 操控类型 | 说明 | 特点 |
|----------|------|------|
| **自律型** | 预设程序自主行动 | 无需操控者，按预设行为模式行动 |
| **手动操控型** | 由操控者远程控制 | 需要操控者，可执行复杂战术 |

**操控方式详细机制**：

- **自律型**：
  - 预设程序决定行为模式
  - 无需实时操控，可大规模部署
  - 行为相对可预测，适合简单任务
  - 代表：Bamster、Marmod、Rad、Ilgar、Bander、Dog

- **手动操控型**：
  - 操控者远程控制，实时响应
  - 可执行复杂战术和精确操作
  - 需要操控者投入注意力
  - 代表：Idra（支持Manual Mode）

### 2.3 按来源国分类

| 国家 | 技术水平 | 主要Trion兵类型 |
|------|----------|-----------------|
| **Aftokrator** | 最先进 | Bamster, Rabbit, Marmod, Rad, Ilgar, Bander |
| **Galopoula** | 使用他国提供 | Marmod（来自Aftokrator）, Idra, Dog（来自Rhodokhroun） |
| **Rhodokhroun** | 中等 | Idra, Dog |
| **Chion** | 未知 | Rad, Ilgar |
| **Calvaria** | 未知 | Marmod |

**各国技术特点概述**：

- **Aftokrator**：技术最先进，拥有完整Trion兵体系，可整合黑触发器能力
- **Galopoula**：附属国，混合使用多国提供的Trion兵
- **Rhodokhroun**：专注于Idra和Dog开发，无法派遣人类战斗员
- **Chion**：使用Rad和Ilgar，技术水平未知
- **Calvaria**：使用Marmod，技术水平未知

## 第三章：已知Trion兵详解

### 3.1 Bamster（捕获型）

**基本信息**：
- **来源国**：Aftokrator
- **功能类型**：捕获型
- **操控方式**：自律型

**外观特征**：
- 多层楼高的巨型四足生物
- 白色鳞片覆盖全身
- 兔耳状头部结构
- 与Bander相似但颈部较短

**能力与特点**：
- 体型巨大
- 用于捕获普通人
- 坚韧皮肤提供良好防御
- 可在体内隐藏Rabbit，形成"特洛伊木马"战术

**弱点**：
- 整体战斗力不强，主要依靠体型优势
- 缺乏精确攻击能力
- 移动速度相对缓慢

### 3.2 Marmod（战斗型）

**基本信息**：
- **来源国**：Aftokrator / Galopoula / Calvaria
- **功能类型**：战斗型
- **操控方式**：自律型

**外观特征**：
- 关节处装有刀刃的战斗型Trion兵
- 身体结构为战斗优化设计
- 刀刃既用于移动也用于战斗

**能力与特点**：
- 近战刀刃攻击
- 拥有Trion兵中最硬的刀刃材质
- 刀刃用于移动和战斗双重目的
- 高效的近战战斗单位

**弱点**：
- 眼部是主要弱点（位于口中）
- 攻击眼部可造成致命伤害

### 3.3 Rad（侦察型）

**基本信息**：
- **来源国**：Aftokrator / Chion
- **功能类型**：侦察型
- **操控方式**：自律型

**外观特征**：
- 小型侦察单位
- 外观信息有限，通常隐藏在其他Trion兵体内
- 体型小巧便于潜伏

**能力与特点**：
- 设置门（Gate）
- 潜伏侦察
- 类似门设置装置，通过从附近行人处收集少量Trion来开启门
- 通常隐藏在其他Trion兵（如Bamster）体内
- 以数千为单位大规模部署

**弱点**：
- 单体极弱，几乎没有战斗能力
- 依赖数量优势和隐蔽性
- 被发现后容易被消灭

### 3.4 Ilgar（轰炸型）

**基本信息**：
- **来源国**：Aftokrator / Chion
- **功能类型**：轰炸型
- **操控方式**：自律型

**外观特征**：
- 小学校大小的飞鱼状巨型Trion兵
- 细长身体，配有鱼鳍
- 头部有钳状结构
- 口中有类似人类的牙齿

**能力与特点**：
- 空中轰炸
- 自爆攻击
- 从腹部释放炸弹进行轰炸
- 背部配备多门炮进行远程攻击
- 受重伤时进入自爆模式：装甲增强，冲向人群密集处

**弱点**：
- 自爆前有充能时间，可被狙击或炮击拦截
- 体型巨大，目标明显
- 自爆模式可被预判和规避

### 3.5 Rabbit（高级捕获型）

**基本信息**：
- **来源国**：Aftokrator
- **功能类型**：高级捕获型
- **操控方式**：自律型 / 可改造

**外观特征**：
- 约3米高，双足行走
- 宽肩膀，长臂三指无拇指
- 短粗腿，两趾脚
- 头部类似兔子
- 白色身体配黑色条纹
- 绿色眼睛位于黑色口腔内

**能力与特点**：
- 可将Trion体方块化（能力来自黑触发器Alektor）
- 存在多种变体版本
- 用于捕获触发器使用者
- 约3米高，双足行走
- 拥有高战斗力，可对抗多个B级小队

**变体类型**：
1. **Plain Form（基础型）**：物理力量最强，背部可伸出尖刺产生电流
2. **Ranbanein-Type（兰巴内因型）**：可从口中发射爆炸光束，配备喷气背包可短暂飞行
3. **Enedora-Type（艾内多拉型）**：手臂可液化渗入地下，弹出刀刃攻击
4. **Hyuse-Type（修斯型）**：从胸部发射磁性碎片固定对手
5. **Replica's Rabbit（雷普利卡型）**：由Replica制造，使用游真的黑触发器印记（Boost、Shield、Bolt）代替方块化能力

**弱点**：
- **眼睛是主要弱点**：位于口腔内的绿色眼睛，足够的伤害可摧毁整体
- 需要精确瞄准才能有效击败
- 单个A级特工无法击败，但可被多个B级小队联合对抗

### 3.6 Idra（人形战斗型）

**基本信息**：
- **来源国**：Rhodokhroun
- **功能类型**：人形战斗型
- **操控方式**：自律型 / 手动操控型（Manual Mode）

**外观特征**：
- 人形机械装甲设计
- 具有明显的机械关节结构
- 前臂可展开护盾
- 肩部和手背有刀刃伸出口

**能力与特点**：
- 群体战斗
- 可组合护盾（多个Idra可组合护盾增加耐久度）
- 支持手动操控模式（Manual Mode）
- 从手背和肩部伸出刀刃进行近战
- 从口中发射光束进行远程攻击
- 可在前臂上创建较小版本的护盾

**弱点**：
- Wiki未明确记载具体弱点
- 基于战斗表现推断：单体战斗力有限，依赖群体配合
- 手动操控模式需要操控者，操控者被干扰会影响战斗效率

### 3.7 Bander（捕获型/炮击型）

**基本信息**：
- **来源国**：Aftokrator
- **功能类型**：捕获型/炮击型
- **操控方式**：自律型

**外观特征**：
- 体型如"瘦身版Bamster"
- 四足生物，白色鳞片，兔耳
- 颈部比Bamster更长
- 存在彩色变体（在Calvaria vs. Spinthir战争中出现）

**能力与特点**：
- 专门捕获高Trion潜力目标
- 配备光束炮（Beam Cannon）进行炮击
- 可被触发器使用者骑乘
- 有时用来隐藏Rabbit
- 可撞击建筑物而不受损

**弱点**：
- 可被B级特工使用标准触发器击败
- 相比Rabbit战斗力较弱
- 体型大，目标明显

### 3.8 Dog（多功能型）

**基本信息**：
- **来源国**：Rhodokhroun（供应给Galopoula使用）
- **功能类型**：多功能型（含侦察、支援、战斗三种型号）
- **操控方式**：自律型

**外观特征**：
- 犬形Trion兵
- 三种型号名称均为犬种名的变位词（Takia=Akita秋田犬，Manverd=Doberman杜宾犬，Batelier=Retriever寻回犬）

**型号详解**：

#### 3.8.1 Takia（侦察型）
- 可使用"侦察模式"隐形躲避雷达
- 单体较弱，但群体作战时较为棘手
- 配备光束炮

#### 3.8.2 Manverd（支援型）
- 配备护盾生成器进行战斗支援
- 被称为"战场上的好伙伴"
- 配备光束炮，可群体作战

#### 3.8.3 Batelier（战斗型）
- 三种型号中战斗力最强
- 无法从眼部发射光束
- 头部呈刀状，用于近战

**战术应用**：
- B级战争篇中，Rhodokhroun向Galopoula提供Dog以弥补其触发器使用者不足
- Galopoula在入侵时分波次部署Dog，将Border的注意力引向外围

**弱点**：
- 单体较弱，群体作战时较为棘手
- 依赖数量和协调配合
- 各型号有不同的局限性（如Batelier无法发射光束）

### 3.9 其他已知类型

目前已知的主要Trion兵类型已在上述章节中详细描述。根据现有资料，各国可能还存在其他未公开的Trion兵类型，但Wiki中尚无详细记载。

**已知但信息有限的类型**：
- 各国可能拥有特化的侦察或支援型Trion兵
- Aftokrator作为技术最先进的国家，可能还有未展示的高级型号

## 第四章：Trion兵的制造与技术

### 4.1 制造原理（Trion物质化）

**基本原理**：
- Trion兵完全由Trion能量物质化构成
- 制造过程**通过制造设施（媒介）**将Trion能量转化为固态物质形态
- 功能越多，制造成本越高

> **重要说明**：Trion本身无法自行转化，需要媒介。制造设施是Trion兵的转化媒介，定义了Trion兵的形态、功能和存续方式。这与触发器作为战斗体的媒介是同一原理。

**制造策略**：
- **单体型**：
  - 标准型（如Marmod）：功能单一，成本较低，适合大规模部署
  - 高性能型（如Rabbit）：功能复杂，成本较高，战斗力强
- **团队战术型**：
  - 4-5单位联动设计
  - 功能较少但注重团队配合
  - 适合大规模部署

### 4.2 能量供给机制

**能量来源**：
- Trion兵完全由Trion能量构成并驱动
- 制造时注入的Trion能量决定其持续时间

**能量补充**：
- Rad可通过收集附近人类的少量Trion来补充能量
- 这种能量收集机制也用于开启门（Gate）
- 大多数Trion兵无法自主补充能量，依赖初始注入量

### 4.3 弱点与破坏方式

**通用弱点**：
- **主要弱点**：位于口中的单眼，足够伤害可摧毁整体
- 这是大多数Trion兵的共同弱点设计

**破坏方式**：
- 精确攻击眼部弱点可高效击败
- 较弱型号（如Rad、Dog）不需要精确瞄准弱点即可击败
- 高级型号（如Rabbit）需要精确攻击弱点或集中火力

**防御机制**：
- 部分Trion兵有坚韧皮肤或装甲保护
- Ilgar受重伤时会进入自爆模式，装甲增强

### 4.4 改造与定制（Rabbit变体案例）

Rabbit是Trion兵改造定制的典型案例，展示了如何基于基础型号进行能力扩展：

**基础型（Plain Form）**：
- 物理力量最强
- 背部尖刺产生电流
- 标准的方块化捕获能力

**兰巴内因型（Ranbanein-Type）**：
- 新增能力：口中发射爆炸光束
- 新增装备：喷气背包，可短暂飞行
- 增强了远程攻击和机动性

**艾内多拉型（Enedora-Type）**：
- 新增能力：手臂可液化渗入地下
- 攻击方式：从地下弹出刀刃攻击
- 增强了突袭和奇袭能力

**修斯型（Hyuse-Type）**：
- 新增能力：从胸部发射磁性碎片
- 功能：固定对手，限制移动
- 增强了控制能力

**雷普利卡型（Replica's Rabbit）**：
- 特殊来源：由Replica制造
- 能力替换：使用黑触发器印记（Boost、Shield、Bolt）代替方块化
- 展示了跨技术体系的整合可能性

## 第五章：Trion兵的操控机制

### 5.1 自律型的行为模式

**基本特征**：
- 按预设程序自主行动
- 无需操控者实时控制
- 行为模式相对简单可预测

**行为模式类型**：
- **巡逻模式**：在指定区域内巡逻侦察
- **攻击模式**：发现目标后自动攻击
- **捕获模式**：识别高Trion目标并执行捕获
- **自爆模式**：受重伤时的最终攻击（如Ilgar）

**优缺点**：
- 优点：无需操控者，可大规模部署
- 缺点：行为可预测，难以应对复杂战术情况

### 5.2 手动操控模式（Manual Mode）

**基本特征**：
- 由操控者远程控制
- 可执行复杂战术
- Idra是支持此模式的典型代表

**操控机制**：
- 操控者通过某种接口与Trion兵建立连接
- 可实时控制Trion兵的行动
- 允许更灵活的战术执行

**应用场景**：
- 需要精确控制的任务
- 复杂战术配合
- 特工伪装成Idra进行渗透

**局限性**：
- 需要操控者投入注意力
- 操控者被干扰会影响Trion兵表现
- 操控数量受限于操控者能力

### 5.3 群体协调与战术配合

**团队战术型设计**：
- 4-5单位联动设计
- 功能互补，协同作战
- 如Dog的三种型号（侦察、支援、战斗）配合

**协调机制**：
- **Idra组合护盾**：多个Idra可组合护盾增加耐久度
- **Dog分波次部署**：分散敌方注意力，消耗敌方资源
- **Bamster隐藏Rabbit**：形成"特洛伊木马"战术

**战术配合模式**：
- 侦察型先行评估敌方防御
- 战斗型作为第一波攻击力量
- 支援型提供护盾和火力支援
- 高级型号执行关键任务

## 第六章：各国Trion兵技术差异

### 6.1 Aftokrator的Trion兵体系

**技术特点**：
- 技术最先进
- 种类最多
- 拥有完整的Trion兵生态系统

**主要类型**：
- Bamster（捕获型）
- Rabbit（高级捕获型）
- Marmod（战斗型）
- Rad（侦察型）
- Ilgar（轰炸型）
- Bander（运输型）

**详细技术分析**：
- **黑触发器整合**：可将黑触发器能力整合到Trion兵中（如Alektor的方块化能力赋予Rabbit）
- **改造定制能力**：可根据不同触发器使用者的能力特点定制Rabbit变体
- **完整生态系统**：拥有从侦察到捕获到轰炸的完整Trion兵体系
- **大规模部署能力**：可同时部署数千单位进行入侵

### 6.2 Galopoula的Trion兵使用

**技术特点**：
- 使用Aftokrator提供的Trion兵
- 也使用来自Rhodokhroun的Idra

**详细分析**：
- **附属国地位**：作为Aftokrator的附属国，获得其Trion兵技术支持
- **混合使用**：同时使用来自Aftokrator的Marmod和来自Rhodokhroun的Idra、Dog
- **入侵部署**：Galopoula入侵时部署约200只Dog和95只Idra
- **战术灵活性**：通过混合使用不同来源的Trion兵，实现多样化战术

### 6.3 Rhodokhroun的Idra技术

**技术特点**：
- 专注于Idra的开发
- 提供给Galopoula使用

**详细分析**：
- **专业化发展**：专注于Idra和Dog的开发，形成技术专长
- **人力限制**：无法派遣人类战斗员，完全依赖Trion兵作战
- **技术输出**：向Galopoula等国提供Idra和Dog
- **战术依赖**：由于无法派遣人类，战术完全围绕Trion兵设计

### 6.4 其他国家

**Chion**：
- 已知使用类型：Rad、Ilgar
- 技术水平：未知
- 信息有限，仅知道使用这两种类型

**Calvaria**：
- 已知使用类型：Marmod
- 技术水平：未知
- 在Calvaria vs. Spinthir战争中出现彩色变体Bander

**其他国家**：
- 近界民各国可能都有自己的Trion兵技术
- 但目前Wiki中记载的信息有限
- 技术水平和类型多样性因国家而异

## 第七章：Trion兵的战术应用

### 7.1 大规模入侵战术

**Aftokrator大规模入侵战术（Large-Scale Invasion Arc）**：

**第一阶段：初始波次**
- 标准Trion兵从多点向外攻击
- 分散敌方防御力量
- 消耗敌方资源和Trion

**第二阶段：Rabbit引入**
- 单个A级特工无法击败Rabbit
- Rabbit可对抗多个B级小队
- 形成压倒性战斗力优势

**第三阶段：嵌套释放战术（特洛伊木马）**
- Rabbit从被击败的Bamster/Bander中涌出（预先隐藏在其体内）
- 防御方以为消灭了大型低威胁单位，实际释放了高性能Rabbit
- 利用防御方对大型Trion兵的认知惯性，形成延迟释放的战术欺骗

**情报利用**：
- 利用情报（如C级无法Bail Out）针对性捕获
- 优先捕获高Trion容量目标

### 7.2 潜伏与渗透战术

**Galopoula渗透战术（Galopoula Invasion）**：

**侦察阶段**：
- Dog先行侦察评估防御力量
- 收集敌方部署和战斗力信息
- 确定渗透路线和目标

**攻击阶段**：
- Idra作为第一波攻击力量
- 分散敌方注意力
- 为渗透创造机会

**渗透手段**：
- 特工伪装成Idra进行渗透
- 使用隧道触发器进行地下移动
- 多路分散敌方注意力

**部署规模**：
- 约200只Dog
- 约95只Idra
- 分波次部署，持续施压

### 7.3 与触发器使用者的配合

**角色分工**：
- **Trion兵**：作为前锋消耗敌方资源
- **人形邻居（触发器使用者）**：执行关键任务

**配合模式**：

**分散注意力**：
- Trion兵大规模攻击分散敌方注意力
- 触发器使用者趁机执行渗透或关键任务

**骑乘配合**：
- Bander可被触发器使用者骑乘
- 提供快速移动和火力支援

**协调攻击**：
- Trion兵从多方向攻击
- 触发器使用者从意想不到的方向突袭
- 形成立体攻击态势

**资源消耗战**：
- Trion兵持续消耗敌方Trion和战斗力
- 为触发器使用者创造有利战斗条件
