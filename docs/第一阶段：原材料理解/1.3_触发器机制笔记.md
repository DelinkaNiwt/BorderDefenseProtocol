# 1.3 《境界触发者》触发器机制笔记

## ⚠️ 术语警示

在《境界触发者》原作和网络讨论中，"触发器"一词被广泛混用——既可能指物理装置（触发体），也可能指功能模块（触发芯片），还可能指武器实体（Trion表现）。本文档采用三层架构术语体系进行严格区分：**触发体**（物理载体）、**触发芯片**（功能配置）、**Trion表现**（能量物化）。阅读时请注意上下文判断实际指代。详见[附录A：术语对照表](#附录a术语对照表)。

## 第一章：触发器的核心定义

### 1.1 什么是触发器（本质定义）

**触发器是Trion能量的物理接口与转化表现系统。**

这个定义包含三个核心维度：

1. **物理承载体**：作为能量接口的物理装置，形态多样（Border的手持握把、Aftokrator的手镯等）
2. **能量转化机制**：将使用者体内的Trion生物能释放并转化为可控能量体的过程
3. **功能配置系统**：根据战术需要动态调整的能力集合

### 1.2 触发器与Trion的关系

触发器是Trion能量从"体内潜能"转化为"外部战力"的**主要通道之一**：

```
Trion腺体（心脏附近）
  ↓ 产生Trion能量
转化媒介（触发器/制造设施等）
  ↓ 能量释放与转化
Trion战斗体/武器/Trion兵/建筑等
  ↓ 能量物化
各种Trion物质化表现
```

> **重要说明**：Trion本身无法自行转化，需要媒介。触发器是战斗场景下的主要媒介，但不是唯一的转化媒介。Trion兵、建筑设施等更可能由专门的制造设施输出。媒介定义了Trion的形态、功能和存续方式。

### 1.3 触发器存在的意义

1. **战斗能力转化**：将Trion潜能转化为实际战斗力
2. **能力标准化**：提供可复制、可训练、可量产的战斗系统
3. **安全保护**：通过战斗体机制保护真实身体
4. **战术灵活性**：通过配置组合实现多样化战术

## 第二章：触发器的分类体系

### 2.1 按稀有度分类

| 类型 | 特征 | 获取方式 |
|------|------|---------|
| **普通触发器** | 标准化、可量产、可复制、可控、可调 | Border开发室研发/近界技术引进 |
| **黑触发器** | 极限性能、独一无二、不可复制、不可调控 | 由创造者倾注所有生命力，结合强烈的愿望、情绪、意志创造 |

**普通触发器与黑触发器的本质区别**：
- **普通触发器**：用**技术手段**制造而成，可控、可调、可量产，性能取决于技术水平
- **黑触发器**：用**生命力和意志**创造而成，不可调控、无法复制量产，性能强力，能力极端。是技术掌控之外的特殊产物

黑触发器特点：
- 性能远超普通触发器
- 能力特性通常与制作者本人自身的能力特点相关
- 能力固定，无法更换配置，不可调控
- 每把都是独一无二的存在


### 2.2 按派系来源分类

| 派系 | 设计理念 | 典型特征 |
|------|---------|---------|
| **Border** | 标准化、模块化 | 8槽芯片系统、热插拔配置 |
| **近界各国** | 多样化、一体化 | 不同触发器形态功能各有不同。无芯片模块配置，能力固定不变的一体化设计 |

### 2.3 BORDER触发器

#### 2.3.1 按功能用途分类（Border官方分类）

Border将触发芯片分为三大功能类别：

| 类型 | 功能定位 | 代表芯片 |
|------|---------|---------|
| **攻击触发器** | 生成武器和直接攻击能力 | 弧月、蝎子、突击步枪、狙击枪 |
| **防御触发器** | 生成防护和逃脱能力 | 护盾、紧急脱离 |
| **选项触发器** | 辅助和战术支援 | 蜘蛛、变色龙、伪装信标 |

#### 2.3.2 按战斗职位分类（Border战斗员定位）

Border根据战斗员使用的触发器类型和战斗定位，将其分为以下职位：

| 职位 | 英文 | 战斗定位 | 主要触发器类型 |
|------|------|---------|---------------|
| **攻击手** | Attacker | 近战战斗员 | 弧月、蝎子、光魂等近战武器 |
| **枪手** | Gunner | 中距离战斗员 | 使用枪型触发器（手枪/步枪/霰弹枪等）发射子弹，操作简单（瞄准射击即可），射程比射手+20%，但弹药类型固定（最多预设2种） |
| **射手** | Shooter | 中距离战斗员 | 创建**Trion立方体**后发射子弹，可实时调整参数（威力/射程/速度）、分割立方体发射多发、实时组合弹种创建合成弹，操作复杂但灵活性极高 |
| **狙击手** | Sniper | 远距离战斗员 | 鹭、朱鹮、闪电等狙击武器 |
| **全能手** | All-Rounder | 多职位精通 | 攻击手+枪手/射手双精通 |
| **特殊工作兵** | Specialist | 陷阱/机关操作 | 变换箱、雷达操控等 |

**职位分类的意义**：
- **战术定位**：不同职位在团队中承担不同角色（前卫、火力支援、远程狙击等）
- **触发器配置**：职位决定了战斗员的主要触发器选择方向
- **团队配合**：合理的职位搭配是团队战斗力的基础

### 3.2 物理载体层（触发体）

#### 物理定义

**触发体（トリガーホルダー / Trigger Holder）是无刃柄状的物理硬件平台**。

日语原文验证：
> トリガー自体は小さいチップ状で、「トリガーホルダー」と呼ばれる刃の無い柄状の本体に左右で4種類ずつ、合計で8種類までセットされる

#### 核心功能

**1. 芯片槽位系统（8槽配置）**

| 侧面 | 槽位数 | 用途 | 切换方式 |
|------|--------|------|---------|
| 主触发（利手侧） | 4个 | 主要武器和攻击能力 | 战斗中实时切换 |
| 副触发（另一侧） | 4个 | 副武器、防御、辅助能力 | 战斗中实时切换 |

**2. 战斗体存储空间**

日语原文验证：
> トリオン体になっている時、実体の肉体はどこにあるんですか？トリガーホルダー内の、トリオン体が入っていたスペースに、入れ替わりで格納されています

激活机制：
- 激活前：Trion战斗体存储在触发体内
- 激活时：Trion战斗体释放，真实身体收容到触发体内
- 解除时：战斗体消散，真实身体释放

**3. 内置系统**

日语原文验证：
> トリガーホルダーには、それに加えて、基本装備として「緊急脱出(ベイルアウト)」、「戦闘体格納機能」、「通信回線」、「レーダー」がデフォルトで備わっている

| 功能 | 说明 |
|------|------|
| 紧急脱离 | Trion量过低或受致命伤时自动触发，传送真实身体到预设安全区（通常是小队作战室）。有效范围：三门市内保证有效，远征任务中距离母舰3km内有效。是所有触发器中**Trion消耗最大**的系统 |
| 战斗体收容 | 存储和释放Trion战斗体的空间 |
| 通讯 | 与本部和队友的实时通信 |
| 雷达 | 探测周边敌人和队友位置 |

> **术语说明**：包括紧急脱离（Bail Out）在内的内置系统，本质上依然是触发芯片的一种，但作为B级以上队员的默认内置标配，不占用8槽位，属于额外固定配置。

### 3.3 功能配置层（触发芯片）

**触发芯片（トリガーチップ / Trigger Chip）是小型芯片状的配置模块**，决定Trion能量以何种形式表现。

> 以下为各类别的**代表性示例**，完整触发器清单请参见术语词典。

#### 攻击触发器（代表性示例）

| 类型 | 代表芯片 | 特点 |
|------|---------|------|
| 近战武器 | 弧月 | 日本刀外观，攻防平衡，最通用的攻击手武器 |
| 远程子弹 | 小行星 | 基础直线子弹，枪手/射手的入门选择 |

#### 防御触发器（代表性示例）

| 芯片名 | 功能 |
|--------|------|
| 护盾 | 生成能量护盾阻挡攻击。可在用户25米范围内任意位置部署，可远程为队友施放。防御范围与强度成反比（越小越坚固）。**固定模式**：固定后防御力增强但无法移动。**Full Guard**：同时使用主副两侧护盾，防御最大化但无法反击 |

#### 选项触发器（代表性示例）

| 芯片名 | 功能 |
|--------|------|
| 蜘蛛 | 生成金属线作为陷阱或移动辅助，Trion消耗低，可设置大量丝线 |
| 蓑衣虫 | 穿戴式棕色斗篷，隐藏Trion信号使用户在**雷达上不可见**（非光学隐身），持续消耗Trion，是狙击手必备装备 |
| 变色龙 | **光学隐身**，使身体融入周围环境变得透明，但隐身期间无法使用其他触发器，且仍会出现在雷达上 |

### 3.4 能量表现层（Trion表现）

**Trion表现是芯片被激活后，Trion能量转化而成的具体武器、能力或效果。**

表现形式：
- **武器实体**：弧月剑、能量步枪、能量护盾等
- **子弹类型**：小行星（基础）、猎犬弹（追踪）、毒蛇（可控弹道）等
- **特殊效果**：能量丝线、光学迷彩、假人投影等

本质理解：
```
芯片 = 程序代码
Trion能量 = 运行环境
Trion表现 = 程序输出
```

## 第四章：触发器的工作原理

### 4.1 激活机制（Trigger On/Off）

**激活（Trigger On）**：
1. 使用者发出激活指令
2. Trion战斗体从触发体释放
3. 真实身体收容到触发体内（完全停滞状态）
4. 进入战斗状态

**解除（Trigger Off）**：
1. 主动解除或被动触发（能量耗尽/紧急脱离）
2. Trion战斗体消散
3. 真实身体从触发体释放
4. 身体回滚到收容时的状态（伤害不继承）

### 4.2 能量消耗机制

日语原文验证：
> トリガーチップやベイルアウトシステムなどに使用されるトリオン量を差し引いたトリオン量が実際に戦闘で使用できるトリオン量となる

**能量分配公式**：

```
总Trion能量
  ├── 系统基础消耗
  │   ├── 触发体系统（雷达、通讯、紧急脱离）
  │   └── 战斗体维持
  │
  ├── 触发芯片消耗
  │   ├── 芯片基础激活消耗
  │   └── 武器/能力维持消耗
  │
  └── 战斗可用能量
      ├── 武器使用（射击、挥砍）
      ├── 技能释放
      └── 受伤导致的额外流失
```

**消耗类型**：
- **持续消耗**：维持战斗体、维持武器实体
- **使用消耗**：每次射击、挥砍、技能释放
- **流失消耗**：受伤导致能量加速流失

### 4.3 切换机制

**芯片实时切换规则**：
- 两侧触发器可同时使用（左右各一个）
- 同侧最多使用一个，总计同时最多使用两个
- 切换时当前武器先消失，经过引导时间后新武器生成（存在空窗期）
- 切换速度取决于使用者熟练度

### 4.4 失效/解除机制

**主动解除**：使用者发出Trigger Off指令

**被动解除**：
1. **能量耗尽**：Trion消耗完毕，战斗体强制解除
2. **紧急脱离**：Trion量低于阈值或受到致命伤时自动触发（Border显示不同消息：战斗体限界/Trion限界/供给系统破坏/中继系统切断）
3. **供给器官破坏**：战斗体的Trion供给部位被破坏

## 第五章：触发器的限制与约束

### 5.1 能量限制

| 限制类型 | 说明 |
|---------|------|
| **Trion上限** | 由使用者的Trion腺体决定，无法超越 |
| **消耗累积** | 所有消耗从同一Trion池扣除，需要权衡分配 |
| **恢复限制** | 战斗中无法快速恢复Trion，需要休息 |

### 5.2 槽位限制

| 限制类型 | 说明 |
|---------|------|
| **槽位数量** | Border触发体最多8槽（左4右4） |
| **同时激活** | 同侧最多1个，两侧最多同时2个 |
| **配置固定** | 战斗中无法更换芯片，只能在已装载的芯片间切换 |

### 5.3 使用者限制

| 限制类型 | 说明 |
|---------|------|
| **Trion量要求** | 低Trion者可用能量有限，高消耗芯片不实用 |
| **熟练度要求** | 切换速度、组合技巧需要训练 |
| **身体适配** | 部分高级芯片需要特定条件或等级才能使用 |
| **定制权限** | A级以上才能申请定制芯片 |

### 5.4 物理/规则限制

| 限制类型 | 说明 |
|---------|------|
| **同侧互斥** | 同一侧的4个槽位同一时间只能激活1个 |
| **武器排斥** | 部分武器组合可能存在使用限制 |
| **战斗体依赖** | 所有触发器能力依赖战斗体存在，真身无法使用 |
| **通讯/雷达范围** | 内置系统有距离限制 |

## 第六章：不同派系触发器的特征

### 6.1 Border触发器（模块化、标准化）

**设计理念**：标准化、可量产、易于培训、灵活配置

**特征**：
- 手持握把形态，8槽芯片热插拔
- 统一的训练体系和评级系统
- 开发室持续研发新芯片
- B级以下使用标准配置，A级可定制

**优势**：
- 组织化程度高，便于团队协作
- 新人上手快，训练体系成熟
- 可根据情况调整配置

**劣势**：
- 单体性能有上限
- 依赖组织支持

### 6.2 近界触发器（一体化、个性化）

**设计理念**：个性化特制，发挥个体潜能。一体化设计，不可灵活调整配置。

**特征**：
- 形态多样（手镯、融合器等）
- 与使用者身体更紧密结合
- 功能固定但性能往往更强
- 配置方式各国不同

**代表性示例**：

| 触发器 | 所属国家 | 使用者 | 特点 |
|--------|---------|--------|------|
| 蝶之盾（Lampyris） | Aftokrator | 休斯 | 磁性碎片操控，可变形为护手/盾牌/枪械/翅膀等多种形态 |
| 雷之羽（Chelidon） | Aftokrator | 兰巴涅因 | 变形火器+喷气背包，重火力压制型，火力/射速/射程均超越Border触发器 |
| 处刑者（Vasilissa） | Galopoula | 嘉特林 | 机械臂力量型，展示不同国家的设计风格 |

> 注：近界民常植入**Trion接收器（角）**来增强Trion能力，但这是身体植入物，不是触发器本身。

### 6.3 黑触发器（极限性能、独一无二）

**特征**：
- 性能远超普通触发器
- 每把独一无二，无法复制
- 能力固定，不可更换配置
- 需要特定适格者才能使用
- 能力通常反映创造者本人的能力特点

**代表性示例**：

| 黑触发器 | 持有者 | 核心能力特点 |
|---------|--------|-------------|
| 游真的黑触发器 | 空闲游真 | 印系统，可学习并复制对手能力 |
| 风刃（Fūjin） | 迅悠一 | 远程斩击传导，通过物体表面传递攻击 |

> 完整黑触发器列表请参见术语词典。

### 常见混淆情况

| 说法示例 | 实际指代 | 严谨表达 |
|---------|---------|---------|
| "装备触发器" | 安装芯片到触发体 | 安装**触发芯片**到**触发体** |
| "使用弧月触发器" | 激活弧月芯片生成剑 | 激活**弧月芯片**生成**弧月剑** |
| "切换触发器" | 切换激活的芯片 | 切换**触发芯片** |
| "触发器被破坏" | 战斗体或武器受损 | **战斗体**/**武器实体**受损 |

### 判断方法

- 提到"装备"、"配置" → 通常指**触发芯片**
- 提到"握持"、"存储" → 通常指**触发体**
- 提到"挥砍"、"射击" → 通常指**Trion表现**（武器实体）
