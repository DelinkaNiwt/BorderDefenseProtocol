# 1.5 BORDER触发器介绍笔记

## ⚠️ 前置说明

本文档是1.3触发器机制笔记的**BORDER专题深化版**，详细展开BORDER的标准触发器配置体系。阅读本文档前，建议先阅读1.3笔记了解三层架构术语体系（触发体/触发芯片/Trion表现）。

**术语约定**：本文档中的"触发器"特指**触发芯片**，即装载到触发体中的功能配置模块。

---

## 第一章：BORDER触发器系统概述

### 1.1 BORDER触发器的设计理念

BORDER触发器系统遵循三大核心理念：

| 理念 | 说明 |
|------|------|
| **模块化** | 触发芯片独立设计，可自由组合搭配 |
| **标准化** | 统一规格，便于训练和维护 |
| **可量产** | 批量生产，降低成本，快速补充 |

### 1.2 8槽芯片系统回顾

BORDER触发体采用8槽芯片系统：

| 侧面 | 槽位数 | 用途 | 切换规则 |
|------|--------|------|----------|
| 主触发（Main） | 4个 | 主要武器和攻击能力 | 同侧同时仅能激活1个 |
| 副触发（Sub） | 4个 | 副武器、防御、辅助 | 两侧可同时各激活1个 |

**同时激活上限**：最多同时使用2个触发芯片（主副各1个）。

### 1.3 触发芯片的三大功能分类

| 类型 | 功能定位 | 代表芯片 |
|------|----------|----------|
| **攻击触发器** | 生成武器和直接攻击能力 | 弧月、蝎子、小行星、鹭 |
| **防御触发器** | 生成防护能力 | 护盾、护城盾 |
| **选项触发器** | 辅助和战术支援 | 蜘蛛、蓑衣虫、蚱蜢 |

### 1.4 触发器联动关系说明

#### 可选触发器（Optional Trigger）

可选触发器是提供辅助功能的触发芯片，分为两类：

| 类型 | 说明 | 示例 |
|------|------|------|
| **通用可选触发器** | 可独立使用，不依赖特定武器 | 蓑衣虫、蚱蜢、变色龙 |
| **专属可选触发器** | 必须与特定武器配合使用 | 旋空（弧月专属）、推进器（光魂专属） |

#### 原型触发器（Prototype Trigger）

原型触发器是处于测试阶段的触发芯片，通常由特定小队申请开发。特点：
- 性能强大但可能有限制或风险
- 未获HQ正式批准，使用者无法参加等级战
- 代表：韦驮天、传送器、全武装、连接器

本文档按以下结构组织BORDER触发芯片：

## 第二章：攻击触发器 - 近战武器系

### 2.1 近战武器的共同特征

近战武器（攻击手触发器）具有以下共同特点：

| 特征 | 说明 |
|------|------|
| **高DPS** | 所有触发器中输出最高 |
| **低Trion消耗** | 相比远程武器消耗更低 |
| **Trion依赖度低** | 高Trion容量对攻击手是次要因素 |
| **近距离作战** | 需要接近敌人才能发挥威力 |

### 2.2 弧月（Kogetsu）- 标准刀型武器

| 属性 | 评级 | 说明 |
|------|------|------|
| 攻击力 | A | 高威力斩击 |
| 耐久度 | A | 可承受多次碰撞 |
| 轻便性 | C | 相对较重 |

**基本特征**：
- 最受欢迎的全能刀型触发器
- 日本刀外观，攻防平衡
- 可变形为长矛型（Kogetsu: Spear）

**专属可选触发器**：

#### 旋空（Senkū）

| 特性 | 说明 |
|------|------|
| 功能 | 短暂延伸刀刃范围 |
| 标准激活 | 1秒激活 = 15米范围 |
| 生驹旋空 | 0.2秒激活 = 40米范围（需极快挥刀速度） |
| 威力 | 可切穿多个护城盾，斩断重装甲Rabbit |
| 要求 | 必须与弧月装载在同一侧 |

#### 幻踊（Gen'yō）

| 特性 | 说明 |
|------|------|
| 功能 | 自由操控刀刃形状 |
| 应用 | 攻击中途改变刀刃形态 |
| 效果 | 跟随对手闪避动作、绕过防御屏障 |
| 特点 | 变形可同步到对手难以察觉的程度 |

### 2.3 蝎子（Scorpion）- 变形刃

| 属性 | 评级 | 说明 |
|------|------|------|
| 攻击力 | A | 与弧月相当 |
| 耐久度 | D | 极低，首次碰撞即可能破裂 |
| 轻便性 | S | 几乎无重量 |

**核心能力**：
- **形状变化**：可自由改变刀刃形状和长度
- **任意部位生成**：可从身体任何部位生成刀刃
- **双持友好**：重量极轻，适合双持战斗

**代表技巧**：
- 鼹鼠爪（Mole Claw）：从地下突刺
- 腿刃（Leg Blade）：覆盖腿部的刀刃
- 分支刃（Branch Blade）：从多个位置生成刀刃
- 螳螂（Mantis）：双蝎子组合的中距离快攻

### 2.4 光魂（Raygust）- 重型刃+盾牌模式

**双模式系统**：

| 模式 | 攻击力 | 耐久度 | 说明 |
|------|--------|--------|------|
| 刀刃模式 | B | B | 大型双刃剑，可变形 |
| 盾牌模式 | E | SS | 高耐久防御，几乎无攻击力 |

**基本特征**：
- 三大攻击手触发器中最重、最不受欢迎
- 性能受使用者Trion量影响较大
- 由寺�的雷藏开发，用于对抗枪手子弹

**专属可选触发器 - 推进器（Thruster）**：

| 特性 | 说明 |
|------|------|
| 物理介质 | 内置于光魂本体中 |
| 原理 | 向相反方向喷射Trion产生推进力 |
| 外观 | 激活时光魂发光并留下光迹 |

**推进器用途**：
1. **强化斩击**：加速刀刃进行高速斩击，可切穿护城盾
2. **投掷攻击**：在盾牌模式下发射光魂作为投射武器
3. **机动辅助**：空中调整位置，快速接近或闪避
4. **近战推击**：短暂激活可击退对手

**限制**：推进器无法远程激活。

### 2.5 双月（Sōgetsu）- 斧型武器

| 属性 | 说明 |
|------|------|
| 外观 | 刀斧混合体，黄色弯曲刀刃 |
| 重量 | 轻量，可单手持握 |
| 耐久 | 可承受多次攻击不破裂 |
| 威力 | 可损伤Rabbit装甲 |

**特殊机制**：
- 可收鞘以激活其他触发器而不完全解除
- 可与**连接器（Connector）**配合，将两把双月合并为巨型战斧

> 注：双月目前仅有一名使用者，属于玉狛支部特色武器。

### 2.6 近战武器对比表

| 武器 | 攻击力 | 耐久度 | 轻便性 | 特点 | 使用者数 |
|------|--------|--------|--------|------|----------|
| 弧月 | A | A | C | 全能平衡，可配旋空/幻踊 | 最多 |
| 蝎子 | A | D | S | 变形奇袭，机动性极高 | 16人 |
| 光魂 | B | B/SS | D | 双模式切换，可配推进器 | 4人 |
| 双月 | A | A | B | 可合体为战斧 | 1人 |

## 第三章：攻击触发器 - 枪手系统

### 3.1 枪手系统的核心机制

枪手（Gunner）使用**枪型触发器**发射子弹：

| 特征 | 说明 |
|------|------|
| 操作方式 | 瞄准射击，操作简单 |
| 射程加成 | 比射手+20% |
| 弹药限制 | 最多预设2种弹药类型 |
| 战斗定位 | 中距离战斗，控制敌人位置 |

### 3.2 枪型触发器分类

| 枪型 | 特点 |
|------|------|
| 手枪（Handgun） | 轻便，单手操作 |
| 突击步枪（Assault Rifle） | 平衡型，最常用 |
| 霰弹枪（Shotgun） | 近距离散射 |
| 榴弹发射器（Grenade Launcher） | 曲射弹道，配合Meteor |
| 机枪（Minigun） | 高射速压制 |

### 3.3 基础子弹类型详解

枪手和射手共用以下4种基础子弹：

#### 小行星（Asteroid）- 基础直线弹

| 属性 | 说明 |
|------|------|
| 日文 | 通常弾（アステロイド） |
| 弹道 | 直线飞行，无特殊效果 |
| 威力 | 略高于其他子弹（无额外功能消耗） |
| 用途 | 集中火力、破盾 |
| 使用者 | 42人（最受欢迎） |

#### 猎犬（Hound）- 追踪弹

| 属性 | 说明 |
|------|------|
| 日文 | 追尾弾（ハウンド） |
| 弹道 | 自动追踪目标 |
| 追踪模式 | 探测引导（自动）/ 视觉引导（跟随视线） |
| 追踪强度 | 可调节0-100% |
| 威力 | 略低于小行星（Trion分配给追踪功能） |
| 使用者 | 26人（第二受欢迎） |

#### 流星（Meteor）- 炸裂弹

| 属性 | 说明 |
|------|------|
| 日文 | 炸裂弾（メテオラ） |
| 弹道 | 直线或曲射（榴弹发射器） |
| 效果 | 命中后爆炸，产生范围伤害和烟雾 |
| 爆炸范围 | 与Trion消耗成正比 |
| 威力 | 略低于小行星，缺乏穿透力 |
| 使用者 | 20人 |

#### 毒蛇（Viper）- 可控弹道

| 属性 | 说明 |
|------|------|
| 日文 | 変化弾（バイパー） |
| 弹道 | 可预设复杂飞行轨迹 |
| 操作难度 | 最高，仅3-4人能实时编程弹道 |
| 威力 | 略低于小行星 |
| 使用者 | 8人（最不受欢迎） |

### 3.4 枪手配置策略

- 最多装载**2种弹药类型**
- 可预先组合为**合成弹**（详见第四章）
- 弹药类型在战斗前确定，战斗中无法更改

### 3.5 枪手与射手的对比

| 对比项 | 枪手 | 射手 |
|--------|------|------|
| 操作方式 | 枪械瞄准射击 | Trion立方体操控 |
| 操作难度 | 简单 | 复杂 |
| 射程 | +20% | 基准 |
| 弹药灵活性 | 固定2种 | 实时调整 |
| 合成弹 | 仅能使用预设 | 可实时组合 |
| 参数调整 | 不可 | 可调威力/射程/速度 |

## 第四章：攻击触发器 - 射手系统

### 4.1 射手系统的核心机制

射手（Shooter）不使用枪械，而是创建**Trion立方体**后发射子弹：

| 特征 | 说明 |
|------|------|
| 操作方式 | 创建Trion立方体，操控发射 |
| 立方体大小 | 由使用者Trion能力决定 |
| 分割机制 | 可分割为多发子弹（更多=单发更弱） |
| 发射方式 | 可散射、延迟或整体发射 |

### 4.2 Trion立方体的创建与分割

**立方体机制**：
- 立方体大小由使用者Trion能力决定
- 可分割为多发子弹（更多子弹 = 单发威力更低）
- 可散射、延迟或整体发射

### 4.3 射手可用的子弹类型

射手可使用与枪手相同的4种基础子弹：
- 小行星（Asteroid）
- 猎犬（Hound）
- 流星（Meteor）
- 毒蛇（Viper）

### 4.4 射手的高级技术

#### 实时参数调整

射手可在战斗中实时调整子弹参数：
- **威力**：增加单发伤害
- **射程**：延长有效距离
- **速度**：提高弹速

#### 分割射击

将Trion立方体分割为多发子弹，实现多目标攻击。

#### 合成弹系统

射手可通过组合基础子弹创建**合成弹**（枪手无法实时组合）：

| 合成弹 | 组合 | 效果 |
|--------|------|------|
| 钻头弹（Gimlet） | 小行星+小行星 | 双倍规格，增强穿透力 |
| 眼镜蛇（Cobra） | 小行星+毒蛇 | 具体特性未明确 |
| 黄蜂弹（Hornet） | 猎犬+猎犬 | 更灵活精准的强力追踪 |
| 火蜥蜴（Salamander） | 猎犬+流星 | 带追踪能力的炸裂弹 |
| 战斧（Tomahawk） | 毒蛇+流星 | 可控弹道的炸裂弹 |

### 4.5 射手与枪手的详细对比

| 对比项 | 射手优势 | 枪手优势 |
|--------|----------|----------|
| 灵活性 | 可随意操控子弹属性 | - |
| 近距离 | 额外操控能力 | - |
| 残肢战斗 | 即使肢体残缺也能战斗 | - |
| 射程 | - | +20%射程加成 |
| 操作 | - | 简单易上手 |
| Trion需求 | 需要充足储备 | 相对较低 |

## 第五章：攻击触发器 - 狙击手系统

### 5.1 狙击手系统的核心机制

狙击手（Sniper）使用远程高威力武器：

| 特征 | 说明 |
|------|------|
| 战斗定位 | 远程战斗专家，位于小队最后方 |
| 威力特点 | 高威力，一枪可逆转战局 |
| 消耗特点 | 高Trion消耗 |
| 弱点 | 低连射速、弹道可预测 |

### 5.2 鹭（Egret）- 高威力标准型

| 属性 | 评级 | 说明 |
|------|------|------|
| 威力 | B | 可与攻击手触发器竞争 |
| 射程 | S | 所有狙击触发器中最远 |
| 弹速 | A | 高速弹道 |
| 射速 | C | 中等 |

**特点**：
- 最受欢迎的狙击步枪（17人使用）
- 平衡型，适合大多数任务
- 熟练者可命中1公里外目标
- 单发可破盾（除非Trion差距过大）

**特殊技巧**：双狙击（Twin Snipe）- 同时使用主副两把鹭射击

### 5.3 闪电（Lightning）- 速射型

| 属性 | 评级 | 说明 |
|------|------|------|
| 威力 | C | 较低，可被普通护盾阻挡 |
| 射程 | B | 中等 |
| 弹速 | SS | 所有狙击触发器中最快 |
| 射速 | B | 狙击触发器中最高 |
| 重量 | A | 轻便，可移动射击 |

**特点**：
- 11人使用
- 适合持续压制和削弱敌人
- 可在移动中射击
- 能见度差的环境中仍有效
- 高Trion使用者（如千佳）可配合铅弹使用

### 5.4 朱鹮（Ibis）- 最大威力型

| 属性 | 评级 | 说明 |
|------|------|------|
| 威力 | SS | 所有狙击触发器中最高 |
| 射程 | A | 最大不足1公里 |
| 弹速 | B | 较慢，相对容易躲避 |
| 射速 | D | 最低 |
| 重量 | D | 最重，难以移动射击 |

**特点**：
- 8人使用（最不受欢迎）
- 专为对抗大型Trion士兵设计
- 可穿透多面墙壁
- 通常需要脚架稳定
- 高Trion使用者可达到黑触发器级别输出

### 5.5 狙击手触发器对比表

| 武器 | 威力 | 射程 | 弹速 | 射速 | 重量 | 使用者数 |
|------|------|------|------|------|------|----------|
| 鹭 | B | S | A | C | B | 17人 |
| 闪电 | C | B | SS | B | A | 11人 |
| 朱鹮 | SS | A | B | D | D | 8人 |

## 第六章：防御触发器系统

### 6.1 防御触发器的核心机制

防御触发器用于生成防护能力，保护使用者和队友。

### 6.2 护盾（Shield）- 标准防御

| 属性 | 说明 |
|------|------|
| 日文 | シールド |
| 外观 | 透明绿色六边形屏障 |
| 部署范围 | 使用者25米内任意位置 |
| 远程施放 | 可为队友施放 |

**核心机制**：
- **面积与强度反比**：护盾越小越坚固
- 全尺寸护盾可挡多发枪手子弹
- 缩小到数英寸可挡鹭狙击或蝎子突刺

**特殊模式**：

| 模式 | 效果 | 限制 |
|------|------|------|
| 固定模式（Fixed Mode） | 耐久度增强 | 无法移动，需棱镜形状包裹使用者 |
| Full Guard | 同时使用主副护盾，防御最大化 | 无法反击 |

**特殊属性**：
- 可完全变形、分割为多个小屏障
- 可形成球形进行全方位防护
- **无法阻挡铅弹**（铅弹无直接破坏力）

**使用情况**：93+人使用，最受欢迎的触发器

### 6.3 护城盾（Escudo）- 大型防御墙

| 属性 | 说明 |
|------|------|
| 日文 | エスクード |
| 外观 | 城堡状墙壁结构，带BORDER标志 |
| 生成方式 | 从表面（地面、墙壁、Trion体）生长 |
| 激活方式 | 手或脚接触连接表面 |
| 有效范围 | 25米内（高Trion可扩展） |

**核心能力**：
- 耐久度远超护盾
- 可同时召唤多面墙壁形成路障
- 可作为陷阱将对手困在墙壁之间
- 可将使用者/队友弹射到空中
- **可阻挡铅弹**（将Trion转化为物质形态）

**限制**：
- 需要表面才能生长（无法在空中使用）
- 一旦定位无法移动或变形
- 不透明，阻挡视线
- Trion消耗大
- 被BORDER视为"老旧过时"

**使用情况**：4-5人使用，最不受欢迎的触发器之一

### 6.4 防御触发器对比表

| 触发器 | 耐久度 | 灵活性 | Trion消耗 | 特殊能力 | 使用者数 |
|--------|--------|--------|-----------|----------|----------|
| 护盾 | 中等（可调） | 高 | 低 | 远程施放、变形 | 93+ |
| 护城盾 | 极高 | 低 | 高 | 阻挡铅弹、弹射 | 4-5 |

## 第七章：选项触发器系统

### 7.1 选项触发器的定义和分类

选项触发器提供辅助功能，分为以下类别：

| 类别 | 功能 | 代表触发器 |
|------|------|------------|
| 陷阱类 | 设置陷阱和传送点 | 蜘蛛、变换箱 |
| 隐身类 | 隐藏自身 | 蓑衣虫、变色龙 |
| 机动类 | 增强移动能力 | 蚱蜢、推进器 |
| 攻击增强类 | 强化武器性能 | 旋空、幻踊、铅弹 |
| 侦察标记类 | 探测和标记敌人 | 星形标记 |

### 7.2 陷阱类

#### 蜘蛛（Spider）- 金属线陷阱系统

| 属性 | 说明 |
|------|------|
| 日文 | スパイダー |
| 外观 | 小型Trion立方体，两端带箭头状钩子 |
| Trion消耗 | 低，可设置大量丝线 |

**核心能力**：
- 创建绊线阻碍敌人移动
- 丝线颜色可自定义（配合操作员仅对队友可见）
- 灵活队友可利用丝线进行空中机动
- 使用者紧急脱离后丝线仍然存在

**限制**：
- 丝线容易被切断
- 敌人仔细观察可发现
- 易被范围攻击（流星、旋空）破坏

**使用情况**：5人使用

#### 变换箱（Switchbox）- 传送点系统

| 属性 | 说明 |
|------|------|
| 日文 | スイッチボックス |
| 外观 | 类似笔记本电脑/公文包 |
| 使用者 | 特殊工作兵（Trapper） |

**核心能力**：
- 在地图表面放置隐形标记
- 标记对应不同陷阱效果
- 可远程激活
- 陷阱效果可在放置后更改

**已知陷阱类型**：
- **短距传送（Short Warp）**：队友踩踏时在标记间传送
- **弹簧刃（Spring Blade）**：从地面弹出刀刃刺穿目标

**限制**：
- 标记需要表面，无法在空中放置
- 标记在雷达上显示为Trion读数
- Trion消耗高，操作复杂

**使用情况**：3人使用

### 7.3 隐身类

#### 蓑衣虫（Bagworm）- 雷达隐身

| 属性 | 说明 |
|------|------|
| 日文 | バッグワーム |
| 外观 | 棕色斗篷，左领口有天线划掉的徽章 |
| 功能 | 隐藏Trion信号，在雷达上不可见 |
| Trion消耗 | 持续消耗 |

**核心特点**：
- **非光学隐身**：仅隐藏雷达信号，肉眼仍可见
- 狙击手必备装备
- 可用于隐藏伤口、隐藏下一步攻击准备
- 脱下后斗篷仍存在，可作为诱饵

**限制**：
- 装备时无法即时激活Full Guard
- 无法绕过Galopoula的高级雷达系统

**使用情况**：89+人使用（第二受欢迎）

#### 变色龙（Chameleon）- 光学隐身

| 属性 | 说明 |
|------|------|
| 日文 | 隠密（カメレオン） |
| 功能 | 身体融入周围环境，逐渐变透明直至完全隐形 |
| Trion消耗 | 较高 |

**核心特点**：
- **光学隐身**：肉眼不可见
- 可用于潜入攻击位置或撤退

**重大限制**：
- **隐身期间无法使用其他触发器**
- 仍会出现在雷达上
- 队友也看不到使用者
- 快速对手可在使用者现身攻击时反击

**反制手段**：
- 猎犬（自动模式）可探测并攻击隐形目标
- Trion泄漏会暴露位置
- 菊地原的副作用使其几乎无效

**使用情况**：11人使用

### 7.4 机动类

#### 蚱蜢（Grasshopper）- 移动跳板

| 属性 | 说明 |
|------|------|
| 日文 | グラスホッパー |
| 外观 | 手中生成青色球体，召唤悬浮的方形光屏 |
| 功能 | 平台将使用者高速推向面对方向 |

**核心能力**：
- 空中位置控制，用于闪避和假动作
- 平台数量与输出成反比（类似射手子弹）
- 平台使用一次后消失
- 可为队友创建平台
- 可消除伤势或地形（雪、水）造成的机动障碍

**代表技巧**：
- **弹球（Pinball）**：在多个平台间快速弹跳同时攻击

**限制**：
- 无法反弹Trion子弹等Trion物体

**使用情况**：10人使用，主要是攻击手

#### 推进器（Thruster）- 喷射推进

详见第二章2.4节（光魂专属可选触发器）。

### 7.5 攻击增强类

#### 旋空（Senkū）- 弧月专属

详见第二章2.2节。

#### 幻踊（Gen'yō）- 弧月专属

详见第二章2.2节。

#### 铅弹（Lead Bullet）- 枪手/射手/狙击手专属

| 属性 | 说明 |
|------|------|
| 日文 | レッドバレット |
| 外观 | 激活后触发器和子弹变黑，重物呈六角棱柱状 |
| 功能 | 将Trion转化为重量，减速或固定目标 |

**核心机制**：
- **穿透纯Trion构造**：可穿过护盾（无直接破坏力）
- **被物质化触发器阻挡**：护城盾、弧月刀刃可阻挡

**可配合的触发器**：
- 枪手触发器（猎犬保持追踪、流星覆盖区域）
- 狙击触发器（需高Trion使用者，闪电效果最佳）

**限制**：
- Trion消耗大，降低射程和速度
- 需要近距离战斗技能
- 被视为专家级触发器
- 无法与蓑衣虫同时使用（射击暴露位置）

**使用情况**：3人使用（铅弹）

### 7.6 侦察与标记类

#### 星形标记（Starmaker）- 枪手触发器专属

| 属性 | 说明 |
|------|------|
| 日文 | スタアメーカー |
| 外观 | 茄紫色十字准星，中心白色圆圈 |
| 功能 | 子弹命中处放置标记 |

**核心能力**：
- 被标记目标即使使用蓑衣虫也会在雷达上显示
- 即使目标使用变色龙标记也保持可见
- 可透过Trion体看到标记，提供3D位置信息
- 连接操作员监视器，由操作员决定高亮哪些目标

**限制**：
- 必须在射击前装备到枪手触发器
- 作为通用可选触发器，阻止同列其他触发器激活
- 目标常在标记前被击败
- 标记肢体被切断后失效

**使用情况**：3人使用

## 第八章：标准装备系统

### 8.1 标准装备的定义

标准装备是**内置于触发体**的系统，**不占用8槽位**，属于B级以上队员的默认配置。

### 8.2 紧急脱离（Bail Out）

| 属性 | 说明 |
|------|------|
| 日文 | 緊急脱出（ベイルアウト） |
| 功能 | Trion战斗体被破坏时自动传送真实身体到预设坐标 |
| 外观 | 光束从烟雾中射出，留下光迹飞向目的地 |

**激活条件**（4种不同消息）：
1. 战斗体限界
2. Trion限界
3. 供给系统破坏
4. 中继系统切断

**传送机制**：
- 传送期间真实身体处于独立维度，免疫攻击
- 可手动触发进行自愿撤退

**范围限制**：
- 三门市内：保证有效
- 远征任务：距离母舰3km内有效

**Trion消耗**：所有触发器中**最高**

**目的地设置**：
- 大多数队员：小队作战室
- 支部成员：支部设施
- HQ内部：专用管道

**使用权限**：
- ✓ A/B级队员
- ✗ C级练习生（资源限制）
- ✗ S级队员（黑触发器不兼容）

### 8.3 雷达（Radar）

| 属性 | 说明 |
|------|------|
| 功能 | 探测周边敌人和队友位置 |
| 反制 | 可被蓑衣虫屏蔽 |

### 8.4 战斗体（Trion Body）

| 属性 | 说明 |
|------|------|
| 存储 | Trion战斗体存储在触发体内 |
| 激活 | 战斗体释放，真实身体收容到触发体内 |
| 解除 | 战斗体消散，真实身体释放 |

### 8.5 通讯系统

| 属性 | 说明 |
|------|------|
| 功能 | 与本部和队友的实时通信 |
| 范围 | 有距离限制 |

## 第九章：原型触发器与特殊触发器

### 9.1 原型触发器的定义

原型触发器是处于测试阶段的触发芯片：
- 通常由特定小队申请开发
- 性能强大但可能有限制或风险
- 未获HQ正式批准
- 使用者无法参加等级战（规则不同）

### 9.2 已知原型触发器

#### 韦驮天（Idaten）- 高速移动系统

| 属性 | 说明 |
|------|------|
| 日文 | 韋駄天 |
| 功能 | 沿预定轨迹以极快速度推进使用者 |
| 外观 | 激活时身体闪烁，留下光迹 |

**能力**：
- 速度快到大脑无法处理
- 可连续攻击多个敌人
- 高精度打击要害

**限制**：
- 轨迹预设后固定，无法自由移动
- 对手在轨迹上放置刀刃可反制
- 激活闪烁可能提醒警觉的敌人

#### 传送器（Teleporter）- 瞬间移动

| 属性 | 说明 |
|------|------|
| 日文 | テレポーター |
| 功能 | 瞬间传送到视线方向数十米内 |
| 外观 | 身体闪烁后消失，出发点留下短暂痕迹 |

**能力**：
- 速度几乎瞬时，可在子弹发射后躲避
- 即使被铅弹固定也能使用
- 短距传送（2-3米）冷却约0.5秒
- 长距传送（30+米）冷却数秒

**限制**：
- 需要视线，无法穿透障碍物或烟雾
- 传送轨迹可预测（沿视线方向直线）
- 有声音提示

**使用情况**：5人使用

#### 全武装（Full Arms）- 多触发器同时使用

| 属性 | 说明 |
|------|------|
| 日文 | フルアームズ |
| 功能 | 移除双触发器限制，可同时使用最多6个触发器 |
| 外观 | 背部生成两条机械臂和肩部武器 |
| 所属 | 玉狛支部专属 |

**能力**：
- 压倒性火力
- 可同时操作多种武器和防御

**限制**：
- 快速消耗Trion
- 未获HQ批准（原型状态）
- 需要特殊14槽触发体

**使用情况**：1人使用（木�的玲二）

#### 连接器（Connector）- 双月合体

| 属性 | 说明 |
|------|------|
| 日文 | コネクター |
| 功能 | 将两把双月合并为巨型战斧 |
| 所属 | 玉狛支部专属 |

**能力**：
- 大幅增加打击力
- 普通双月只能划伤Rabbit装甲，合体后可轻松切开

**限制**：
- Trion消耗大
- 不适合持久战
- 未获HQ批准

**使用情况**：1人使用（小南桐绘）

### 9.3 其他原型/特殊触发器

| 触发器 | 日文 | 功能 | 状态 |
|--------|------|------|------|
| 假人信标（Dummy Beacon） | ダミービーコン | 待补充 | 原型 |
| 盖斯特（Geist） | ガイスト | 待补充 | 原型 |
| 真虎（Makō） | マコー | 待补充 | 原型 |
| 计时器（Timer） | タイマー | 待补充 | 原型 |

### 9.4 原型触发器的获取和使用限制

- 需要向HQ申请开发
- 通常由特定小队根据战术需求提出
- 使用原型触发器的队员无法参加正式等级战
- 玉狛支部的原型触发器来自近界技术
