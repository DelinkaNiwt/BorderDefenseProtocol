# 1.4 Trion战斗体系统笔记

> **文档说明**：本文档专注于"Trion战斗体"这一核心概念的系统性定义和描述。与1.3触发器机制笔记不同，本文档聚焦于战斗体本身的特性和机制，而非触发器系统的整体架构。
>
> **概念区分**："Trion战斗体"（又称"Trion体"）是Trion物质化产物中的**特定形态**——专指承载人类意识的替代躯体。其他Trion物质化产物（如Trion兵、Trion构成的建筑设施等）虽然同样由Trion能量物质化而成，但**不属于**"战斗体"范畴。区分标准在于：战斗体承载使用者意识，是人类的替代躯体；其他产物是无意识的武器、工具或设施。

## 第一章：Trion战斗体的核心定义

### 1.1 什么是Trion战斗体（本质定义）

**Trion战斗体**（日文：トリオン体/戦闘体，英文：Trion Body/Combat Body）是由Trion能量物质化构成的人工躯体。战斗体是触发器激活后的基础产出，是所有Border队员进入战斗状态的必要载体。

**与真身的本质区别**：
- **真身**：使用者的原始肉体，在战斗体激活期间被收容在触发体内的异空间中
- **战斗体**：完全由Trion能量构成的替代躯体，用于战斗和任务执行

**意识转移机制**：
- 当触发器激活时，用户的意识会从真身转移到战斗体中
- 用户通过战斗体进行感知、思考和行动
- 真身在收容期间处于完全停滞状态

**副作用（Side Effect）跟随**：
- 副作用是与使用者意识绑定的能力，意识转移到战斗体中时副作用自然跟随
- 这意味着高Trion者的特殊感知能力在战斗体状态下依然有效

**战斗体存在的意义**：
- 提供可替换的战斗载体，保护使用者的真身安全
- 实现"无伤战斗"的战略目标——战斗体受损不影响真身

### 1.2 战斗体与Trion能量的关系

**Trion是战斗体的构成材料**：
- 战斗体完全由Trion能量物质化而成
- Trion能量是一种可以在能量态和物质态之间转换的特殊物质

**战斗体是Trion能量的物质化表现**：
- 战斗体代表了Trion能量的"固态"形式
- 受伤时，Trion会以气态形式从伤口泄漏

**内部结构**：
- 战斗体内部有**5条血管状通道**，呈十字形排列
- 中央通道的颜色比周围四条更深
- 这些通道负责Trion能量在体内的流动和分配

### 1.3 战斗体在触发器系统中的位置

**与触发器的关系**：
- 战斗体通过触发器激活创建
- 触发器是生成和控制战斗体的核心设备

**与触发体（Trigger Holder）的关系**：
- 触发体是装载触发芯片的物理硬件平台（无刃柄状）
- 激活时，触发体同时负责收容真身
- 真身被移动到触发体内的近界空间中保护

**8槽位系统**：
- Border的触发体采用8个槽位系统（4 Main + 4 Sub）
- 8槽位全部可用于装备武器和技能触发芯片
- 战斗体、紧急脱离、雷达、通讯等为不占槽位的内置标配功能

**分类定位**：
- 战斗体是触发器系统的基础产出，而非可选配置
- 这是所有Border队员的基础装备
- **参考**：详见《1.3_触发器机制笔记》

## 第二章：战斗体的生成机制

### 2.1 激活过程（Trigger On）

**激活方式**：
- 触摸触发体并明确表达激活意图
- 常用语音指令"Trigger On"（トリガーオン），但**语音指令并非必须**
- 关键是使用者的意图表达，而非特定词语

**激活时发生的事件**：
1. 使用者的真身被移动到近界空间保护
2. Trion能量开始物质化，构建战斗体
3. 使用者的意识转移到新生成的战斗体中
4. 战斗体完成构建，使用者获得控制权

**激活条件**：
- 必须持有触发体
- 必须有足够的Trion能量
- 使用者必须有明确的激活意图

### 2.2 真身的收容机制

**收容位置**：
- 真身被存储在触发体内部
- 具体位置是近界（Neighborhood）的一个独立空间

**收容期间的状态**：
- 真身在收容期间处于**完全停滞**状态
- 不会老化、不会受伤、不会消耗能量
- 时间对真身几乎没有影响

**收容的安全性**：
- 真身在异空间中完全安全
- 即使战斗体被完全摧毁，真身也不会受到任何伤害
- 这是战斗体系统的核心战略价值所在

### 2.3 战斗体的构筑过程

**能量消耗**：
- 构建战斗体需要消耗Trion能量
- 消耗量与战斗体的复杂度相关

**外观定制**：
- 可选择预设服装或当前服装
- 服装可设为可脱卸（但会消耗更多Trion）
- 可进行发型等外观修改

**构筑限制**：
- 极端的外观改变（如身高、体型大幅变化）会导致控制困难
- 建议保持与真身相近的体型以确保操控精度

**初始状态**：
- 新构建的战斗体处于完好状态
- 所有Trion通道完整，能量充足

## 第三章：战斗体的物理特性

### 3.1 基础物理属性

**对非Trion伤害的免疫**：
- 战斗体对非Trion攻击**几乎无敌**
- 坦克炮弹、导弹等常规武器无法造成有效伤害
- 甚至可以承受大气层再入的极端条件
- 只有Trion武器才能对战斗体造成实质伤害

**耐久度统一性**：
- **所有用户的战斗体耐久度相同**
- 耐久度与使用者的Trion容量**无关**
- 这意味着低Trion用户和高Trion用户的战斗体同样坚固

**缺陷修复功能**：
- 战斗体会自动修复真身的身体缺陷
- 例如：视力不良者在战斗体状态下拥有正常视力
- 其他身体残疾也可能在战斗体中得到补偿

**增强的物理能力**：
- 战斗体提供超越真身的物理能力
- 力量、速度、反应等都有所提升

### 3.2 外观与定制

**服装选项**：
- 可选择预设服装（Border制服等）
- 可选择保持当前服装外观
- 服装可设为可脱卸状态（消耗更多Trion）

**外观修改**：
- 可进行发型、发色等外观修改
- 可调整面部特征等细节

**外观限制**：
- **极端改变会导致控制困难**
- 大幅改变身高、体型会影响操控精度
- 建议保持与真身相近的基本体型

**与真身的关系**：
- 默认外观基于真身
- 可在一定范围内进行个性化定制

### 3.3 感知与体验

**疼痛感知**：
- 伤害只产生**轻微的疼痛反馈**
- 疼痛反馈可以被禁用
- 这使得战斗体用户可以在受伤后继续战斗

**体力系统**：
- 战斗体拥有**无限体力**
- 不会因为运动而感到身体疲劳
- **但是**：精神疲劳仍然存在
- **但是**：Trion消耗仍然是限制因素

**感知能力**：
- 视觉、听觉等基本感知正常运作
- 可能比真身更敏锐（取决于设定）

### 3.4 行动能力与限制

**部位依赖性**：
- 战斗体的行动依赖身体部位的完整性
- 手臂受损会影响武器使用
- 腿部受损会影响移动能力

**部位损伤的影响**：
- 损伤程度与功能丧失程度相关
- 完全切断的部位会导致该功能完全丧失
- 部分损伤会导致功能下降

**物理极限**：
- 虽然战斗体增强了物理能力，但仍有上限
- 极限取决于战斗体的设计和Trion供给

### 3.5 通讯与感知共享

**心灵感应通讯**：
- 战斗体用户之间可进行心灵感应通讯
- 需要操作员（Operator）协助建立连接
- 可跨越远距离，甚至穿过门（Gate）

**自动语言翻译**：
- 战斗体内置自动语言翻译功能
- 可与不同语言的使用者无障碍交流

**感知共享机制**：
- 操作员可共享战斗体用户的感知和视觉数据
- 感知可被共享、抑制或增强
- 操作员可上传视觉信息（建筑图、地图、弹道分析等）

**战术应用**：
- 团队成员可实时共享战场信息
- 操作员可为队员提供战术支援和情报分析

### 3.6 生存支持系统

**氧气系统**：
- 氧气消耗降低**数百倍**
- 可在低氧环境中长时间活动

**睡眠模式**：
- 在临界氧气水平时**自动激活**
- 可在太空等极端环境防止死亡
- 是一种紧急保护机制

**食物消化**：
- 消化效率接近**100%**
- 几乎无废物产生
- 大幅降低对食物的需求

**温度调节**：
- 温度感知可调节
- 可适应极端温度环境

**生理功能保留**：
- 大部分生理功能仍然保留
- 包括：哭泣、出汗、打哈欠等
- 这些功能可能是为了保持心理健康和人性化体验

## 第四章：战斗体的伤害机制

### 4.1 伤害的本质

**伤害表现形式**：
- 伤害导致**气态Trion泄漏**
- 无血液、无肉体伤口
- 伤口呈现为Trion能量的流失点

**内部结构**：
- 战斗体内部有**5条血管状通道**
- 呈十字形排列
- 中央通道颜色较深，可能是主要能量通道

**视觉表现**：
- 受伤时可见气态Trion从伤口逸出
- 严重伤害会导致裂纹在表面扩散

### 4.2 伤害与能量流失

**Trion泄漏机制**：
- 伤口导致Trion能量加速流失
- 伤口大小和数量影响流失速度
- 累积伤害会加速能量耗尽

**自动止血**：
- 伤口导致的Trion泄漏约**1分钟后会自动停止**
- 这是战斗体的内置保护机制
- 但已流失的Trion无法恢复

**警告系统**：
- Border有"**Critical Trion Loss**"警告系统
- 当Trion流失达到危险水平时会发出警报
- 提醒使用者注意能量状态

### 4.3 伤害不回传机制（核心）

**核心原理**：
- 战斗体伤害**完全不影响真身**
- 真身在异空间中完全安全
- 即使战斗体被完全摧毁，真身也毫发无损

**战略意义**：
- 这是战斗体系统的**核心战略价值**
- 实现了"无伤战斗"的目标
- 使用者可以进行高风险战斗而不担心生命安全

**与传统战斗的本质区别**：
- 传统战斗：伤害直接作用于肉体，可能致命
- 战斗体战斗：伤害只影响Trion构成的替代躯体
- 最坏情况：战斗体被摧毁，真身安全释放

### 4.4 战斗体的不可修复性

**无法自我修复**：
- 战斗体受伤后**无法自愈**
- 伤口会停止泄漏，但不会愈合
- 损失的部位不会再生

**修复方式**：
- 只能通过**解除并重新构筑**来"修复"
- 这意味着需要消耗Trion重建整个战斗体

**重建时间**：
- 重建时间与Trion容量成正比
- 低Trion用户（如三云修）需要**1-2小时**才能重建
- 高Trion用户重建速度更快

**破坏的视觉表现**：
- 严重伤害导致**裂纹在表面扩散**
- 达到临界点后**爆炸成烟雾**
- 真身随即出现在原位置

**战术影响**：
- 战斗中需要权衡继续战斗还是撤退重建
- 累积伤害会降低战斗效率

## 第五章：战斗体的能量供给

### 5.1 Trion供给系统的双核心结构

**Trion中继中心（伝達脳）**：
- 位置：**脑部**
- 功能：类似大脑，传输控制信号
- 作用：协调战斗体的所有动作和反应
- 被破坏后果：战斗体失去控制，立即失效

**Trion供给系统（供給機関）**：
- 位置：**心脏处**
- 功能：类似心脏，调节Trion流动
- 作用：为战斗体各部位提供能量
- 被破坏后果：能量供给中断，战斗体失效

**双核心的重要性**：
- 任一核心被破坏都会导致战斗体**立即失效**
- 这两个位置是战斗体的致命弱点
- 战斗中需要重点保护这两个区域

### 5.2 能量消耗模式

**维持消耗**：
- 战斗体存在的基础消耗
- 只要战斗体处于激活状态就会持续消耗
- 消耗量相对较低

**使用消耗**：
- 武器/技能使用的消耗
- 不同触发器消耗不同
- 高威力技能通常消耗更多

**流失消耗**：
- 受伤导致的额外消耗
- 伤口越多/越大，流失越快
- 约1分钟后自动停止流失

**理论极限**：
- 理论上：最小活动 + 只吃东西 + 无消耗/伤害 = 可无限维持
- 但实际战斗中很难达到这种理想状态

### 5.3 能量耗尽的后果

**Trion器官的限制**：
- Trion器官在战斗体状态下**无法恢复Trion**
- 只有解除战斗体后，真身的Trion器官才能开始恢复
- 这是战斗体系统的重要限制

**能量不足的表现**：
- 战斗体功能开始下降
- 可能无法使用高消耗技能
- 最终触发强制解除或紧急脱离

**重建时间**：
- 重建时间与Trion容量成正比
- 低Trion用户（如三云修）需要**1-2小时**
- 高Trion用户重建更快

**强制解除触发条件**：
- Trion完全耗尽
- 供给系统被破坏
- 中继系统被切断

## 第六章：战斗体的解除机制

### 6.1 主动解除（Trigger Off）

**解除方式**：
- 语音指令"Trigger Off"（トリガーオフ）
- **语音指令并非必须**，关键是使用者的意图
- 可以通过意念触发解除

**解除过程**：
- 战斗体"剥离"，开始消散
- 真身从异空间释放，出现在原位置
- 意识从战斗体转回真身

**战斗体消散与Trion返还**：
- 主动解除时战斗体消散，芯片占用的Trion解锁返还
- 使用者保留较多Trion储备，可较快重新激活

### 6.2 被动解除

**触发条件**：
- 能量耗尽导致的解除
- 供给系统（心脏处）被破坏
- 中继系统（脑部）被切断

**被动解除的表现**：
- 战斗体开始出现裂纹
- 裂纹扩散至全身
- 最终爆炸成烟雾，真身出现

**与主动解除的区别**：
- 被动解除通常是非预期的
- 可能发生在危险环境中
- 真身出现时可能面临危险

### 6.3 紧急脱离（Bail Out）

**紧急脱离的本质**：
- 本质是触发芯片的一种，但作为B级以上队员的内置标配
- 不占用8槽位，属于额外固定配置（与雷达、通讯并列）
- 是所有内置系统中**Trion消耗最高**的功能
- 在战斗体失效时自动激活，将真身传送到安全位置

**工作原理**：
- 检测到触发条件后自动激活
- 真身在传送过程中处于异空间，**无法被攻击**
- 传送完成后真身出现在指定安全位置

**自动触发条件（四种）**：
1. **战斗体极限超过**（Combat Body Limit Exceeded）：战斗体耐久度耗尽
2. **Trion极限超过**（Trion Limit Exceeded）：用户Trion耗尽
3. **供给系统被破坏**（Trion Supply System Destroyed）：供给系统受损
4. **中继系统被切断**（Relay System Severed）：中继系统被切断

**传送范围限制**：
- **三门市内**：保证有效，传送至小队战略室或分部设施
- **远征任务**：需在远征船**3km范围内**才能传送回船
- 超出范围则无法使用紧急脱离

**使用者限制**：
| 等级 | 是否有Bail Out | 原因 |
|------|---------------|------|
| A/B级队员 | ✓ 有 | 约130人拥有此功能 |
| C级训练生 | ✗ 无 | 资源不足 |
| S级队员 | ✗ 无 | 黑触发器无法修改 |
| 操作员 | ✗ 通常无 | Trion量低 |

**其他特性**：
- **Trion消耗**：触发器组中消耗最高的触发器
- **Rank Wars规则**：60米内无敌人时可主动触发，不给对方分数

### 6.4 解除后的状态回滚

**真身状态回滚**：
- 真身回滚到**收容时的状态**
- 战斗体期间的所有伤害完全消失
- 真身不会保留任何战斗体受到的损伤

**战斗体消散**：
- 无论哪种解除方式，战斗体都会消散
- 每次激活都是全新构筑

**重新激活的等待时间**：
- 取决于解除方式
- 主动解除后Trion储备较充足，等待较短
- 被动破裂后Trion接近枯竭，等待较长（低Trion者如三云修需1-2小时）

## 第七章：特殊情况与例外

### 7.1 永久Trion体（空�的游真案例）

**背景**：
- 空闲游真（Yūma Kuga）是近界民，父亲是空闲有吾（Yūgo Kuga）
- 游真在战斗中身受致命伤
- 父亲有吾用自己的生命创造了黑触发器，将游真濒死的肉体封存

**创造机制**：
- 黑触发器封存了游真濒死的真身
- 创造了一个永久性的Trion战斗体作为替代
- 这是用生命换来的特殊存在

**游真永久战斗体的独特特性**：

| 特性 | 说明 |
|------|------|
| **自我修复** | 唯一能自我修复的战斗体 |
| **耐久度** | 低于标准战斗体 |
| **Trion恢复** | 可在战斗体状态下恢复Trion（普通战斗体无法做到） |
| **睡眠需求** | 无需睡眠 |
| **触发器激活** | 可在战斗体状态下激活触发器（普通用户需先解除再激活） |
| **外观变化** | 头发从黑色变为白色，眼睛变为红橙色 |
| **生长停止** | 身体停止生长（保持11岁时的状态） |

**特殊性说明**：
- 这是极其罕见的特例
- 需要黑触发器和牺牲生命才能实现
- 不具有普遍复制性

### 7.2 近界民的战斗体使用

**基本原理相同**：
- 近界民同样需要触发器生成战斗体
- 战斗体的基本机制与地球人相同

**Trion接收器（触发角）**：
- 近界民可能拥有Trion接收器（也称触发角）
- Trion接收器可**大幅提升Trion容量和质量**
- 这使得近界民通常拥有更强的Trion能力

**技术差异**：
- 不同近界国家有不同的触发器技术
- 技术水平和特点各有不同
- 例如：Aftokrator的技术与Border有显著差异

### 7.3 黑触发器与战斗体

**特殊能力**：
- 黑触发器生成的战斗体拥有特殊能力
- 这些能力由创造黑触发器的人的意志决定
- 每个黑触发器都是独一无二的

**外观变化**：
- 使用黑触发器时，触发角会变黑
- 这是识别黑触发器使用者的标志

**与普通触发器的区别**：
- 黑触发器无法被修改或升级
- **无法安装Bail Out功能**
- 只有特定的人才能使用特定的黑触发器（兼容性问题）

**S级队员的限制**：
- S级队员使用黑触发器
- 因此没有紧急脱离功能
- 战斗风险相对更高

### 7.4 其他特殊战斗体

**Vorvoros黑触发器**：
- 使用者：Enedra（Aftokrator）
- **物理状态变化**：可改变物理状态（固态/液态/气态）
- **要害位置移动**：使用者可随意移动要害（供给系统/中继系统）的位置
- **高耐久外壳**：要害可用高耐久外壳保护
- **诱饵战术**：可创建多个诱饵外壳，迷惑敌人

**Galopoula版紧急脱离**：
- 使用从Aftokrator获得的信息开发
- **不同的传送方式**：在身体位置召唤门（Gate）而非光束传送
- 这表明不同国家对同一功能有不同的技术实现

- Trion量决定战斗体的基础属性
- Trion消耗与战斗体维持的关系
- Trion恢复与战斗体重新构筑的关系

- 副作用在战斗体状态下的表现

- 副作用与Trion量的关联

### 8.4 与Trion兵的区别

- Trion兵的身体也是Trion构成
- 战斗体与Trion兵的本质区别
- 控制方式和自主性的差异
- **详见**：《1.7_Trion兵系统笔记》
