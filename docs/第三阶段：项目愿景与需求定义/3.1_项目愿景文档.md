# 3.1 项目愿景文档

## 一、创作者视角：为什么做这个模组

### 1.1 创作动机

作为 RimWorld 和《境界触发者》的双重爱好者，我一直觉得 WT 的机制设定非常适合做成游戏内容——Trion能量的资源管理、触发器的可配置性、战斗体的安全机制——这些机制天然具有游戏性。这个模组首先是做给自己的个人作品，满足自己的创作梦，同时也是一个学习和成长的过程。

### 1.2 设计哲学：有选择性的机制移植

BDP 不是原作搬运，而是有选择性的机制移植。核心原则是"为游戏服务"：

- 有的机制会尽量完整还原
- 有的重视内在逻辑，外在表现可以视情况改变
- 有的重视外在表现，内在逻辑可以偏离原作
- 有的会完全舍弃不引入

这种取舍的判断标准始终是：它是否让 RimWorld 的游戏体验更好。

原作是动漫作品，而我们要做的是游戏中的模组。这不但是从影视到游戏的跨载体转换，还因产物是模组所以必须受到游戏自身系统框架的约束。100%完全还原既不可能也不是目标。

### 1.3 个人意义

这个项目不仅是一个模组，也是一个跨领域学习的过程（游戏设计、C#编程、RimWorld框架、项目管理等）。社区认可是加分项，但不是必须的。

## 二、玩家视角：这个模组带来什么

### 2.1 一句话定义

BDP 是一个将《境界触发者》的核心机制融入 RimWorld 的模组，为游戏引入 Trion 能量体系、可配置的触发器装备系统、战斗体状态切换机制，以及与近界民势力的对抗。

### 2.2 核心体验

- 管理 Trion 能量的产出与消耗，在日常运营和战斗之间做出资源权衡
- 为殖民者配置触发器（选择触发体 + 搭配触发芯片），打造个性化的战斗风格
- 激活战斗体进入战斗——伤害不回传真身，可以更大胆地战斗，但 Trion 耗尽或弱点被击中则战斗体破裂
- 面对通过"门"入侵的近界民派系和 Trion 兵的威胁

### 2.3 与原版 RimWorld 的关系

BDP 不替代原版的战斗和资源系统，而是在其之上叠加一层新的机制。殖民者仍然可以使用原版武器和护甲，Trion 系统是额外的能力维度。模组通过 RimWorld 原生框架（Comp、Hediff、Gene、Def 等）实现，与原版系统自然融合。

## 三、核心机制概览

### 核心机制（ABCD）

- **A. Trion 能量系统** — 整个模组的基础资源。殖民者通过 Trion 腺体产生 Trion 能量，触发器运作、战斗体维持、设施运转都消耗 Trion。Trion 接收器（触发角）可以增强 Trion 的量和质。这是所有其他机制的共同依赖。

- **B. 触发器三层架构** — 模组的装备系统。触发体是物理硬件平台，触发芯片是可替换的功能模块，Trion 表现是芯片激活后的具体输出效果。三层分离使得同一个触发体可以通过更换芯片实现不同功能。

- **C. 战斗体机制** — 模组的战斗核心。激活触发器后，殖民者进入战斗体状态——由 Trion 构成的替代躯体。战斗体受到的伤害不回传真身，战斗体破裂时紧急脱离将真身传送到安全位置。这让战斗变得更大胆，但 Trion 耗尽或弱点（头/心脏）被击中会导致战斗体失效。

- **D. 触发器可配置性** — 模组的自由度来源。玩家可以为殖民者选择不同的触发体，搭配不同的触发芯片组合，创造个性化的战斗风格。

### 支持系统（EF）

- **E. Trion 兵系统** — 由 Trion 构成的无意识人工战斗单位。原作中主要是敌对势力的战力，但在游戏中，敌方和玩家都可以拥有 Trion 兵，类似 RimWorld 的机械体。与战斗体的本质区别：战斗体是意识载体，Trion 兵是无意识武器。

- **F. 派系对抗** — 世界框架。BORDER（地球防卫组织，中立派系）与敌对惑星国家派系（近界民 + Trion 兵）的冲突。敌方通过"门"发动袭击，为游戏提供威胁来源和叙事背景。

## 四、开发路线

### 4.1 MVP（最小可玩版本）

实现核心机制闭环，让模组"能玩起来"：
- Trion 能量系统（产生、消耗、恢复）
- 基础触发器（至少一种触发体 + 若干触发芯片）
- 战斗体状态切换（激活、伤害隔离、破裂、紧急脱离）
- 基础敌对派系（至少一个近界民派系 + 基础 Trion 兵）

MVP 的验收标准：殖民者可以装备触发器、激活战斗体、用触发芯片的能力战斗、面对近界民的袭击。

### 4.2 扩展阶段

在 MVP 基础上逐步丰富内容：
- 更多触发芯片种类和组合
- 更多派系和 Trion 兵类型
- 副作用系统
- 制造设施和玩家自有 Trion 兵
- 黑触发器（特殊事件获取）
- 远征船、门等世界事件

### 4.3 长期愿景

一个内容丰富、机制完整的大型模组，涵盖 Trion 能量、触发器、战斗体、Trion 兵、派系对抗的完整体验。
