# 2.2 真身→战斗体→真身 领域过程分析

## 阶段1：激活——从真身到战斗体

### 1.1 激活前状态

队员处于**真身状态**：
- 手持触发体（无刃柄状物理装置）
- 触发体内预装载了触发芯片配置（8槽：主4+副4）
- Trion腺体中储存着个人Trion总量，全部可用

> 注：触发芯片装载决定的是战斗体的可用能力。即使没有装载任何芯片，也可以激活生成战斗体，只是没有武器和技能可用。

### 1.2 激活瞬间

队员触摸触发体，发出激活意图（语音指令"Trigger On"常用但非必须，关键是意图本身）。

**激活过程中同时发生的事件**：

1. **真身收容**：真身被移入触发体内的近界空间，进入完全停滞状态（不老化、不受伤、不消耗能量）
2. **战斗体构筑**：Trion能量物质化，全新构建一具战斗体
3. **芯片注册**：已装载的触发芯片向Trion资源池注册，锁定各自的占用量
4. **意识转移**：使用者的意识从真身转移到战斗体中

### 1.3 主观体验

从使用者的主观感受来说，激活转换是**瞬间且顺滑的**。使用者通常不会察觉自己"换了一个身体"——意识的连续性没有中断，感知没有断层。就像眨了一下眼，睁开时已经身处战斗体中。

### 1.4 激活完成后的状态

**Trion资源池变化**：

```
激活前：
┌─────────────────────────────────────┐
│  Trion总量 = 100%                    │
│  ████████████████████████████████   │
│  全部可用                            │
└─────────────────────────────────────┘

激活后：
┌─────────────────────────────────────┐
│  占用量(芯片注册)  │  可用量(战斗用)  │
│  ██████████████  │  ████████████████│
│  锁定不可使用     │  战斗中可消耗     │
└─────────────────────────────────────┘
```

**关键约束**：
- 芯片配置在战斗中固定，无法更换（只能在已装载的芯片间切换）
- Trion器官在战斗体状态下**无法恢复Trion**——一切消耗不可逆
- 占用量被芯片锁定，战斗中不可使用

### 1.5 副作用的跟随

如果使用者拥有副作用（Side Effect），副作用会**自然跟随意识存在于战斗体中**。副作用是与使用者意识绑定的能力，意识在哪里，副作用就在哪里。

## 阶段2：战斗——战斗体状态下的活动

### 2.1 战斗体的基础特性

战斗体是完全由Trion能量物质化构成的替代躯体，具备以下特性：

| 特性 | 说明 |
|------|------|
| **非Trion免疫** | 对常规武器（坦克炮弹、导弹等）几乎无敌，只有Trion武器能造成实质伤害 |
| **耐久度统一** | 所有使用者的战斗体耐久度相同，与个人Trion容量无关 |
| **缺陷修复** | 自动修复真身的身体缺陷（如视力不良者获得正常视力） |
| **无限体力** | 不会因运动产生身体疲劳（但精神疲劳仍存在） |
| **疼痛可控** | 伤害只产生轻微疼痛反馈，且可被禁用 |
| **无法自愈** | 受伤后不会愈合，伤口停止泄漏但不会修复 |

### 2.2 战斗中的Trion消耗

| 消耗类型 | 触发时机 | 特点 |
|---------|---------|------|
| **维持消耗** | 持续 | 战斗体存在的基础消耗，不可避免 |
| **激活消耗** | 激活芯片时 | 每次激活芯片的固定消耗 |
| **使用消耗** | 动作触发 | 射击、挥砍、技能释放等 |
| **流失消耗** | 受伤时 | 伤口导致Trion气态泄漏，约1分钟后自动停止 |

### 2.3 战斗体的终结条件

当以下任一条件满足时，战斗体状态终结，进入阶段3：

| 条件 | 说明 | 进入方式 |
|------|------|---------|
| **使用者主动解除** | 使用者发出Trigger Off意图 | → 路径A（主动解除） |
| **Trion可用量耗尽** | 战斗消耗导致可用Trion降至零 | → 路径B（被动破裂） |
| **供给系统被破坏** | 心脏处的Trion供给系统被摧毁 | → 路径B（被动破裂） |
| **中继系统被切断** | 脑部的Trion中继中心被切断 | → 路径B（被动破裂） |
| **战斗体耐久极限** | 累积伤害超过战斗体承受极限 | → 路径B（被动破裂） |
| **主动触发紧急脱离** | 使用者主动引爆战斗体，换取传送撤退时机 | → 路径B（被动破裂） |

> **主动触发紧急脱离**：使用者判断局势不利时，可主动放弃战斗体以换取紧急脱离的传送撤退机会。Trion结算上与被动破裂相同。

## 阶段3：解除——从战斗体回到真身

### 3.1 两条路径的本质区别

无论哪条路径，战斗体都会消散——每次激活都是全新构筑，解除时战斗体连同所有战斗伤害一同消失。

两条路径的区别在于**Trion的结算方式**：

| 对比项 | 路径A：主动解除 | 路径B：被动破裂 |
|--------|---------------|---------------|
| **触发方式** | 使用者主动发出Trigger Off意图 | Trion耗尽/弱点被毁/主动触发紧急脱离 |
| **战斗体去向** | 消散 | 破裂消散（裂纹扩散→爆炸成烟雾） |
| **芯片占用Trion** | **解锁返还**——占用量重新变为可用 | **随战斗体一同流失耗尽** |
| **真身出现位置** | 原地释放 | 原地释放（无紧急脱离）/ 传送至安全位置（有紧急脱离） |
| **Trion恢复起点** | 从较高水平开始恢复（保留了占用量+未消耗量） | 从接近枯竭状态开始恢复 |
| **重新激活等待** | 较短（Trion储备充足） | 较长（需等待Trion从枯竭中恢复） |

### 3.2 路径A：主动解除（Trigger Off）

**过程**：

```
1. 使用者发出解除意图
   └→ 语音指令"Trigger Off"常用但非必须

2. 战斗体消散
   └→ 战斗体"剥离"，Trion能量消散

3. 芯片占用Trion解锁
   └→ 之前被芯片注册锁定的Trion重新变为可用

4. 真身释放
   └→ 真身从触发体内的近界空间释放
   └→ 出现在原位置
   └→ 意识从战斗体转回真身
```

**Trion结算**：

```
主动解除后：
┌─────────────────────────────────────┐
│  已消耗(战斗中)  │  可恢复(占用释放+剩余) │
│  ░░░░░░░░░░░░  │  ████████████████████│
│  需时间恢复      │  占用量已解锁返还      │
└─────────────────────────────────────┘
```

### 3.3 路径B：被动破裂

**触发场景**：
- Trion可用量耗尽
- 供给系统（心脏处）被破坏
- 中继系统（脑部）被切断
- 战斗体耐久极限超过
- 使用者主动触发紧急脱离（自毁换取撤退时机）

**过程**：

```
1. 战斗体开始崩溃
   └→ 裂纹在表面扩散
   └→ 达到临界点后爆炸成烟雾

2. 芯片占用Trion流失
   └→ 占用量随战斗体一同流失耗尽

3. 紧急脱离判定
   ├→ 有紧急脱离（B级以上标配）：
   │   └→ 自动激活传送
   │   └→ 真身在传送过程中处于异空间，无法被攻击
   │   └→ 传送至预设安全位置（小队战略室/分部设施）
   │   └→ 有效范围：三门市内保证有效，远征距母舰3km内有效
   │
   └→ 无紧急脱离（C级训练生/S级黑触发器使用者）：
       └→ 真身直接在原地释放
       └→ 可能面临危险环境
```

**Trion结算**：

```
被动破裂后：
┌─────────────────────────────────────┐
│  已消耗(100%)                        │
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░    │
│  占用量也流失耗尽，需长时间恢复       │
└─────────────────────────────────────┘
```

## 阶段4：回归——真身状态的恢复

### 4.1 真身状态回滚

无论通过哪条路径回归，真身的状态回滚规则相同：

| 回滚项 | 规则 | 说明 |
|--------|------|------|
| **生理状态** | 回滚到收容时的快照 | 战斗体期间的所有伤害完全不继承 |
| **身体缺陷** | 恢复原状 | 战斗体中修复的缺陷（如视力）回到真身原有状态 |
| **意识/记忆** | 完整保留 | 战斗体期间的所有经历、记忆、判断完整保留 |
| **精神疲劳** | 保留 | 战斗中的精神消耗不会因回归真身而消失 |
| **副作用** | 自然跟随 | 副作用与意识绑定，回归真身后继续存在 |

> **核心战略价值**：这就是战斗体系统的根本意义——实现"无伤战斗"。无论战斗多么惨烈，真身始终安全。

### 4.2 Trion恢复

回归真身后，Trion器官重新开始工作，可以通过日常生活（进食、休息）恢复Trion。

| 解除方式 | 恢复起点 | 重新激活等待时间 |
|---------|---------|----------------|
| 主动解除 | 较高（占用量已返还+剩余可用量） | 较短 |
| 被动破裂 | 接近枯竭 | 较长（低Trion者如三云修需1-2小时） |

### 4.3 下次激活

当Trion恢复到足够水平后，队员可以再次激活触发器。**每次激活都是全新构筑一具战斗体**——上一次战斗体的伤害、状态完全不存在，新的战斗体总是以完好状态开始。

## 总结：全过程一览

```
┌──────────────────────────────────────────────────────────┐
│  阶段1：激活                                               │
│  真身(全部Trion可用) → 触发体激活 → 战斗体(Trion分为占用+可用) │
│  · 真身收容至近界空间（停滞）                                │
│  · 全新战斗体构筑                                          │
│  · 意识瞬间顺滑转移                                        │
│  · 副作用自然跟随意识                                       │
│  · 芯片注册锁定占用Trion                                    │
├──────────────────────────────────────────────────────────┤
│  阶段2：战斗                                               │
│  · Trion持续消耗（维持+使用+流失），不可恢复                  │
│  · 战斗体无法自愈，伤害累积                                  │
│  · 终结条件：主动解除 / Trion耗尽 / 弱点被毁 / 主动紧急脱离   │
├──────────────────────────────────────────────────────────┤
│  阶段3：解除                                               │
│  路径A（主动解除）：                                        │
│  · 战斗体消散，芯片占用Trion解锁返还                         │
│  · 真身原地释放                                             │
│                                                            │
│  路径B（被动破裂）：                                        │
│  · 战斗体破裂消散，芯片占用Trion随之流失耗尽                  │
│  · 有紧急脱离→传送至安全位置 / 无紧急脱离→原地释放            │
├──────────────────────────────────────────────────────────┤
│  阶段4：回归                                               │
│  · 真身状态回滚至收容时快照（伤害不继承）                     │
│  · 意识/记忆完整保留                                        │
│  · Trion器官重新开始恢复                                    │
│  · 下次激活 = 全新构筑战斗体                                 │
└──────────────────────────────────────────────────────────┘
```
