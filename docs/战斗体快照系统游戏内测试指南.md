# 战斗体快照系统 - 游戏内测试指南 v2.0

## 如何测试

### 1. 启动游戏并准备测试环境

1. 启动 RimWorld
2. 开启开发模式（Options → Dev mode）
3. 创建新游戏或加载存档

### 2. 给 Pawn 添加 Trion 腺体基因

**方法 A：使用开发工具**
1. 选中一个 Pawn
2. 按 `~` 打开开发菜单
3. 点击 `Add gene...`
4. 搜索 `BDP_Gene_TrionGland`
5. 点击添加

**方法 B：使用基因编辑器**
1. 选中一个 Pawn
2. 打开健康标签
3. 点击 `Add gene`
4. 找到 `Trion腺体`
5. 点击添加

### 3. 查看战斗体按钮

添加基因后，选中 Pawn，你会在屏幕下方看到一个新按钮：
- **激活战斗体** - 当战斗体未激活时显示
- **解除战斗体** - 当战斗体已激活时显示

如果开启了开发模式，还会看到：
- **查看快照** - 输出快照状态到日志

### 4. 测试场景

#### 场景 1：基础激活/解除
1. 点击 **激活战斗体** 按钮
2. 观察 Pawn 的衣物变化（应该穿上战斗体装甲）
3. 点击 **解除战斗体** 按钮
4. 观察 Pawn 恢复原状

#### 场景 2：衣物换装测试
1. 给 Pawn 穿上几件衣物（外套、裤子、帽子等）
2. 右键点击某件衣物 → 选择 **锁定**
3. 点击 **激活战斗体**
4. **预期结果**：
   - Pawn 穿上战斗体装甲
   - 原衣物消失
5. 点击 **解除战斗体**
6. **预期结果**：
   - Pawn 恢复原衣物
   - 锁定状态保持

#### 场景 3：物品转移测试
1. 给 Pawn 背包添加物品（食物、药品等）
2. 右键点击某个物品 → 选择 **不出售**
3. 点击 **激活战斗体**
4. **预期结果**：
   - 背包物品消失
5. 点击 **解除战斗体**
6. **预期结果**：
   - 背包物品恢复
   - "不出售"标记保持

#### 场景 4：物品腐烂测试
1. 给 Pawn 背包添加易腐食物（生肉、蔬菜等）
2. 记录食物的腐烂度
3. 点击 **激活战斗体**
4. 使用开发工具加速时间（按 `4` 键）
5. 等待几天游戏时间
6. 点击 **解除战斗体**
7. **预期结果**：
   - 食物腐烂度未增加（或增加很少）

#### 场景 5：存读档测试
1. 点击 **激活战斗体**
2. 保存游戏
3. 退出并重新加载存档
4. **预期结果**：
   - Pawn 仍然穿着战斗体装甲
   - 按钮显示 **解除战斗体**
5. 点击 **解除战斗体**
6. **预期结果**：
   - 所有状态正确恢复

#### 场景 6：Trion消耗测试（M2伤害系统）
1. 点击 **激活战斗体**
2. 查看当前Trion值（假设为100）
3. 使用开发工具让Pawn受到伤害（伤害量15）
4. **预期结果**：
   - Trion减少（约7.5，按0.5倍率计算）
   - 日志显示Trion消耗信息
   - Pawn没有受到真实伤害

#### 场景 7：影子HP测试（M2伤害系统）
1. 点击 **激活战斗体**
2. 按F12打开日志，查看影子HP初始化信息
3. 使用开发工具让Pawn手臂受到伤害（伤害量10）
4. **预期结果**：
   - 日志显示手臂影子HP减少
   - 手臂仍然正常（未破坏）
   - Trion减少

#### 场景 8：非关键部位破坏测试（M2伤害系统）
1. 点击 **激活战斗体**
2. 使用开发工具让Pawn手臂受到大量伤害（直到影子HP耗尽）
3. **预期结果**：
   - 日志显示"影子部位被毁"
   - 手臂失去功能（无法操作）
   - 没有疼痛和流血
   - 战斗体继续运行（未破裂）
4. 点击 **解除战斗体**
5. **预期结果**：
   - 手臂恢复正常功能

#### 场景 9：关键部位破坏测试（M2伤害系统）
1. 点击 **激活战斗体**
2. 使用开发工具让Pawn头部受到大量伤害（直到影子HP耗尽）
3. **预期结果**：
   - 日志显示"关键部位破坏"
   - 日志显示"战斗体破裂"
   - 战斗体自动解除（紧急脱离）
   - Pawn真身完好（头部未受伤）

#### 场景 10：Trion耗尽破裂测试（M2伤害系统）
1. 使用开发工具将Pawn的Trion设置为较低值（如10）
2. 点击 **激活战斗体**
3. 使用开发工具让Pawn受到大量伤害
4. **预期结果**：
   - Trion快速耗尽
   - 日志显示"Trion耗尽"
   - 日志显示"战斗体破裂"
   - 战斗体自动解除（紧急脱离）
   - Pawn真身完好

### 6. 常见问题

**M1快照系统问题**：

**Q: 按钮不显示？**
A: 确认 Pawn 拥有 `BDP_Gene_TrionGland` 基因，且 Pawn 属于玩家阵营。

**Q: 点击按钮没反应？**
A: 查看日志（F12）是否有错误信息。

**Q: 战斗体装甲看起来像护盾？**
A: 这是临时贴图，未来会替换为专用贴图。

**Q: 激活后 Pawn 没有穿上装甲？**
A: 检查 GeneDef 的 modExtensions 是否正确配置了 `defaultCombatApparel`。

**Q: 解除后衣物没有恢复？**
A: 查看日志中的错误信息，可能是容器序列化问题。

**M2伤害系统问题**：

**Q: 受伤后Trion没有减少？**
A: 检查是否正确激活战斗体，查看日志是否有"伤害拦截"信息。

**Q: 部位破坏后仍然有疼痛？**
A: 检查HediffDef配置，确认`BDP_CombatBodyPartDestroyed`没有设置painOffset。

**Q: 战斗体破裂后真身受伤？**
A: 这是bug，检查拦截点是否正确阻止了原版伤害应用。

**Q: 影子HP没有初始化？**
A: 查看日志中的"ShadowHPTracker 初始化"信息，确认部位已加载。

**Q: 解除后部位没有恢复？**
A: 检查PartDestructionHandler.Clear()是否被正确调用，查看日志中的"恢复部位"信息。

## 按钮说明

### 激活战斗体
- **图标**：攻击图标（临时）
- **功能**：
  - 拍摄当前状态快照（Hediff、需求、衣物、物品）
  - 卸下所有衣物，存入容器
  - 穿上战斗体装甲
  - 转移背包物品到容器
- **限制**：只能在玩家殖民者上使用

### 解除战斗体
- **图标**：攻击图标（临时）
- **功能**：
  - 恢复 Hediff 状态
  - 恢复需求值
  - 卸下战斗体装甲，存入容器
  - 穿上原衣物
  - 恢复背包物品
- **限制**：只能在已激活战斗体的 Pawn 上使用

### 查看快照（开发模式）
- **图标**：集合点图标
- **功能**：输出快照详细状态到日志
- **限制**：只在开发模式下显示
