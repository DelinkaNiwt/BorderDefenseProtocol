# 技术基础知识库

## 文档清单

### 贴图指导规范
- **RimWorld贴图指导规范.md** - 规范化贴图开发指南（简化版 v0.2.1）
  - 内容：分类、尺寸、技术要求、命名规范、GraphicData配置
  - 用途：快速参考和开发指导
  - 推荐：开发过程中作为主要参考文档

### 相关资源
- **占位贴图资源库**：`参考资源/通用资源/占位贴图/`
  - 120个规范化占位贴图
  - 所有变式贴图均包含标识标签
  - 用于开发测试和规范参考

- **生成脚本**：`工具/placeholder_texture_generator/`
  - 可定制的占位贴图生成工具
  - 支持各类贴图类型和变式
  - 可随时重新生成修改配置后的贴图
