# 模组知识库

本目录存储可复用的知识，供多个模组项目参考。

## 核心原则

- **避免重复存储**：RiMCP 可查询的 API 文档不存储，改为记录错误和陷阱
- **强调经验积累**：记录"我犯过什么错"而非"如何正确做"
- **明确知识边界**：题材知识与工程知识分离

## 子目录说明

| 目录 | 用途 | 主要内容 |
|------|------|--------|
| **技术基础/** | RiMCP查不到的技术知识 | Harmony最佳实践、性能优化案例 |
| **工程范例/** | 已验证的架构模式 | 从完成项目提炼（权限系统、能量管理等） |
| **题材库/** | 创意改编领域 | 创意改编案例、文案范例、题材陷阱 |
| **经验教训/** | 实践总结与问题解决 | 项目总结、API陷阱、常见问题解方 |
| **游戏百科/** | 纯游戏知识 | 玩法技巧、数值参考 |
| **头脑风暴/** | 创意灵感初稿 | 未经加工的想法（进入项目后转移） |

## 工作流

创意 → 头脑风暴 → 项目docs → 完成后提炼 → 工程范例/题材库
