## 1. 系统概览与设计哲学

### 1.1 什么是涡轮喷气背包？

**本质**：一个可装备的物品，赋予穿戴者**飞行能力**。

```
┌───────────────────────┐
│   涡轮喷气背包        │
├───────────────────────┤
│  Apparel(装备)        │
│  ↓ equip              │
│  Pawn(角色)           │
│  ↓ ability gained     │
│  Ability(飞行能力)    │
│  ↓ activate           │
│  Movement(飞行移动)   │
│  ↓ fuel consumed      │
│  Magazine(充能系统)   │
└───────────────────────┘
```

### 1.2 设计哲学

| 哲学 | 实现 | 优势 |
|-----|------|------|
| **模块化** | 装备 + 能力分离 | 易扩展（龙背包、其他飞行物品） |
| **易用性** | 装备即获能力 | 玩家无需复杂操作 |
| **可平衡** | XML完全驱动 | 不改代码即可调参数 |
| **多样性** | 多种背包型号 | 提供不同策略选择 |
| **互动性** | 视觉反馈 + 声音 | 沉浸式游戏体验 |

### 1.3 关键特性对比

| 特性 | 涡轮喷气 | 龙背包 | 魔法飞行 |
|-----|--------|-------|---------|
| 获得方式 | 装备 | 装备 | 研究/能力 |
| 飞行距离 | 中等 | 长距离 | 无限 |
| 充能方式 | Magazine | 魔法值 | 能量 |
| 特殊能力 | 快速移动 | 龙息 | 法术 |
| 科技需求 | 中等 | 高等 | 特殊 |

## 2. Biotech DLC能力系统基础

### 2.1 Biotech能力系统概览

RimWorld 1.6 Biotech DLC引入了全新的**Pawn.abilities**系统。

```csharp
// Pawn的能力系统接口
public class Pawn : Thing
{
    /// <summary>
    /// 该Pawn拥有的所有能力
    /// </summary>
    public PawnAbilities abilities { get; set; }
}

// 能力管理器
public class PawnAbilities
{
    // 获得能力
    public void GainAbility(AbilityDef abilityDef);

    // 失去能力
    public void RemoveAbility(AbilityDef abilityDef);

    // 尝试激活能力
    public bool TryActivateAbility(AbilityDef abilityDef, LocalTargetInfo target);

    // 获取能力实例
    public Ability GetAbility(AbilityDef abilityDef);
}
```

### 2.2 关键概念

| 概念 | 说明 | 示例 |
|-----|------|------|
| **AbilityDef** | 能力定义（XML驱动） | KT_TurbojetFly |
| **Ability** | 能力实例（运行时） | pawn.abilities.GetAbility(abilityDef) |
| **CompAbilityEffect** | 能力效果组件 | 充能系统、冷却、副作用 |
| **CompProperties** | 能力属性配置 | maxCharges, reloadTicks |

## 3. 核心架构：3层设计

### 3.1 三层架构图

```
┌─────────────────────────────────────┐
│   第1层：装备层                     │
│   Apparel (Equippable Item)        │
│   - 物理对象                        │
│   - 穿戴/脱下事件                   │
│   - 伤害、质量、外观                │
└────────────┬────────────────────────┘
             │ equip → CompApparelGiveAbility
             ▼
┌─────────────────────────────────────┐
│   第2层：能力层                     │
│   Ability (Running Instance)        │
│   - 获得/失去能力                    │
│   - 能力可用性检查                   │
│   - 触发条件判定                     │
└────────────┬────────────────────────┘
             │ activate → JobDriver_CastAbility
             ▼
┌─────────────────────────────────────┐
│   第3层：效果层                     │
│   CompAbilityEffect + Magazine      │
│   - 充能管理                         │
│   - 冷却系统                         │
│   - 实际移动/伤害                    │
│   - 副作用处理                       │
└─────────────────────────────────────┘
```

### 3.2 数据流动

```
玩家装备背包
  ↓
Notify_Equipped() 事件触发
  ↓
CompApparelGiveAbility.Notify_Equipped()
  ↓
pawn.abilities.GainAbility(abilityDef)
  ↓
Ability 实例创建
  ↓
CompAbility_Magazine 初始化（maxCharges=5）
  ↓
玩家可以使用能力（如果有充能）
  ↓
右键地点/按钮 → JobDriver_CastAbility
  ↓
CompAbility_Magazine.TryConsumeCharge() → true/false
  ↓
true: 执行飞行动作，开始充能回复
false: 显示"充能不足"，不执行
```

## 4. 装备-能力绑定机制

### 4.1 核心组件：CompApparelGiveAbility

```csharp
/// <summary>
/// 最小化、最优雅的设计：仅15行代码
/// 展示了Biotech能力系统的强大
/// </summary>
public class CompApparelGiveAbility : ThingComp
{
    public CompProperties_ApparelGiveAbility Props =>
        (CompProperties_ApparelGiveAbility)props;

    /// <summary>
    /// 装备时触发（穿戴背包时自动调用）
    /// </summary>
    public override void Notify_Equipped(Pawn pawn)
    {
        base.Notify_Equipped(pawn);

        // 空值检查（防御性编程）
        if (pawn?.abilities == null || Props.abilityDef == null)
            return;

        // 赋予能力（一行代码完成！）
        pawn.abilities.GainAbility(Props.abilityDef);

        // 可选：记录日志
        Log.Message($"[涡轮] {pawn.NameShortColored}装备了背包，获得能力：{Props.abilityDef.label}");
    }

    /// <summary>
    /// 脱下时触发（卸下背包时自动调用）
    /// </summary>
    public override void Notify_Unequipped(Pawn pawn)
    {
        base.Notify_Unequipped(pawn);

        if (pawn?.abilities == null || Props.abilityDef == null)
            return;

        // 失去能力
        pawn.abilities.RemoveAbility(Props.abilityDef);

        Log.Message($"[涡轮] {pawn.NameShortColored}卸下了背包，失去能力：{Props.abilityDef.label}");
    }
}

/// <summary>
/// 属性配置
/// </summary>
public class CompProperties_ApparelGiveAbility : CompProperties
{
    // 赋予的能力定义
    public AbilityDef abilityDef;

    // 卸下时是否移除能力？
    public bool removeAbilityOnUnequip = true;

    public CompProperties_ApparelGiveAbility()
    {
        compClass = typeof(CompApparelGiveAbility);
    }
}
```

### 4.3 生命周期详解

```
时间轴：

[时间 0] 背包在地上，未装备
  - background: 无任何Component运行

[时间 1] 角色装备背包
  - 调用 Apparel.Notify_Equipped(pawn)
  - 调用 CompApparelGiveAbility.Notify_Equipped(pawn)
    - pawn.abilities.GainAbility(abilityDef)
    - Ability实例创建
    - CompAbility_Magazine初始化
  - 状态：能力已获得，充能满

[时间 2-N] 角色穿着背包移动
  - 定期检查充能系统
    - 每tick消耗充能（如使用）
    - 每tick恢复充能（无使用时）
  - 能力可被激活

[时间 X] 角色取下背包
  - 调用 Apparel.Notify_Unequipped(pawn)
  - 调用 CompApparelGiveAbility.Notify_Unequipped(pawn)
    - pawn.abilities.RemoveAbility(abilityDef)
    - CompAbility_Magazine清除
    - 充能重置
  - 状态：能力已失去

[时间 X+1] 背包在地上
  - 回到初始状态
```

### 5.2 CompAbility_Magazine完整实现

```csharp
/// <summary>
/// Magazine系统：离散充能管理
/// 适合爆发型、用途明确的能力
/// </summary>
public class CompAbility_Magazine : CompAbilityEffect
{
    // 内部状态
    private int charges;                        // 当前充能数
    private int ticksToNextCharge;              // 下一次充能的计时器

    // 属性访问
    public int Charges => charges;
    public int MaxCharges => Props.maxCharges;

    // 充能进度（用于UI显示）
    public float ReloadProgress =>
        1f - (float)ticksToNextCharge / (float)Props.reloadTicks;

    public bool IsFullyCharged => charges >= MaxCharges;
    public bool IsCharging => ticksToNextCharge > 0 && charges < MaxCharges;

    // ========== 生命周期 ==========

    /// <summary>
    /// 初始化：设置初始充能数
    /// </summary>
    public override void Initialize(CompProperties props)
    {
        base.Initialize(props);

        // 从Props或设置初始值
        charges = Props.startingCharges > 0 ? Props.startingCharges : Props.maxCharges;
        ticksToNextCharge = 0;  // 立即可用
    }

    /// <summary>
    /// 每tick调用：处理充能逻辑
    /// </summary>
    public override void CompTick()
    {
        base.CompTick();

        // 只有在未满充且有充能进行中时才处理
        if (charges >= MaxCharges)
            return;  // 充能满，不需要进一步处理

        if (IsCharging)
        {
            // 递减计时器
            ticksToNextCharge--;

            // 计时到，补充一个充能
            if (ticksToNextCharge <= 0)
            {
                ReloadOneCharge();
            }
        }
        else if (charges < MaxCharges)
        {
            // 首次进入充能（之前没有充能在进行）
            StartCharging();
        }
    }

    /// <summary>
    /// 尝试消耗一次充能
    /// 返回true=成功，false=充能不足
    /// </summary>
    public bool TryConsumeCharge()
    {
        if (charges > 0)
        {
            // 消耗一个充能
            charges--;

            // 如果还有更多充能要恢复，重置计时器
            if (charges < MaxCharges)
            {
                ticksToNextCharge = Props.reloadTicks;
            }

            return true;
        }

        return false;  // 充能不足！
    }

    // ========== 内部方法 ==========

    /// <summary>
    /// 恢复一个充能
    /// </summary>
    private void ReloadOneCharge()
    {
        charges++;

        // 如果还需要继续恢复
        if (charges < MaxCharges)
        {
            ticksToNextCharge = Props.reloadTicks;
        }
        else
        {
            // 已满充，停止计时
            ticksToNextCharge = 0;
        }

        // 触发事件（可选）
        OnChargeReloaded(charges);
    }

    /// <summary>
    /// 开始充能流程
    /// </summary>
    private void StartCharging()
    {
        if (charges < MaxCharges)
        {
            ticksToNextCharge = Props.reloadTicks;
        }
    }

    /// <summary>
    /// 强制充满充能
    /// </summary>
    public void RestoreAllCharges()
    {
        charges = MaxCharges;
        ticksToNextCharge = 0;
    }

    // ========== 保存/加载 ==========

    public override void ExposeData()
    {
        base.ExposeData();
        Scribe_Values.Look(ref charges, "charges");
        Scribe_Values.Look(ref ticksToNextCharge, "ticksToNextCharge");
    }
}

/// <summary>
/// Magazine属性配置
/// </summary>
public class CompProperties_Magazine : CompProperties_AbilityEffect
{
    // 最大充能次数
    public int maxCharges = 5;

    // 每次充能所需时间（ticks）
    public int reloadTicks = 300;

    // 初始充能数（可选，默认=maxCharges）
    public int startingCharges = -1;

    // UI是否显示充能进度条？
    public bool showReloadBar = true;

    public CompProperties_Magazine()
    {
        compClass = typeof(CompAbility_Magazine);
    }
}
```

## 6. 飞行力学与运动控制

### 6.1 飞行系统概述

```csharp
/// <summary>
/// 飞行能力的执行者：JobDriver_CastAbility
/// 当玩家激活"飞行"能力时触发
/// </summary>
public class JobDriver_TurbojetFly : JobDriver_CastAbility
{
    public override bool TryMakePreToilToils()
    {
        // 前置条件检查
        AddChain(new Toil()
        {
            initAction = () =>
            {
                // 检查是否有充能
                var ability = pawn.abilities.GetAbility(AbilityDefOf.KT_TurbojetFly);
                var magazine = ability?.GetComponent<CompAbility_Magazine>();

                if (magazine == null || !magazine.TryConsumeCharge())
                {
                    // 充能不足
                    pawn.jobs.EndCurrentJob(JobCondition.Incompletable);
                    return;
                }

                // 播放起飞特效
                SpawnTakeoffEffect();
            }
        });

        // 飞行主过程
        AddChain(new Toil()
        {
            tickAction = () =>
            {
                // 向目标位置飞行
                Vector3 targetPos = job.targetA.Cell.ToVector3Shifted();
                float flightSpeed = 5f;  // 格/秒

                // 移动Pawn
                pawn.Position = Vector3Lerp(pawn.Position, targetPos, flightSpeed * Time.deltaTime);

                // 每帧生成轨迹特效
                SpawnTrailEffect(pawn.Position);
            },

            defaultCompleteMode = ToilCompleteMode.Instant
        });

        // 着陆
        AddChain(new Toil()
        {
            initAction = () =>
            {
                // 播放着陆特效
                SpawnLandingEffect();

                // 结束工作
                pawn.jobs.EndCurrentJob(JobCondition.Succeeded);
            }
        });

        return true;
    }

    private void SpawnTakeoffEffect()
    {
        MoteMaker.ThrowExplosion(pawn.Position.ToVector3Shifted(), pawn.Map, 1.5f, ThingDefOf.Mote_ThoughtFlash);
    }

    private void SpawnTrailEffect(Vector3 pos)
    {
        // 创建拖尾粒子
        MoteMaker.ThrowDustPuff(pos, pawn.Map, 0.5f);
    }

    private void SpawnLandingEffect()
    {
        MoteMaker.ThrowExplosion(pawn.Position.ToVector3Shifted(), pawn.Map, 2f, ThingDefOf.Mote_ExplosionFlash);
    }
}
```

## 8. 能力生命周期

### 8.1 完整时间轴

```
[游戏时间]  [事件]                    [状态]
─────────────────────────────────────────
0秒        游戏开始
          研究"涡轮喷气"

10秒       制作背包
          背包制作完成
          放在地上

          背包属性：
          - 等待被装备
          - 有CompApparelGiveAbility
          - 尚未赋予能力

15秒       角色装备背包
          → Notify_Equipped()触发
          → CompApparelGiveAbility.Notify_Equipped()
          → pawn.abilities.GainAbility(abilityDef)

          背包属性：
          - 已装备在角色身上
          - 能力已获得
          - Magazine已初始化：charges=5

16秒       角色有飞行能力
          - 能力可被激活
          - Magazine: 5/5充能

18秒       角色使用飞行能力
          → pawn.jobs.TryTakeOrderedJob(flyJob)
          → JobDriver执行
          → Magazine.TryConsumeCharge()
          → charges: 5→4

          状态：正在飞行...

20秒       飞行完成，着陆
          → JobDriver结束
          → charges = 4/5
          → 开始充能恢复

          充能过程：
          0-5秒:  4/5 (充能中...)
          5秒:    4→5, 满充

25秒       再次使用飞行
          → charges: 5→4

...

1000秒     角色卸下背包
          → Notify_Unequipped()触发
          → CompApparelGiveAbility.Notify_Unequipped()
          → pawn.abilities.RemoveAbility(abilityDef)

          背包属性：
          - 已卸下
          - 能力已失去
          - Magazine已清除
          - charges重置

          背包现在：
          - 在地上
          - 可被再次装备
          - 等待被其他角色或同一角色再次使用
```

### 8.2 能力可用性检查

```csharp
public class Pawn_AbilitiesChecker
{
    /// <summary>
    /// 检查能力是否可用
    /// </summary>
    public bool CanUseAbility(AbilityDef abilityDef)
    {
        // 1. 是否拥有该能力？
        var ability = abilities.GetAbility(abilityDef);
        if (ability == null)
            return false;  // 不拥有

        // 2. 是否满足能力前置条件？
        if (!ability.CanCast())
            return false;  // 前置不满足

        // 3. Magazine系统是否有充能？
        var magazine = ability.GetComponent<CompAbility_Magazine>();
        if (magazine != null && magazine.Charges <= 0)
            return false;  // 充能不足

        // 4. 冷却时间是否已过？
        if (ability.CooldownTicksRemaining > 0)
            return false;  // 还在冷却

        // 所有条件满足
        return true;
    }
}
```

### 10.2 能力查询缓存

```csharp
public class AbilitiesCache
{
    private Dictionary<AbilityDef, Ability> cachedAbilities = new();
    private int cacheValidTick = -1;

    public Ability GetAbility(AbilityDef abilityDef)
    {
        // 检查缓存是否有效
        if (cacheValidTick == Find.TickManager.TicksGame)
        {
            if (cachedAbilities.TryGetValue(abilityDef, out var cached))
                return cached;
        }

        // 缓存无效或未找到，进行查询
        var ability = pawn.abilities.GetAbility(abilityDef);
        cachedAbilities[abilityDef] = ability;
        cacheValidTick = Find.TickManager.TicksGame;

        return ability;
    }

    public void InvalidateCache()
    {
        cacheValidTick = -1;
    }
}
```

## 12. 常见问题与故障排查

### 问题1：装备背包后无法飞行

```
症状：背包已装备，但无法使用飞行能力

诊断步骤：
1. 检查背包是否真的装备上了
   Log.Message($"背包状态: {backpack.Wearer != null}");

2. 检查能力是否获得
   Log.Message($"能力数量: {pawn.abilities.AllAbilities.Count()}");
   var ability = pawn.abilities.GetAbility(abilityDef);
   Log.Message($"飞行能力: {ability != null}");

3. 检查Magazine是否初始化
   var magazine = ability?.GetComponent<CompAbility_Magazine>();
   Log.Message($"Magazine: {magazine != null}, 充能: {magazine?.Charges}");

4. 检查其他前置条件
   - 角色是否有"Moving"能力？
   - 角色是否昏迷/死亡？
   - 是否在禁飞区域？

解决方案：
- 重新装备背包（卸下再装上）
- 检查Notify_Equipped是否被正确调用
- 检查CompApparelGiveAbility是否配置正确
```

### 问题2：充能不恢复

```
症状：使用飞行后，充能一直显示为0

诊断：
1. 检查Magazine.CompTick是否被调用
   #if DEBUG
   Log.Message($"[Magazine] Tick: charges={Charges}, toNextCharge={ticksToNextCharge}");
   #endif

2. 检查能力是否被移除了
   var ability = pawn.abilities.GetAbility(abilityDef);
   if (ability == null)
       Log.Warning("能力已被移除！");

3. 检查Pawn是否被冻结/暂停
   if (pawn.Dead || pawn.Downed)
       Log.Message("Pawn状态异常");

解决方案：
- 确保CompTick被正确调用
- 检查能力定义的comps配置
- 重启游戏（强制重新初始化）
```

### 问题3：多个背包冲突

```
症状：同一角色装备多个背包时出现异常

原因：多个CompApparelGiveAbility尝试赋予不同能力

解决方案：
- 设计上避免多背包同装
- 或者实现能力合并机制（同一个能力，不同配置）

代码示例：
public override void Notify_Equipped(Pawn pawn)
{
    // 检查是否已经有该能力
    if (pawn.abilities.GetAbility(Props.abilityDef) != null)
    {
        Log.Warning($"角色已拥有能力{Props.abilityDef.label}，跳过");
        return;
    }

    pawn.abilities.GainAbility(Props.abilityDef);
}
```
