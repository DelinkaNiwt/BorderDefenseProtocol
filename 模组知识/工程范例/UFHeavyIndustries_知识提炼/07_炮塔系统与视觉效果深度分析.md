## 1. 炮塔系统概览

### 1.1 炮塔的核心角色

**在游戏中**：防守基地的关键防线

```
敌军进攻
  ↓
炮塔自动检测目标
  ↓
锁定并瞄准
  ↓
开火射击
  ↓
投射体飞向目标
  ↓
造成伤害/摧毁敌人
```

```
┌─────────────────────────────────────────┐
```

## 2. 三大炮塔类型

### 2.1 重型海军炮（HeavyNavalGun）- 21个文件

**设计理念**：大口径、低射速、高伤害的压制火力

```
炮身
  ↑
  ├─ 旋转机制（目标追踪）
  ├─ 仰角调节（高度补偿）
  └─ 开火控制（精准计时）

架座（基础）
  ├─ 能源接入点
  ├─ 目标信息处理
  └─ 炮弹装填机制
```

**关键组件**：
```csharp
public class CompCannonFire : ThingComp
{
    // 基础参数
    public float damagePerShot = 80f;        // 每次伤害
    public float fireRate = 0.2f;            // 每秒射击频率（0.2 = 5秒/次）
    public float accuracy = 0.95f;           // 命中精度
    public float range = 70f;                // 有效射程

    // 目标系统
    public LocalTargetInfo currentTarget;    // 当前目标
    public int targetingMode = TargetingMode.Automatic;

    // 旋转与仰角
    public float currentRotation = 0f;       // 水平旋转（度）
    public float currentElevation = 0f;      // 仰角（度）

    // 充能/装填
    public float currentChargeLevel = 0f;    // 当前充能（0-1）
    public const float CHARGE_SPEED = 0.02f;// 充能速度
}
```

**开火流程**：
```
1. 检测敌方目标（CompTargeting）
2. 计算射击参数（枪口位置、角度）
3. 旋转炮身至目标方向
4. 仰角补偿（考虑目标距离）
5. 装填炮弹（播放装填动画）
6. 开火（播放爆炸动画、声音）
7. 投射体飞向目标
8. 冷却（等待下一轮）
```

### 2.2 电磁炮（ElectromagneticCannon）- 17个文件

**设计理念**：快速、稳定、能量射击

```
线圈组
  ↓
强磁场加速
  ↓
能量弹射出
  ↓
直线飞行
  ↓
目标命中
```

**与海军炮的区别**：
- 更快的射速（无装填等待）
- 能量驱动而非实体弹药
- 更稳定的弹道（无随机偏差）
- 能耗更高

### 2.3 自由电子轨道激光（FreeElectronOrbitalLaser）- 7个文件

**设计理念**：最先进的光学武器，极高成本

```
激光源
  ↓
光学聚焦
  ↓
轨道镜定向（可选）
  ↓
激光束发射
  ↓
目标瞬时摧毁
```

**特点**：
- 最高技术等级
- 最高能耗
- 无弹道（类似Beam）
- 可穿透护盾（可选）
- 视觉效果最炫彩

### 3.2 建筑生命周期

```csharp
public class Building_Turret : Building
{
    private CompCannonFire compFire;
    private CompTargeting compTargeting;
    private bool isActive = false;

    // ========== 初始化 ==========

    public override void SpawnSetup(Map map, bool respawningAfterLoad)
    {
        base.SpawnSetup(map, respawningAfterLoad);

        // 获取关键组件
        compFire = GetComp<CompCannonFire>();
        compTargeting = GetComp<CompTargeting>();

        isActive = true;
    }

    // ========== 运行时更新 ==========

    public override void Tick()
    {
        base.Tick();

        // 检查是否有电源
        if (!HasPower())
        {
            SetActive(false);
            return;
        }

        SetActive(true);

        // 目标选择
        LocalTargetInfo target = compTargeting.FindTarget();
        if (target.IsValid)
        {
            compFire.currentTarget = target;
        }

        // 开火
        compFire.UpdateFiring(this);
    }

    private bool HasPower()
    {
        var compPower = GetComp<CompPowerTrader>();
        return compPower != null && compPower.PowerOn;
    }

    // ========== 销毁 ==========

    public override void Destroy(DestroyMode mode = DestroyMode.Vanish)
    {
        // 清理正在进行的开火
        compFire?.OnDestroy();

        base.Destroy(mode);
    }
}
```

## 4. 目标选择与AI

### 4.1 目标选择算法

```csharp
public class CompTargeting : ThingComp
{
    // 目标选择参数
    public TargetingMode targetingMode = TargetingMode.Automatic;
    public float maxTargetDistance = 70f;
    public bool preferHumanoids = true;
    public bool preferThreats = true;

    /// <summary>
    /// 寻找最优目标
    /// </summary>
    public LocalTargetInfo FindTarget()
    {
        // 步骤1：获取范围内的所有敌方事物
        List<Thing> potentialTargets = GetPotentialTargets();

        if (potentialTargets.Count == 0)
            return LocalTargetInfo.Invalid;

        // 步骤2：按优先级排序
        potentialTargets.Sort((a, b) =>
        {
            float scoreA = CalculateTargetScore(a);
            float scoreB = CalculateTargetScore(b);
            return scoreB.CompareTo(scoreA);  // 降序
        });

        // 步骤3：返回最高分的目标
        return LocalTargetInfo.FromThing(potentialTargets[0]);
    }

    /// <summary>
    /// 获取范围内的所有敌方事物
    /// </summary>
    private List<Thing> GetPotentialTargets()
    {
        var result = new List<Thing>();
        var map = parent.Map;
        var position = parent.Position;

        // 遍历所有Pawns（角色）
        foreach (var pawn in map.mapPawns.AllPawns)
        {
            // 跳过友方
            if (pawn.Faction == parent.Faction || pawn.Faction == Faction.OfPlayer)
                continue;

            // 跳过已死亡
            if (pawn.Dead)
                continue;

            // 检查距离
            int distance = position.DistanceTo(pawn.Position);
            if (distance > maxTargetDistance)
                continue;

            // 可被攻击
            result.Add(pawn);
        }

        return result;
    }

    /// <summary>
    /// 计算目标优先级分数（越高越优先）
    /// </summary>
    private float CalculateTargetScore(Thing thing)
    {
        float score = 100f;  // 基础分

        if (thing is Pawn pawn)
        {
            // 威胁等级加分
            if (preferThreats && pawn.equipment?.Primary != null)
                score += 50f;

            // 人形优先（比动物优先）
            if (preferHumanoids && pawn.RaceProps.Humanoid)
                score += 30f;

            // 血量少的优先（更容易击杀）
            float healthPercent = pawn.health.summaryHealth.SummaryHealthPercent;
            score += (1f - healthPercent) * 50f;

            // 距离近的优先
            int distance = parent.Position.DistanceTo(pawn.Position);
            score += (maxTargetDistance - distance) / maxTargetDistance * 30f;
        }

        return score;
    }

    public enum TargetingMode
    {
        Automatic,      // 自动选择
        Manuel,         // 玩家手动
        Priority,       // 按玩家设置的优先级
    }
}
```

### 4.2 目标评分例子

```
目标A（士兵，健康）：
  基础分：100
  + 威胁等级（有武器）：50
  + 人形优先：30
  + 距离（30格）：30 × (70-30)/70 = 17
  = 197分

目标B（野蛮人，已伤）：
  基础分：100
  + 威胁等级（有武器）：50
  - 人形（不是人类）：-10
  + 伤害（血量50%）：25
  + 距离（50格）：30 × (70-50)/70 = 8.6
  = 173分

目标C（动物，健康）：
  基础分：100
  - 人形（动物）：-10
  + 距离（20格）：30 × (70-20)/70 = 21.4
  = 111分

优先级：A > B > C
```

## 5. 开火机制详解

### 5.1 完整开火流程

```csharp
public class CompCannonFire : ThingComp
{
    // ========== 状态机 ==========

    public enum FireState
    {
        Idle,           // 空闲
        Aiming,         // 瞄准
        Charging,       // 装填/充能
        Ready,          // 准备就绪
        Firing,         // 发射中
        Cooldown        // 冷却中
    }

    private FireState currentState = FireState.Idle;
    private int stateTicksRemaining = 0;

    // ========== 参数 ==========

    public float damagePerShot = 80f;
    public float fireRate = 0.2f;  // 每秒射击次数
    public float accuracy = 0.95f;
    public LocalTargetInfo currentTarget;

    // 计算的值
    public float ticksBetweenShots => 60f / (fireRate * GenDate.TicksPerDay);  // 转换为Tick

    // ========== 主要方法 ==========

    public void UpdateFiring(Building building)
    {
        // 状态转移
        switch (currentState)
        {
            case FireState.Idle:
                UpdateIdle(building);
                break;
            case FireState.Aiming:
                UpdateAiming(building);
                break;
            case FireState.Charging:
                UpdateCharging(building);
                break;
            case FireState.Ready:
                UpdateReady(building);
                break;
            case FireState.Firing:
                UpdateFiring_State(building);
                break;
            case FireState.Cooldown:
                UpdateCooldown(building);
                break;
        }
    }

    // ========== 状态处理 ==========

    private void UpdateIdle(Building building)
    {
        // 检查是否有目标
        if (!currentTarget.IsValid)
            return;

        // 有目标，进入瞄准状态
        TransitionTo(FireState.Aiming, aimingDurationTicks: 30);
    }

    private void UpdateAiming(Building building)
    {
        // 旋转炮身对准目标
        RotateTowardsTarget(building, currentTarget);

        // 计算仰角（补偿距离）
        float elevation = CalculateElevation(building, currentTarget);
        ApplyElevation(building, elevation);

        // 倒计时
        stateTicksRemaining--;
        if (stateTicksRemaining <= 0)
        {
            TransitionTo(FireState.Charging, chargingDurationTicks: 20);
        }
    }

    private void UpdateCharging(Building building)
    {
        // 播放装填动画（可选）
        PlayChargingAnimation(building);

        // 倒计时
        stateTicksRemaining--;
        if (stateTicksRemaining <= 0)
        {
            TransitionTo(FireState.Ready, readyDurationTicks: 5);
        }
    }

    private void UpdateReady(Building building)
    {
        // 最后检查：目标仍然有效？
        if (!currentTarget.IsValid)
        {
            TransitionTo(FireState.Idle);
            return;
        }

        // 等待一个Tick确保所有动画就位
        stateTicksRemaining--;
        if (stateTicksRemaining <= 0)
        {
            TransitionTo(FireState.Firing);
        }
    }

    private void UpdateFiring_State(Building building)
    {
        // 发射投射体
        Fire(building, currentTarget);

        // 播放爆炸/闪光特效
        PlayFiringEffect(building);

        // 播放声音
        SoundDef.Named("Explosion_CannonShot").PlayOneShot(building.Position);

        // 进入冷却
        TransitionTo(FireState.Cooldown, cooldownDurationTicks: (int)ticksBetweenShots);
    }

    private void UpdateCooldown(Building building)
    {
        // 冷却倒计时
        stateTicksRemaining--;
        if (stateTicksRemaining <= 0)
        {
            // 冷却完毕，检查是否还有有效目标
            TransitionTo(FireState.Idle);
        }
    }

    // ========== 辅助方法 ==========

    private void Fire(Building building, LocalTargetInfo target)
    {
        // 计算枪口位置（从建筑模型上取）
        Vector3 muzzlePos = building.Position.ToVector3Shifted() + Vector3.up * 2.5f;

        // 计算弹道
        Vector3 targetPos = target.CenterVector3;

        // 应用精度（随机偏差）
        if (Random.value > accuracy)
        {
            // 偏离一定角度
            Vector3 error = Random.insideUnitSphere * 2f;
            targetPos += error;
        }

        // 创建投射体
        Thing projectile = ThingMaker.MakeThing(projectileDef);
        if (projectile is Projectile proj)
        {
            proj.Launch(muzzlePos, targetPos, building);
        }
    }

    private void RotateTowardsTarget(Building building, LocalTargetInfo target)
    {
        Vector3 buildingPos = building.Position.ToVector3();
        Vector3 targetPos = target.CenterVector3;

        Vector3 direction = (targetPos - buildingPos).normalized;
        float targetRotation = Mathf.Atan2(direction.x, direction.z) * Mathf.Rad2Deg;

        // 平滑旋转
        building.Rotation = Mathf.MoveTowards(building.Rotation, targetRotation, rotationSpeed);
    }

    private float CalculateElevation(Building building, LocalTargetInfo target)
    {
        // 根据距离计算仰角
        float distance = building.Position.DistanceTo(target.Cell);

        // 弹道学：距离越远，仰角越大
        float elevation = Mathf.Atan2(distance, 20f) * Mathf.Rad2Deg;

        return elevation;
    }

    private void TransitionTo(FireState newState, int durationTicks = 0)
    {
        if (currentState == newState)
            return;

        currentState = newState;
        stateTicksRemaining = durationTicks;

        // 触发事件（日志、动画等）
        OnStateChanged(newState);
    }
}
```

### 6.3 升级管理实现

```csharp
public class CompUpgradeable : ThingComp
{
    // 升级槽
    private List<UpgradeDef> installedUpgrades = new();
    private int upgradeSlots = 3;

    public bool CanAddUpgrade(UpgradeDef upgrade)
    {
        return installedUpgrades.Count < upgradeSlots
            && !installedUpgrades.Contains(upgrade);
    }

    public void AddUpgrade(UpgradeDef upgrade)
    {
        if (CanAddUpgrade(upgrade))
        {
            installedUpgrades.Add(upgrade);

            // 应用升级效果
            ApplyUpgradeEffect(upgrade);

            Log.Message($"[升级] 已安装 {upgrade.label}");
        }
    }

    public void ApplyUpgradeEffect(UpgradeDef upgrade)
    {
        // 获取开火组件
        var compFire = parent.GetComp<CompCannonFire>();
        if (compFire == null)
            return;

        // 应用各项加成（乘法）
        if (upgrade.damageBonus > 0)
            compFire.damagePerShot *= (1f + upgrade.damageBonus);

        if (upgrade.fireRateBonus > 0)
            compFire.fireRate *= (1f + upgrade.fireRateBonus);

        if (upgrade.rangeBonus > 0)
            compFire.maxTargetDistance *= (1f + upgrade.rangeBonus);

        if (upgrade.accuracyBonus > 0)
            compFire.accuracy = Math.Min(1f, compFire.accuracy + upgrade.accuracyBonus);
    }

    public float GetTotalDamageMultiplier()
    {
        float multiplier = 1f;
        foreach (var upgrade in installedUpgrades)
        {
            multiplier *= (1f + upgrade.damageBonus);
        }
        return multiplier;
    }
}
```

```csharp
public class CompAnimationRenderer : ThingComp

    public override void Initialize(CompProperties props)

    }

    public override void CompTick()
    {
        base.CompTick();

        // 更新动画状态
        UpdateAnimationState();
    }

    private void UpdateAnimationState()
    {
        var compFire = parent.GetComp<CompCannonFire>();
        if (compFire == null)
            return;

        // 根据开火状态更新动画
        switch (compFire.CurrentState)
        {
            case CompCannonFire.FireState.Aiming:
                sraRenderer.PlayAnimation("Aiming");
                break;

            case CompCannonFire.FireState.Charging:
                sraRenderer.PlayAnimation("Charging");
                break;

            case CompCannonFire.FireState.Firing:
                sraRenderer.PlayAnimation("Fire");
                sraRenderer.PlayEffects("MuzzleFlash");
                break;

            case CompCannonFire.FireState.Cooldown:
                sraRenderer.PlayAnimation("Idle");
                break;
        }
    }
}
```

```csharp
public class DamageCalculator
{
    /// <summary>
    /// 计算最终伤害
    /// </summary>
    public static float CalculateFinalDamage(
        Pawn target,
        float baseDamage,
        DamageDef damageType,
        float accuracyFactor = 1f)
    {
        float damage = baseDamage;

        // 1. 精度影响（低精度可能偏离或减伤）
        damage *= accuracyFactor;

        // 2. 装甲减伤
        if (damageType == DamageTypeDefOf.Bullet)
        {
            float armor = target.GetStatValue(StatDefOf.ArmorRating_Blunt);
            damage *= 1f - armor;
        }

        // 3. 护盾吸收（如果被护盾拦截，伤害为0）
        // ... 由AT-Field系统处理

        // 4. 生物体质修饰
        float bodyModifier = target.RaceProps.bodySize;
        damage *= bodyModifier;

        // 5. 部位特定修饰（如果击中特殊部位）
        // ... 可选的高级系统

        return Mathf.Max(0, damage);
    }
}
```

## 10. 伤害计算与平衡

### 10.1 DPS（伤害/秒）对比

```
重型海军炮：
  伤害/发：80
  射速：0.2 次/秒（5秒/次）
  DPS = 80 × 0.2 = 16 DPS

电磁炮：
  伤害/发：40
  射速：0.67 次/秒（1.5秒/次）
  DPS = 40 × 0.67 = 26.8 DPS

激光：
  伤害/发：35
  射速：0.5 次/秒（2秒/次）
  DPS = 35 × 0.5 = 17.5 DPS

能耗/DPS：
  重型海军炮：500W / 16DPS = 31.25 W/DPS
  电磁炮：300W / 26.8DPS = 11.2 W/DPS
  激光：400W / 17.5DPS = 22.9 W/DPS

结论：电磁炮最经济
```

### 10.2 平衡方向

```
如果玩家觉得"炮火太弱"：
  方案1: 增加 damagePerShot
  方案2: 提高 fireRate
  方案3: 降低电源消耗（使其可装备更多）

如果玩家觉得"防守太强"：
  方案1: 降低 accuracy（更容易偏离）
  方案2: 增加 cooldown（射速变慢）
  方案3: 提高建造成本或研究要求
  方案4: 增加电源需求（电网压力）
```
