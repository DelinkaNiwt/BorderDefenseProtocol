# 基因获得与移除途径

**总览**：基因的获得有5种途径（出生继承、种族默认、Xenogerm植入、基因组装、代码直接添加），移除有3种途径（Xenogerm替换、基因提取、代码直接移除）。核心流程是"基因提取→基因包存储→基因组装→Xenogerm植入"的工业化流水线。

## 获得途径对比表

| # | 途径 | 基因类型 | 触发方式 | 核心代码 | 说明 |
|---|------|---------|---------|---------|------|
| 1 | **出生继承** | Endogene | 怀孕→出生 | `PregnancyUtility.GetInheritedGenes()` | 父母Endogene按规则遗传给后代 |
| 2 | **种族默认** | Endogene | Pawn生成 | `PawnGenerator.GenerateGenes()` | XenotypeDef定义的默认基因 |
| 3 | **Xenogerm植入** | Xenogene | 使用Xenogerm物品 | `GeneUtility.ImplantXenogermItem()` | 清除旧Xenogene，添加新基因 |
| 4 | **Xenogerm再植入** | Xenogene | 吸血鬼能力 | `GeneUtility.ReimplantXenogerm()` | 将自身Xenogene复制给目标 |
| 5 | **代码直接添加** | 任意 | 模组/调试 | `Pawn_GeneTracker.AddGene()` | 直接API调用 |

## 移除途径对比表

| # | 途径 | 影响范围 | 触发方式 | 核心代码 | 说明 |
|---|------|---------|---------|---------|------|
| 1 | **Xenogerm替换** | 全部Xenogene | 植入新Xenogerm | `SetXenotype()` → 清除旧Xenogene | 植入新Xenogerm时自动清除全部旧Xenogene |
| 2 | **基因提取** | 单个基因 | 基因提取器 | `Building_GeneExtractor.Finish()` | 从Pawn提取基因生成Genepack |
| 3 | **代码直接移除** | 单个基因 | 模组/调试 | `Pawn_GeneTracker.RemoveGene()` | 直接API调用 |

## 核心流程：基因工业化流水线

```
Step 1: 基因提取
  Pawn → Building_GeneExtractor → Genepack（含1~4个基因）
  └── 提取后Pawn获得XenogermReplicating Hediff（恢复期12~20天）

Step 2: 基因包存储
  Genepack → 基因库（CompGenepackContainer）
  └── 通电时修复HP，断电时劣化

Step 3: 基因组装
  多个Genepack → Building_GeneAssembler → Xenogerm
  └── 需要：Genepack + 时间（按复杂度曲线）+ Archite胶囊（如需）
  └── 产出：Xenogerm物品（含xenotypeName + iconDef + GeneSet）

Step 4: Xenogerm植入
  Xenogerm → Pawn
  └── 清除全部旧Xenogene → 添加新基因（全部作为Xenogene）
  └── 设置xenotypeName和iconDef
```

## 遗传机制详解

`PregnancyUtility.GetInheritedGenes(father, mother, success)`的遗传规则：

| 规则 | 说明 |
|------|------|
| **仅Endogene参与遗传** | Xenogene不遗传（除非XenotypeDef.inheritable=true） |
| **双亲都有的基因** | 100%遗传给后代 |
| **仅一方有的基因** | 50%概率遗传 |
| **exclusionTags互斥** | 同标签基因只能遗传一个 |
| **passOnDirectly=false** | 该基因不直接遗传（如纯外观基因） |
| **endogeneCategory=Melanin** | 肤色基因有特殊遗传规则（混合） |

## 代码操作API速查

### 添加基因

```csharp
// 添加单个基因
Gene gene = pawn.genes.AddGene(geneDef, xenogene: true);

// 植入Xenogerm（清除旧Xenogene + 添加新基因）
GeneUtility.ImplantXenogermItem(pawn, xenogerm);

// 再植入（复制Xenogene给目标）
GeneUtility.ReimplantXenogerm(caster, recipient);
```

### 移除基因

```csharp
// 移除单个基因实例
pawn.genes.RemoveGene(gene);

// 设置异种类型（会清除旧Xenogene）
pawn.genes.SetXenotype(xenotypeDef);
```

### 查询基因

```csharp
// 检查是否拥有某基因
bool has = pawn.genes.HasGene(geneDef);

// 获取基因实例
Gene gene = pawn.genes.GetGene(geneDef);

// 遍历所有基因
List<Gene> all = pawn.genes.GenesListForReading;
List<Gene> xeno = pawn.genes.Xenogenes;
List<Gene> endo = pawn.genes.Endogenes;
```

### 基因资源操作

```csharp
// 修改Hemogen值
GeneUtility.OffsetHemogen(pawn, amount);

// 获取资源型基因
Gene_Resource resource = pawn.genes.GetGene(geneDef) as Gene_Resource;
resource.Value = newValue;
```

## 开发者要点

1. **Xenogerm植入是"全替换"**：植入新Xenogerm会清除全部旧Xenogene，这是设计意图——防止基因无限堆叠
2. **AddGene的xenogene参数很重要**：决定基因加入xenogenes还是endogenes列表，影响遗传和替换行为
3. **基因提取有恢复期**：提取后Pawn获得XenogermReplicating Hediff，恢复期12~20天（`GeneTuning.GeneExtractorRegrowingDurationDaysRange`）
4. **遗传仅限Endogene**：模组设计可遗传基因时，应确保基因作为Endogene添加，或设置XenotypeDef.inheritable=true
5. **代码操作绕过物理流程**：`AddGene()`/`RemoveGene()`直接操作基因列表，不需要基因组装器/提取器，适合模组的程序化基因管理
