# 原版派系类型总览

**总览**：RimWorld的派系系统采用三层架构：`FactionDef`（XML定义，描述派系的静态属性）→ `Faction`（运行时实例，持有好感度、领袖、成员等动态状态）→ `FactionManager`（全局管理器，`Find.FactionManager`访问，管理所有派系实例的创建、查询和序列化）。每个存档中的派系由世界生成时根据FactionDef创建，`requiredCountAtGameStart`控制必须生成的数量，`maxConfigurableAtWorldCreation`控制玩家可配置的上限。

## 4. FactionDef关键字段分组

### 4.1 核心字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `isPlayer` | bool | 是否为玩家派系 |
| `humanlikeFaction` | bool | 是否为人形派系（影响Pawn生成、社交等） |
| `hidden` | bool | 是否在世界地图上隐藏（不生成定居点） |
| `techLevel` | TechLevel | 科技等级（Animal/Neolithic/Medieval/Industrial/Spacer/Ultra/Archotech） |
| `categoryTag` | string | 分类标签（Outlander/Tribal/Empire等） |
| `requiredCountAtGameStart` | int | 世界生成时必须创建的数量 |
| `maxConfigurableAtWorldCreation` | int | 玩家可配置的最大数量 |
| `replacesFaction` | FactionDef | 替换的派系（Biotech种族变体用） |

### 4.2 敌对性字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `permanentEnemy` | bool | 永久敌对，好感度锁定-100，不可和解 |
| `naturalEnemy` | bool | 自然敌对，初始好感度-80，可通过外交和解 |
| `permanentEnemyToEveryoneExcept` | List&lt;FactionDef&gt; | 对所有派系永久敌对，除了列表中的例外 |
| `permanentEnemyToEveryoneExceptPlayer` | bool | 对所有非玩家派系永久敌对 |
| `hostileToFactionlessHumanlikes` | bool | 对无派系人形敌对 |
| `mustStartOneEnemy` | bool | 是否必须至少有一个敌对派系（默认true） |

### 4.3 袭击相关字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `raidCommonalityFromPointsCurve` | SimpleCurve | 袭击频率权重曲线（点数→权重） |
| `raidsForbidden` | bool | 完全禁止作为袭击来源 |
| `earliestRaidDays` | float | 最早袭击天数（Mechanoid=45） |
| `canSiege` | bool | 能否执行围攻策略 |
| `canStageAttacks` | bool | 能否执行分阶段进攻 |
| `canPsychicRitualSiege` | bool | 能否执行灵能仪式围攻（Anomaly） |
| `autoFlee` | bool | 是否自动撤退 |

### 4.4 贸易相关字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `caravanTraderKinds` | List&lt;TraderKindDef&gt; | 商队贸易商类型 |
| `visitorTraderKinds` | List&lt;TraderKindDef&gt; | 访客贸易商类型 |
| `baseTraderKinds` | List&lt;TraderKindDef&gt; | 基地贸易商类型 |
| `orbitalTraderKinds` | List&lt;TraderKindDef&gt; | 轨道贸易商类型（Odyssey） |
| `canRequestTraders` | bool | 能否请求贸易商 |
| `canRequestMilitaryAid` | bool | 能否请求军事援助 |
| `canRequestOrbitalTrader` | bool | 能否请求轨道贸易商 |

### 4.5 Pawn生成字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `basicMemberKind` | PawnKindDef | 基础成员PawnKind |
| `pawnGroupMakers` | List&lt;PawnGroupMaker&gt; | Pawn组生成器（Combat/Peaceful/Trader/Settlement等） |
| `backstoryFilters` | List&lt;BackstoryCategoryFilter&gt; | 背景故事过滤器 |
| `xenotypeSet` | XenotypeSet | 异种类型概率集（Biotech） |
| `fixedLeaderKinds` | List&lt;PawnKindDef&gt; | 固定领袖PawnKind |
| `maxPawnCostPerTotalPointsCurve` | SimpleCurve | 单个Pawn最大点数曲线 |

## 5. 抽象模板继承链

```
FactionBase（Core/Defs/FactionDefs/Factions_Base.xml）
  ├── PlayerFactionBase（Abstract）
  │     ├── PlayerColony
  │     ├── PlayerTribe
  │     └── GravshipCrew（Odyssey）
  ├── OutlanderFactionBase（Abstract）
  │     ├── OutlanderCivil
  │     └── OutlanderRoughBase（Abstract）
  │           ├── OutlanderRough
  │           └── OutlanderRoughPig（Biotech, replaces OutlanderRough）
  ├── TribeBase（Abstract）
  │     ├── TribeCivil
  │     ├── TribeRoughBase（Abstract）
  │     │     ├── TribeRough
  │     │     └── TribeRoughNeanderthal（Biotech, replaces TribeRough）
  │     ├── TribeSavageBase（Abstract）
  │     │     ├── TribeSavage
  │     │     └── TribeSavageImpid（Biotech）
  │     ├── TribeCannibal（Ideology）
  │     └── NudistTribe（Ideology）
  ├── PirateBandBase（Name="PirateBandBase"，非Abstract但作为父级）
  │     ├── PirateYttakin（Biotech）
  │     ├── PirateWaster（Biotech）
  │     └── CannibalPirate（Ideology）
  ├── AncientsBase（Abstract）
  │     ├── Ancients
  │     └── AncientsHostile
  ├── Mechanoid
  ├── Insect
  ├── Empire（Royalty）
  ├── TradersGuild（Odyssey）
  ├── Salvagers（Odyssey）
  ├── HoraxCult（Anomaly）
  ├── Entities（Anomaly）
  ├── OutlanderRefugee（Royalty）
  ├── Beggars（Ideology）
  ├── Pilgrims（Ideology）
  └── Sanguophages（Biotech）
```

> **要点**：
> 1. `FactionBase`是所有派系的根模板，仅定义`raidLootValueFromPointsCurve`和`messageDefendersAttacking`
> 2. Biotech种族变体通过`replacesFaction`替换Core派系，继承父级Abstract的所有属性，仅覆盖种族相关字段（xenotypeSet、pawnGroupMakers等）
> 3. `PirateBandBase`（defName=Pirate）既是具体派系又是其他海盗变体的父级——这是一个特殊设计
