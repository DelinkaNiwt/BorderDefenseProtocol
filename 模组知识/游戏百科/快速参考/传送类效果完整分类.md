# 传送类效果完整分类

**总览**：RimWorld的传送/位移效果分为两大技术路径——**PawnFlyer**（物理飞行，Pawn被临时替换为飞行Thing沿轨迹移动）和**CompAbilityEffect_Teleport**（灵能瞬移，直接修改Position无飞行过程）。在此基础上，按作用范围分三层：**地图内**（同一张地图上的位移）、**跨地图**（世界地图层的远距离传送）、**地图间传送门**（连接两张地图的双向通道）。

## 1. 两大技术路径对比

| 维度 | PawnFlyer（物理飞行） | CompAbilityEffect_Teleport（灵能瞬移） |
|------|---------------------|--------------------------------------|
| **核心类** | `PawnFlyer : Thing` | `CompAbilityEffect_Teleport : CompAbilityEffect_WithDest` |
| **移动方式** | Pawn被DeSpawn→替换为PawnFlyer Thing→沿抛物线/直线飞行→到达后RespawnPawn | 直接修改`target.Thing.Position = destination.Cell` |
| **飞行过程** | 有（可见飞行轨迹+阴影） | 无（瞬间传送） |
| **Job** | `CastJump`（专用JobDef） | `CastAbilityOnThingMelee`等（通用能力Job） |
| **Verb** | `Verb_CastAbilityJump`（→`Verb_Jump`装备版） | `Verb_CastAbility`（通用） |
| **眩晕** | `PawnFlyerProperties.stunDurationTicksRange`（着陆后） | `CompProperties_AbilityTeleport.stunTicks`（传送后） |
| **携带物品** | `flyWithCarriedThing`支持携带倒地者 | 不支持 |
| **视觉效果** | 飞行特效+着陆特效+阴影 | Skip入口/出口特效+音效 |
| **体型限制** | 无（任何Pawn可飞） | `maxBodySize`（默认3.5，EntitySkip=10） |
| **DLC** | Core（基础设施） | Royalty（灵能系统） |

## 2. 地图内传送效果完整列表

### 2.1 PawnFlyer系列（物理飞行，6种）

| # | 效果 | DLC | 来源 | 核心Verb/Ability | 射程 | 冷却 | 消耗 | 特殊行为 |
|---|------|-----|------|-----------------|------|------|------|---------|
| 1 | **跳跃包** | Royalty | 装备（Apparel_PackJump） | `Verb_Jump` | LOS | 无 | Chemfuel×20/次，5次 | 可携带倒地者 |
| 2 | **基因长跳** | Biotech | 基因能力（Longjump） | `Verb_CastAbilityJump` | 19.9格 | 60 ticks | Hemogen 0.05 | 基因赋予，无限使用 |
| 3 | **机械体跳跃** | Biotech | 机械体能力（LongjumpMech） | `Verb_CastAbilityJump` | 9.9格 | 480 ticks | 无 | minRange=5.9，不携带物品 |
| 4 | **机械体跳跃发射器** | Biotech | 机械体能力（LongjumpMechLauncher） | `Verb_CastAbilityJump` | 15格 | 20000 ticks(8h) | 2次充能 | 可携带物品，专用特效 |
| 5 | **吞噬跳跃** | Anomaly | Devourer能力（ConsumeLeap） | `Verb_CastAbilityConsumeLeap` | 9.9格 | 3600 ticks | 无 | 着陆后开始消化目标 |
| 6 | **肉兽/深渊门涌出** | Anomaly | PitGate/PitBurrow生成 | 代码直接调用`PawnFlyer.MakeFlyer` | — | — | — | `PawnFlyer_Stun`，着陆后眩晕 |

**PawnFlyer核心流程**：
```
Verb_CastAbilityJump.TryCastShot()
  → JumpUtility.DoJump(pawn, target, comp, verbProps)
    → PawnFlyer.MakeFlyer(flyingDef, pawn, destCell, ...)
      ├── pawn.DeSpawn(WillReplace)           ← Pawn从地图消失
      ├── innerContainer.TryAdd(pawn)          ← Pawn存入飞行容器
      └── GenSpawn.Spawn(pawnFlyer, cell, map) ← 飞行Thing出现
    → PawnFlyer.TickInterval()
      ├── RecomputePosition()                  ← 计算当前位置（抛物线）
      └── ticksFlying >= ticksFlightTime?
          → RespawnPawn()                      ← 着陆：恢复Pawn+征召状态+Job队列
```

**PawnFlyerProperties配置字段**：

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `flightSpeed` | float | 12f | 飞行速度（格/秒） |
| `flightDurationMin` | float | 0.5f | 最短飞行时间（秒） |
| `heightFactor` | float | 2f | 飞行高度因子 |
| `stunDurationTicksRange` | IntRange | Zero | 着陆后眩晕时间 |
| `workerClass` | Type | PawnFlyerWorker | 飞行轨迹计算器 |
| `progressCurve` | SimpleCurve | null | 自定义飞行进度曲线 |
| `shadowTexPath` | string | SkyfallerShadowCircle | 阴影贴图 |

**原版PawnFlyer ThingDef**（3个）：
- `PawnFlyer`（Core）：标准飞行，无眩晕
- `PawnFlyer_Stun`（Anomaly）：着陆后眩晕（肉兽涌出用）
- `PawnFlyer_ConsumeLeap`（Anomaly）：吞噬跳跃专用

### 2.2 灵能Skip系列（瞬间传送，4种）

| # | 效果 | DLC | AbilityDef | 等级 | 射程 | 目标模式 | 眩晕 | 消耗 |
|---|------|-----|-----------|------|------|---------|------|------|
| 1 | **Chaos Skip** | Royalty | `ChaosSkip` | 3 | 24.9格 | RandomInRange(6.9~24.9) | 70~120 ticks | 熵18+灵能0.02 |
| 2 | **Skip** | Royalty | `Skip` | 4 | 27.9格 | Selected（玩家选择） | 18~60 ticks | 熵25+灵能0.02 |
| 3 | **Entity Skip** | Anomaly | `EntitySkip` | — | 99.9格 | Selected（无LOS） | 无 | 无 |
| 4 | **Unnatural Corpse Skip** | Anomaly | `UnnaturalCorpseSkip` | — | 99.9格 | Selected（无LOS） | 无 | 无，CD 1200 ticks |

**CompAbilityEffect_Teleport.Apply()核心流程**：
```csharp
public override void Apply(LocalTargetInfo target, LocalTargetInfo dest)
{
    LocalTargetInfo destination = GetDestination(dest);  // 根据destination模式计算目标位置
    // 视觉特效：Skip_Entry + Skip_Exit
    target.Thing.Position = destination.Cell;            // ★ 核心：直接修改Position
    if (target.Thing is Pawn pawn2)
    {
        pawn2.stances.stunner.StunFor(Props.stunTicks.RandomInRange, ...);  // 传送后眩晕
        pawn2.Notify_Teleported();                       // 通知传送（更新寻路缓存等）
        SendSkipUsedSignal(pawn2.Position, pawn2);       // 发送信号（任务系统可监听）
    }
}
```

**CompProperties_AbilityTeleport配置字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `destination` | AbilityEffectDestination | 目标模式：Selected/Caster/RandomInRange |
| `requiresLineOfSight` | bool | 是否需要视线 |
| `range` | float | 目标位置最大距离（-1=无限） |
| `randomRange` | FloatRange | RandomInRange模式的随机范围 |
| `stunTicks` | IntRange | 传送后眩晕时间 |
| `maxBodySize` | float | 最大体型限制（默认3.5） |
| `destClamorType` | ClamorDef | 目标位置噪音类型 |
| `destClamorRadius` | int | 目标位置噪音半径 |

**AbilityEffectDestination枚举**（3值）：
- `Selected`：玩家选择目标位置（Skip使用）
- `Caster`：传送到施法者位置
- `RandomInRange`：随机传送到范围内位置（ChaosSkip使用）

### 2.3 Entity Skip（异常实体专用传送）

Entity Skip是Anomaly DLC异常实体（如Nociosphere）的专用传送能力，与Royalty的灵能Skip共用`CompAbilityEffect_Teleport`但配置差异巨大：

| 维度 | Royalty Skip | Anomaly EntitySkip |
|------|-------------|-------------------|
| 射程 | 27.9格 | 99.9格（几乎无限） |
| 视线要求 | 需要LOS | 不需要LOS |
| 眩晕 | 18~60 ticks | 无眩晕 |
| 体型限制 | 3.5 | 10（几乎无限制） |
| 消耗 | 熵+灵能 | 无消耗 |
| 冷却 | 无（受熵限制） | 180~480 ticks |
| Verb | Verb_CastAbility | Verb_EntitySkip（自定义） |

`Verb_EntitySkip`的特殊行为：重写`TryCastShot()`在施法前播放Skip特效和音效，然后调用标准能力施放流程。

**NociosphereUtility.SkipTo()**：Nociosphere（异常球体）使用`SkipUtility`直接调用EntitySkip能力进行战术传送——逃离威胁、追踪目标、随机位移。

## 3. 跨地图传送机制（3种）

| # | 机制 | DLC | 核心类 | 方向 | 燃料 | 距离 | 可重用 |
|---|------|-----|--------|------|------|------|--------|
| 1 | **运输舱** | Core | `CompLaunchable` → `TravellingTransporters` | 单向 | Chemfuel 2.25/格 | ~66格 | 否（发射后销毁） |
| 2 | **穿梭机** | Royalty/Odyssey | `CompShuttle` + `TransportShip` | 双向 | 帝国无限/玩家Chemfuel 3/格 | 70/62格 | 帝国否/玩家是 |
| 3 | **Farskip** | Royalty | `CompAbilityEffect_Farskip` | 单向 | Psyfocus 0.7 | 无限（需盟友信标） | 是（能力） |

### 3.1 Farskip（灵能远距传送）

Farskip是唯一的灵能跨地图传送——将施法者及周围4.9格内的所有Pawn传送到世界地图上任意有盟友的地点。

**核心限制**：
- 目标地图必须有至少1个友方Pawn（作为"导航信标"）
- 传送后出现在目标地图随机友方Pawn附近
- 消耗Psyfocus 0.7（极高，几乎清空）
- 5级灵能（需要Knight+等级）
- 不能传送Lodger（任务NPC），否则任务失败

**CompAbilityEffect_Farskip.Apply()核心**：
```
选择世界地图目标格 → 检查目标地图有友方Pawn
  → 收集施法者周围4.9格内所有Pawn
    → 对每个Pawn: 播放Skip特效 → 传送到目标地图友方Pawn附近
      → 眩晕18~60 ticks → 解雾（如果是玩家Pawn）
```

## 4. 地图间传送门（2种）

| # | 机制 | DLC | 核心类 | 连接方式 | 双向 | 生成地图 |
|---|------|-----|--------|---------|------|---------|
| 1 | **MapPortal** | Anomaly | `MapPortal : Building` | 主地图↔口袋地图 | 是 | PocketMap（小型独立地图） |
| 2 | **PitGate** | Anomaly | `Building_PitGate : Building` | 主地图↔深渊地图 | 是 | Undercave（地下洞穴） |

### 4.1 MapPortal（传送门建筑）

`MapPortal`继承自`Building`，是连接两张地图的双向通道。Pawn通过右键菜单"进入"传送门，被传送到另一张地图的对应出口位置。

**核心机制**：
- `GeneratePocketMap()`：首次进入时生成口袋地图（PocketMapParent）
- `GetOtherMap()`：获取连接的另一张地图
- `GetDestinationLocation()`：获取目标位置（出口位置）
- `OnEntered(Pawn)`：Pawn进入后的处理（自动征召等）
- 支持装载物品（`leftToLoad`列表+`ITab_ContentsMapPortal`）

**MapPortalProperties配置**：
- `autoDraftOnEnter`：进入后自动征召
- `enterTex`/`cancelEnterTex`：进入/取消按钮贴图

## 5. 模组扩展传送系统的技术路径

| # | 需求 | 推荐方案 | 复杂度 | 说明 |
|---|------|---------|--------|------|
| 1 | 地图内物理跳跃 | 自定义PawnFlyer子类 + AbilityDef(jobDef=CastJump) | 中 | 重写RespawnPawn()添加着陆效果 |
| 2 | 地图内瞬间传送 | AbilityDef + CompProperties_AbilityTeleport | 低 | 纯XML，配置destination/range/stunTicks |
| 3 | 跨地图传送 | CompLaunchable + 自定义TransportersArrivalAction | 中~高 | 参考运输舱/穿梭机模式 |
| 4 | 灵能远距传送 | 自定义CompAbilityEffect子类 | 高 | 参考Farskip，需处理跨地图Pawn转移 |
| 5 | 地图间传送门 | MapPortal子类 + PocketMapParent | 高 | 参考Anomaly传送门 |
| 6 | 装备赋予跳跃 | Verb_Jump + CompApparelReloadable | 低 | 参考跳跃包，纯XML可配 |

**关键扩展点**：
- `PawnFlyer.RespawnPawn()`（virtual）：着陆后行为，可重写添加伤害/Hediff/特效
- `ICompAbilityEffectOnJumpCompleted`：跳跃完成回调接口，AbilityComp实现此接口可在着陆时执行自定义逻辑
- `CompAbilityEffect_Teleport.SendSkipUsedSignal()`：传送信号，任务系统可通过QuestPart监听
