# 原版机械族单位总览

**总览**：原版共约25个机械族单位，分布在3个DLC中——Core 7个、Biotech 17个、Odyssey 1个。所有机械体共享`BaseMechanoid`抽象ThingDef，核心特征：`fleshType=Mechanoid`（非有机体，无食物/休息需求）、免疫毒素和极端温度、通过`MechWeightClassDef`分4级（Light/Medium/Heavy/UltraHeavy）。Biotech DLC引入机械师控制系统后，所有机械体额外获得`CompOverseerSubject`（监管）和`CompMechRepairable`（可修复）组件。

## 1. BaseMechanoid共性定义

所有机械体ThingDef继承自`BaseMechanoid`抽象模板（Core定义），共享以下属性：

### 1.1 种族特征（race字段）

| 字段 | 值 | 说明 |
|------|-----|------|
| `fleshType` | Mechanoid | 非有机体（isOrganic=false），无出血、无腐烂 |
| `needsRest` | false | 无休息需求 |
| `hasGenders` | false | 无性别 |
| `foodType` | None | 无食物需求 |
| `bloodDef` | Filth_MachineBits | 受伤掉落机械碎片而非血液 |
| `lifeExpectancy` | 2500 | 极长寿命 |
| `canFlyInVacuum` | true | 可在真空中飞行（Odyssey太空场景） |

### 1.2 共享Stat基础值

| Stat | 值 | 说明 |
|------|-----|------|
| `Flammability` | 0 | 不可燃 |
| `ToxicResistance` | 1 | 完全免疫毒素 |
| `ToxicEnvironmentResistance` | 1 | 完全免疫毒性环境 |
| `ArmorRating_Heat` | 2.00 | 极高热能护甲（实际上免疫火焰伤害） |
| `ComfyTemperatureMin/Max` | -100 / 250 | 极端温度耐受 |
| `PsychicSensitivity` | 0.5 | 半灵敏度（SuperHeavy为0.75） |
| `MeatAmount` | 0 | 不产出肉 |
| `BandwidthCost` | 1 | 基础带宽消耗（各单位可覆盖） |
| `WorkSpeedGlobal` | 0.5 | 全局工作速度（Biotech） |

### 1.3 共享Comp组件

| # | Comp | 类 | 职责 |
|---|------|-----|------|
| 1 | **休眠/激活** | `CompMechanoid`（继承CompCanBeDormant） | 管理激活/停用/休眠状态 |
| 2 | **唤醒** | `CompWakeUpDormant` | 受伤唤醒（30格检测半径） |
| 3 | **监管** | `CompOverseerSubject`（Biotech） | 机械师控制关系、失控倒计时（60000 ticks）、野化（MTB 10天） |
| 4 | **可修复** | `CompMechRepairable`（Biotech） | 允许机械师修复 |

> **关键发现**：Core机械体（无Biotech）只有Comp 1-2，是纯敌对单位。Biotech添加Comp 3-4后才成为可控单位。这意味着模组可以通过选择性添加Comp来控制机械体的"可控性"。

## 2. 重量级分类体系

### 2.1 四级MechWeightClassDef

| 重量级 | defName | 可复活 | 典型体型 | 典型护甲(Sharp/Blunt) | 典型速度 | 带宽 |
|--------|---------|--------|---------|---------------------|---------|------|
| **Light** | Light | ✅ | 0.7 | 0.20 / 0.10 | 3.4 | 1 |
| **Medium** | Medium | ✅ | 1.0 | 0.40 / 0.20 | 4.7 | 1-3 |
| **Heavy** | Heavy | ✅ | 1.6~3.5 | 0.40~0.72 / 0.20~0.22 | 1.9~2.1 | 3-4 |
| **UltraHeavy** | UltraHeavy | ❌ | 3.6~4.0 | 0.75 / 0.25 | 1.6~3.2 | 5 |

> **关键发现**：`canResurrect=false`是UltraHeavy的核心限制——Boss机械体不可被Apocriton的`ResurrectionMech`能力复活，这是游戏平衡的关键约束。

### 2.2 抽象模板继承链

```
BaseMechanoid (Core) ─── 根抽象，定义所有机械体共性
  ├── BaseMechanoidWalker (Core) ─── Medium级，MoveSpeed=4.7, Sharp=0.40
  │     ├── Lancer, Scyther, Pikeman (Core)
  │     ├── Legionary, Tesseron (Biotech)
  │     └── Cyclops (Odyssey)
  ├── MechCentipede (Core) ─── 机械蜈蚣家族，Sharp=0.72, HealthScale=4.32
  │     └── CentipedeBlaster, CentipedeGunner, CentipedeBurner (Core)
  ├── Mech_Termite (Core) ─── 直接继承BaseMechanoid
  ├── LightMechanoid (Biotech) ─── Light级，MoveSpeed=3.4, Sharp=0.20
  │     └── Lifter, Constructoid, Fabricor, Agrihand, Cleansweeper,
  │         Militor, WarUrchin, Scorcher, Paramedic (Biotech)
  └── HeavyMechanoid (Biotech) ─── Heavy级，MoveSpeed=1.9, Sharp=0.40
        ├── Tunneler (Biotech)
        └── SuperHeavyMechanoid (Biotech) ─── UltraHeavy级，Sharp=0.75
              └── Apocriton, Centurion, Diabolus, WarQueen (Biotech)
```

> **关键发现**：Core机械蜈蚣（MechCentipede）直接继承BaseMechanoid而非HeavyMechanoid——因为Core早于Biotech。机械蜈蚣的Sharp=0.72远高于HeavyMechanoid模板的0.40，是Core时代的"重型"设计。Biotech引入分级体系后，机械蜈蚣被归类为Heavy但保留了原始高护甲。

## 6. DLC分布与XML文件组织

| DLC | 文件路径 | 单位数 | 抽象模板 |
|-----|---------|--------|---------|
| **Core** | `Core/Defs/ThingDefs_Races/Races_Mechanoid.xml` | 7 | BaseMechanoid, BaseMechanoidWalker, MechCentipede |
| **Biotech Light** | `Biotech/Defs/ThingDefs_Races/Races_Mechanoids_Light.xml` | 9 | LightMechanoid |
| **Biotech Medium** | `Biotech/Defs/ThingDefs_Races/Races_Mechanoids_Medium.xml` | 2 | 继承BaseMechanoidWalker |
| **Biotech Heavy** | `Biotech/Defs/ThingDefs_Races/Races_Mechanoids_Heavy.xml` | 1 | HeavyMechanoid |
| **Biotech SuperHeavy** | `Biotech/Defs/ThingDefs_Races/Races_Mechanoids_SuperHeavy.xml` | 4 | SuperHeavyMechanoid |
| **Odyssey** | `Odyssey/Defs/ThingDefs_Races/Races_Mechanoids_Medium.xml` | 1 | 继承BaseMechanoidWalker |

**ThingDef与PawnKindDef的关系**：每个机械体有两个Def——ThingDef定义种族（身体、护甲、速度、Comp），PawnKindDef定义个体变体（战斗力、武器、外观、AI行为）。同一ThingDef可有多个PawnKindDef变体（如3种蜈蚣共享MechCentipede ThingDef）。
