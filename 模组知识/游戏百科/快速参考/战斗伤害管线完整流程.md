## 一、伤害管线总览

RimWorld的伤害管线是一条从攻击发起到伤害最终化的7阶段流水线。一次攻击（无论近战还是远程）从Verb发起，经过命中/闪避判定、投射物飞行、DamageInfo创建、TakeDamage入口、PreApplyDamage拦截链、DamageWorker伤害应用（含护甲计算）、FinalizeAndAddInjury（含即死保护），最终到PostApplyDamage后处理（死亡判定、附加Hediff、通知链）。

整条管线提供了**10个伤害拦截/修改点**，分布在阶段3~7中，模组可以通过XML配置或C#代码在这些点上介入伤害流程。

## 二、7阶段管线详解

### 阶段1：攻击发起（Verb层）

**近战攻击**（`Verb_MeleeAttack.TryCastShot`，RimWorld命名空间）：

```
TryCastShot()
  → GetNonMissChance(thing) → 命中判定（失败→Miss）
  → GetDodgeChance(thing) → 闪避判定（成功→Dodge）
  → ApplyMeleeDamageToTarget(target) → 创建DamageInfo → target.TakeDamage(dinfo)
```

关键源码：
```csharp
// Verb_MeleeAttack.TryCastShot
if (Rand.Chance(GetNonMissChance(thing)))
{
    if (!Rand.Chance(GetDodgeChance(thing)))
    {
        // 命中且未闪避 → 应用伤害
        DamageWorker.DamageResult damageResult = ApplyMeleeDamageToTarget(currentTarget);
    }
    else { /* 闪避 */ }
}
else { /* 未命中 */ }
```

近战伤害由`Verb_MeleeAttackDamage.DamageInfosToApply`生成DamageInfo，支持多次伤害（如双刃武器）。

**远程攻击**（`Verb_LaunchProjectile.TryCastShot`，Verse命名空间）：

```
TryCastShot()
  → Projectile projectile = GenSpawn.Spawn(...)  // 生成投射物
  → projectile.Launch(caster, ...)               // 发射
  → 投射物飞行（Tick驱动）
  → Projectile.CheckForFreeInterceptBetween()    // 途中拦截检查
  → 到达目标 → Impact(hitThing)
```

远程攻击的命中判定在投射物飞行阶段完成（`Projectile.CheckForFreeIntercept`），而非Verb层。

### 阶段2：投射物命中与DamageInfo创建

**Bullet.Impact**（RimWorld命名空间）：

```csharp
// Bullet.Impact(Thing hitThing, bool blockedByShield)
DamageInfo dinfo = new DamageInfo(
    base.DamageDef,           // 伤害类型
    DamageAmount,             // 伤害量
    ArmorPenetration,         // 穿甲值
    ExactRotation.eulerAngles.y, // 攻击角度
    launcher,                 // 攻击者
    null,                     // hitPart（未指定，由DamageWorker选择）
    equipmentDef,             // 武器ThingDef
    DamageInfo.SourceCategory.ThingOrUnknown,
    intendedTarget.Thing      // 预期目标
);
hitThing.TakeDamage(dinfo);  // → 进入阶段3
```

**DamageInfo结构体关键字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `Def` | `DamageDef` | 伤害类型定义（可被护甲半减时修改为Blunt） |
| `Amount` | `float` | 当前伤害量（沿管线逐步修改） |
| `ArmorPenetrationInt` | `float` | 穿甲值 |
| `HitPart` | `BodyPartRecord` | 命中部位（可为null，由DamageWorker选择） |
| `Instigator` | `Thing` | 攻击发起者 |
| `Weapon` | `ThingDef` | 使用的武器 |
| `IgnoreArmor` | `bool` | 是否忽略护甲 |
| `IgnoreInstantKillProtection` | `bool` | 是否忽略即死保护 |
| `InstantPermanentInjury` | `bool` | 是否为即时永久伤 |

### 阶段3：TakeDamage入口

**Thing.TakeDamage**（Verse命名空间）——所有伤害的统一入口：

```csharp
// Thing.TakeDamage(DamageInfo dinfo)
// 1. 前置检查
if (Destroyed || dinfo.Amount == 0f) return;

// 2. 【拦截点①】ThingDef.damageMultipliers
if (def.damageMultipliers != null)
    foreach (var mult in def.damageMultipliers)
        if (mult.damageDef == dinfo.Def)
            dinfo.SetAmount(dinfo.Amount * mult.multiplier);

// 3. PreApplyDamage拦截链 → 阶段4
PreApplyDamage(ref dinfo, out absorbed);
if (absorbed) return;

// 4. DamageWorker应用 → 阶段5
DamageWorker.DamageResult result = dinfo.Def.Worker.Apply(dinfo, this);

// 5. 攻击者侧通知（Hediff.Notify_PawnDamagedThing）
if (dinfo.Instigator is Pawn pawn)
    foreach (Hediff hediff in pawn.health.hediffSet.hediffs)
        hediff.Notify_PawnDamagedThing(this, dinfo, result);

// 6. PostApplyDamage → 阶段7
PostApplyDamage(dinfo, result.totalDamageDealt);
```

### 阶段4：PreApplyDamage拦截链

PreApplyDamage是一条三层调用链，每层都可以修改DamageInfo或完全吸收伤害：

```
Thing.TakeDamage() → PreApplyDamage(ref dinfo, out absorbed)
  │
  ▼
Pawn.PreApplyDamage()                          ← 第1层：倍率修改
  ├── ② genes.FactorForDamage(dinfo)           ← GeneDef.damageFactors (连乘)
  ├── ③ health.FactorForDamage(dinfo)          ← HediffStage.damageFactors (连乘)
  │   dinfo.Amount *= (基因factor × Hediff factor)
  │
  ├── base.PreApplyDamage() ─────────────────→ 第2层：Comp拦截
  │   ▼
  │   ThingWithComps.PreApplyDamage()
  │     └── for each Comp:
  │         ④ PostPreApplyDamage(ref dinfo, out absorbed)
  │            │
  │            ├── CompShield: isRanged||isExplosive → absorbed=true
  │            │                近战 → 放行
  │            │
  │            └── absorbed? ──Yes──→ return [伤害被吸收]
  │
  └── health.PreApplyDamage() ───────────────→ 第3层：服装吸收+通知
      ▼
      Pawn_HealthTracker.PreApplyDamage()
        ├── for each Apparel:
        │   ⑤ CheckPreAbsorbDamage(dinfo)
        │      └── absorbed? ──Yes──→ return [护盾腰带吸收]
        │
        ├── stance.Notify_DamageTaken(dinfo)
        ├── stunner.Notify_DamageApplied(dinfo)  ← 眩晕判定
        ├── faction.Notify_MemberTookDamage()
        ├── guilt.Notify_Guilty()                ← 攻击者内疚
        └── TaleRecorder.RecordTale(Wounded)
```

**Pawn.PreApplyDamage**（Verse命名空间）：

```csharp
public override void PreApplyDamage(ref DamageInfo dinfo, out bool absorbed)
{
    float num = 1f;
    // 【拦截点②】基因伤害倍率
    if (ModsConfig.BiotechActive && genes != null)
        num *= genes.FactorForDamage(dinfo);  // GeneDef.damageFactors
    // 【拦截点③】Hediff伤害倍率
    num *= health.FactorForDamage(dinfo);      // HediffStage.damageFactors
    dinfo.SetAmount(dinfo.Amount * num);

    base.PreApplyDamage(ref dinfo, out absorbed); // → ThingWithComps
    if (!absorbed)
        health.PreApplyDamage(dinfo, out absorbed);
}
```

**Pawn_GeneTracker.FactorForDamage**（RimWorld命名空间）：
- 遍历所有Active基因的`GeneDef.damageFactors`
- 匹配当前DamageDef → 连乘所有匹配的factor
- 结果缓存在`cachedDamageFactors`字典中

**HediffSet.FactorForDamage**（Verse命名空间）：
- 遍历所有Hediff的`CurStage.damageFactors`
- 匹配当前DamageDef → 连乘所有匹配的factor
- 结果缓存在`cachedDamageFactors`字典中

**ThingWithComps.PreApplyDamage**（Verse命名空间）：

```csharp
public override void PreApplyDamage(ref DamageInfo dinfo, out bool absorbed)
{
    base.PreApplyDamage(ref dinfo, out absorbed);
    if (absorbed || comps == null) return;
    // 【拦截点④】遍历所有ThingComp
    for (int i = 0; i < comps.Count; i++)
    {
        comps[i].PostPreApplyDamage(ref dinfo, out absorbed);
        if (absorbed) break;  // 任一Comp吸收即终止
    }
}
```

**CompShield.PostPreApplyDamage**（RimWorld命名空间）——装备护盾：

```csharp
public override void PostPreApplyDamage(ref DamageInfo dinfo, out bool absorbed)
{
    absorbed = false;
    if (ShieldState != ShieldState.Active || PawnOwner == null) return;
    if (dinfo.Def == DamageDefOf.EMP)
    {
        energy = 0f; Break();  // EMP直接击破
    }
    else if (!dinfo.Def.ignoreShields && (dinfo.Def.isRanged || dinfo.Def.isExplosive))
    {
        energy -= dinfo.Amount * Props.energyLossPerDamage;
        if (energy < 0f) Break();
        else AbsorbedDamage(dinfo);
        absorbed = true;  // 远程/爆炸伤害被完全吸收
    }
    // 注意：近战伤害不被CompShield拦截
}
```

**Pawn_HealthTracker.PreApplyDamage**（Verse命名空间）——护盾腰带检查：

```csharp
// 【拦截点⑤】服装吸收检查
if (pawn.apparel != null && !dinfo.IgnoreArmor)
{
    List<Apparel> wornApparel = pawn.apparel.WornApparel;
    for (int i = 0; i < wornApparel.Count; i++)
    {
        if (wornApparel[i].CheckPreAbsorbDamage(dinfo))
        {
            absorbed = true; return;  // 护盾腰带完全吸收
        }
    }
}
// 后续：通知stance、stunner、faction、guilt、tale等
```

### 阶段5：DamageWorker伤害应用

**DamageWorker_AddInjury.ApplyDamageToPart**（Verse命名空间）：

```csharp
private void ApplyDamageToPart(DamageInfo dinfo, Pawn pawn, DamageResult result)
{
    // 1. 确定命中部位
    BodyPartRecord part = GetExactPartFromDamageInfo(dinfo, pawn);
    dinfo.SetHitPart(part);

    // 2. 【拦截点⑥】护甲计算
    float num = dinfo.Amount;
    if (!dinfo.InstantPermanentInjury && !dinfo.IgnoreArmor)
    {
        DamageDef damageDef = dinfo.Def;
        num = ArmorUtility.GetPostArmorDamage(pawn, num, dinfo.ArmorPenetrationInt,
            dinfo.HitPart, ref damageDef, out deflectedByMetalArmor, out diminishedByMetalArmor);
        dinfo.Def = damageDef;  // 半减时锐器→钝器
    }

    // 3. 【拦截点⑦】IncomingDamageFactor
    if (dinfo.Def.ExternalViolenceFor(pawn))
        num *= pawn.GetStatValue(StatDefOf.IncomingDamageFactor);

    // 4. 伤害为0 → 偏转
    if (num <= 0f) { result.deflected = true; return; }

    // 5. 应用伤害
    if (!dinfo.AllowDamagePropagation)
        FinalizeAndAddInjury(pawn, num, dinfo, result);  // → 阶段6
    else
        ApplySpecialEffectsToPart(pawn, num, dinfo, result); // 各Worker重写
}
```

**ArmorUtility.GetPostArmorDamage**（Verse命名空间）——逐层护甲判定：

```csharp
// 1. 遍历覆盖命中部位的服装（从外到内）
foreach (Apparel apparel in wornApparel.Reverse())
    if (apparel.def.apparel.CoversBodyPart(part))
        ApplyArmor(ref amount, armorPenetration, apparel.GetStatValue(armorRatingStat), ...);
        if (amount < 0.001f) return 0f;  // 完全偏转

// 2. 最后检查Pawn自身护甲（如机械体天然护甲）
ApplyArmor(ref amount, armorPenetration, pawn.GetStatValue(armorRatingStat), null, ...);
```

**ArmorUtility.ApplyArmor**（Verse命名空间）——单层护甲公式：

```csharp
private static void ApplyArmor(ref float damAmount, float armorPenetration,
    float armorRating, Thing armorThing, ref DamageDef damageDef, Pawn pawn, out bool metalArmor)
{
    // 护甲磨损：被击中的护甲受到原始伤害25%的损伤
    if (armorThing != null)
        armorThing.TakeDamage(new DamageInfo(damageDef, damAmount * 0.25f));

    float effectiveArmor = Mathf.Max(armorRating - armorPenetration, 0f);
    float roll = Rand.Value;

    if (roll < effectiveArmor * 0.5f)
        damAmount = 0f;                    // 完全偏转
    else if (roll < effectiveArmor)
    {
        damAmount = damAmount / 2f;        // 半减
        if (damageDef.armorCategory == Sharp)
            damageDef = DamageDefOf.Blunt;  // 锐器→钝器
    }
    // else: 全额穿透
}
```

**护甲判定决策树**（单层护甲的三种结果）：

```
  输入: damAmount, armorRating, armorPenetration
  │
  │  effectiveArmor(eA) = max(armorRating - armorPenetration, 0)
  │  roll = Rand.Value (0~1)
  │  护甲磨损: armorThing.TakeDamage(damAmount × 0.25)
  │
  ▼
  roll < eA × 0.5 ?
  ├── Yes ──→ ┌──────────────────────────┐
  │           │ 完全偏转                  │
  │           │ damAmount = 0             │
  │           │ (伤害归零，攻击无效)       │
  │           └──────────────────────────┘
  │
  └── No ──→ roll < eA ?
              ├── Yes ──→ ┌──────────────────────────┐
              │           │ 半减                      │
              │           │ damAmount /= 2            │
              │           │ Sharp → Blunt (锐器转钝器) │
              │           └──────────────────────────┘
              │
              └── No ──→  ┌──────────────────────────┐
                          │ 全额穿透                  │
                          │ damAmount 不变             │
                          │ (护甲未能阻挡)             │
                          └──────────────────────────┘

  注: 逐层判定时，每层服装独立执行上述流程，最后检查Pawn自身护甲
```

### 阶段6：伤害最终化（FinalizeAndAddInjury）

**DamageWorker_AddInjury.FinalizeAndAddInjury**（Verse命名空间）——两个重载：

**重载1**（float totalDamage）：创建Hediff_Injury实例

```csharp
protected float FinalizeAndAddInjury(Pawn pawn, float totalDamage, DamageInfo dinfo, DamageResult result)
{
    if (pawn.health.hediffSet.PartIsMissing(dinfo.HitPart)) return 0f;

    HediffDef hediffDef = HealthUtility.GetHediffDefFromDamage(dinfo.Def, pawn, dinfo.HitPart);
    Hediff_Injury injury = (Hediff_Injury)HediffMaker.MakeHediff(hediffDef, pawn);
    injury.Part = dinfo.HitPart;
    injury.sourceDef = dinfo.Weapon;
    injury.Severity = totalDamage;

    // 社交打架不摧毁部位
    if (instigator?.CurJobDef == JobDefOf.SocialFight)
        injury.destroysBodyParts = false;

    return FinalizeAndAddInjury(pawn, injury, dinfo, result); // → 重载2
}
```

**重载2**（Hediff_Injury injury）：即死保护 + 添加Hediff

```csharp
protected float FinalizeAndAddInjury(Pawn pawn, Hediff_Injury injury, DamageInfo dinfo, DamageResult result)
{
    injury.TryGetComp<HediffComp_GetsPermanent>()?.PreFinalizeInjury();
    float partHealth = pawn.health.hediffSet.GetPartHealth(injury.Part);

    // 【拦截点⑧】即死保护（仅殖民者 + 外部暴力 + 难度允许）
    if (pawn.IsColonist && !dinfo.IgnoreInstantKillProtection
        && dinfo.Def.ExternalViolenceFor(pawn)
        && !Rand.Chance(Find.Storyteller.difficulty.allowInstantKillChance))
    {
        // 最多尝试7次降低Severity，直到不会致死
        for (int i = 0; i < 7; i++)
        {
            if (!pawn.health.WouldDieAfterAddingHediff(injury)) break;
            injury.Severity = Mathf.Clamp(partHealth - threshold, min, max);
            threshold *= 2f;  // 每次阈值翻倍
        }
    }

    // 最终添加Hediff_Injury
    pawn.health.AddHediff(injury, null, dinfo, result);

    // DamageDef.additionalHediffsThisPart（同部位附加Hediff）
    if (!def.additionalHediffsThisPart.NullOrEmpty() && !partIsMissing)
        foreach (HediffDef item in def.additionalHediffsThisPart)
            pawn.health.AddHediff(HediffMaker.MakeHediff(item, pawn, injury.Part));

    return Mathf.Min(injury.Severity, partHealth);
}
```

### 阶段7：PostApplyDamage后处理

**Pawn_HealthTracker.PostApplyDamage**（Verse命名空间）：

```csharp
public void PostApplyDamage(DamageInfo dinfo, float totalDamageDealt)
{
    // 1. 死亡判定
    if (ShouldBeDead())
    {
        if (!ShouldBeDeathrestingOrInComa())
            pawn.Kill(dinfo);       // 死亡
        else
            ForceDeathrestOrComa(); // Biotech死眠/再生昏迷替代死亡
        return;
    }

    // 2. 【拦截点⑨】DamageDef.additionalHediffs（附加Hediff）
    if (dinfo.Def.additionalHediffs != null)
    {
        foreach (var addHediff in dinfo.Def.additionalHediffs)
        {
            float severity = (addHediff.severityFixed > 0f)
                ? addHediff.severityFixed
                : totalDamageDealt * addHediff.severityPerDamageDealt;
            // 可选：按体型反比缩放、按Stat缩放
            Hediff hediff = HediffMaker.MakeHediff(addHediff.hediff, pawn);
            hediff.Severity = severity;
            AddHediff(hediff, null, dinfo);
            if (Dead) return;  // 附加Hediff可能致死
        }
    }

    // 3. 【拦截点⑩】通知所有Hediff
    for (int j = 0; j < hediffSet.hediffs.Count; j++)
        hediffSet.hediffs[j].Notify_PawnPostApplyDamage(dinfo, totalDamageDealt);

    // 4. Job通知（可能中断当前Job）
    if (pawn.Spawned && dinfo.CheckForJobOverride)
        pawn.jobs.Notify_DamageTaken(dinfo);
}
```

**攻击者侧通知**（在Thing.TakeDamage中，阶段3之后）：

```csharp
// 攻击者的所有Hediff收到通知
if (dinfo.Instigator is Pawn pawn)
    foreach (Hediff hediff in pawn.health.hediffSet.hediffs)
        hediff.Notify_PawnDamagedThing(this, dinfo, damageResult);
```

## 四、近战 vs 远程攻击流程对比

| 维度 | 近战 | 远程 |
|------|------|------|
| **Verb类** | `Verb_MeleeAttack` → `Verb_MeleeAttackDamage` | `Verb_LaunchProjectile` → `Verb_Shoot` |
| **命中判定** | Verb层：`GetNonMissChance` + `GetDodgeChance` | 投射物层：飞行偏移 + `CheckForFreeIntercept` |
| **DamageInfo创建** | `DamageInfosToApply`（可多次伤害） | `Bullet.Impact`（单次主伤害 + ExtraDamages） |
| **护盾交互** | CompShield不拦截近战 | CompShield拦截远程/爆炸 |
| **后续流程** | 相同（TakeDamage → ... → PostApplyDamage） | 相同 |

## 五、护盾机制对比

| 维度 | CompShield（装备护盾） | CompProjectileInterceptor（投射物拦截器） |
|------|----------------------|----------------------------------------|
| **拦截阶段** | 阶段4（PostPreApplyDamage） | 投射物飞行阶段（Projectile.CheckForFreeIntercept） |
| **拦截范围** | 仅穿戴者 | 范围内所有友方 |
| **拦截类型** | 远程+爆炸（不拦截近战） | 投射物（可配置是否拦截友方） |
| **能量机制** | `energy -= amount × energyLossPerDamage` | `currentHitPoints -= amount` |
| **EMP弱点** | EMP直接击破（energy=0） | 无EMP弱点 |
| **典型实例** | 护盾腰带（ShieldBelt） | 低护盾（LowShieldPack） |

## 六、HediffComp伤害相关虚方法

| 方法 | 触发时机 | 典型实现 |
|------|---------|---------|
| `Notify_PawnPostApplyDamage(DamageInfo, float)` | 受伤后（阶段7） | `HediffComp_ReactOnDamage`：EMP→植入体故障 |
| `Notify_PawnDamagedThing(Thing, DamageInfo, DamageResult)` | 攻击者造成伤害后（阶段3后） | 攻击者侧Hediff反应 |

**HediffComp_ReactOnDamage**（Verse命名空间）：
- `damageDefIncoming`：指定触发的伤害类型（如EMP）
- `createHediff`：触发时创建的Hediff（如植入体故障）
- `chance`：触发概率
