# 文档类别

# 用户确认状态
- [用户已确认]：置信度高，可参考和引用。**仅用户可填**
- [用户未确认]：置信度低，参考或引用前需经过用户确认

# 文档编辑状态
- [未完成]：当前版本编写未完成
- [已完成]：当前版本已写完

# 文档锁定状态
- [已锁定]：此文档有效，**AI禁止对内容做修改**
- [未锁定]：此文档有效，AI可以直接对原文修改
- [已废弃]：此文档无效，**禁止参考**
- [已归档]：此文档过时，**不建议参考**
