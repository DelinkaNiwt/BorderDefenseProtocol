### 4. 变式标识实现

#### 标识样式示例

**方向标识** - 建筑/物体
```

┌─────────────────┐
│  256×256        │
│                 │
│     North       │  ← 中央显示方向
│                 │
└─────────────────┘
```

**体型+方向标识** - 人物
```

│  128×256        │
│                 │
│  Male_East      │  ← 显示体型和方向
│                 │
└─────────────────┘
```

**变体标识** - 植物

│                 │
└─────────────────┘
```

```
贴图显示：
