## 🧪 修改前后代码示例

### 修改1：激活费用机制

**修改前**：
```csharp
public void Activate(CompTrion comp)
{
    state = MountState.Active;  // 直接激活
}
```

**修改后**：
```csharp
public void Activate(CompTrion comp)
{
    // 检查激活费用
    if (def.activationCost > 0 && comp.Available < def.activationCost)
    {
        Messages.Message("Trion不足");
        return;
    }
    // 支付费用
    comp.Consume(def.activationCost);
    // 进入导引
    state = MountState.Activating;
    guidanceTicksRemaining = def.activationGuidanceTicks;
}
```

### 修改2：状态机

**修改前**：
```csharp
bool IsActive;  // 二进制
```

**修改后**：
```csharp
enum MountState { Disconnected, Dormant, Activating, Active }
MountState state;

bool IsReserving => state >= Dormant && state <= Active;
bool IsActive => state == MountState.Active;
```

### 修改4：Debuff施加

**修改前**：
```csharp
public void OnCombatBodyDestroyed(CompTrion comp, DestroyReason reason)
{
    // 仅日志记录
}
```

**修改后**：
```csharp
public void OnCombatBodyDestroyed(CompTrion comp, DestroyReason reason)
{
    if (reason == DestroyReason.BailOutSuccess)
    {
        var depletionDef = DefDatabase<HediffDef>.GetNamed(
            "ProjectTrion_TrionDepletion", false);
        comp.pawn.health.AddHediff(depletionDef);
    }
}
```

## 🔍 常见问题排查

### 问题1：激活后立即关闭
**原因**：Tick()被多次调用，导引倒计时快速递减
**解决**：确保Tick()每60Tick调用一次，或改为手动管理倒计时

### 问题2：Debuff不显示
**原因**：HediffDef定义有误或路径错误
**解决**：检查defName是否与代码中的名称一致

### 问题3：手臂断连不生效
**原因**：VitalPartUtil.IsArm()判定有误
**解决**：检查body part def的命名规则

### 问题4：OutputPower为0
**原因**：Strategy返回null或计算有误
**解决**：检查GetInitialTalent()是否正确初始化天赋
