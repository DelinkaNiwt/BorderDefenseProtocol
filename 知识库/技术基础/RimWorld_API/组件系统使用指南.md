## 核心概念

### 组件系统架构
RimWorld使用组件系统为游戏对象（Thing）添加功能。每个组件由两部分组成：

1. **ThingComp** - 功能逻辑实现
2. **CompProperties** - 配置参数定义

## 基础实现模板

### 1. 定义组件属性类

```csharp
using Verse;

public class CompProperties_YourFeature : CompProperties
{
    // 配置参数
    public SoundDef activationSound;
    public float cooldownSeconds = 5f;
    public List<MoteDef> visualEffects;
    public bool requiresPower = false;
    
    // 构造函数必须设置compClass
    public CompProperties_YourFeature()
    {
        compClass = typeof(CompYourFeature);
    }
    
    // 配置验证方法
    public override IEnumerable<string> ConfigErrors(ThingDef parentDef)
    {
        foreach (var error in base.ConfigErrors(parentDef))
            yield return error;
            
        if (cooldownSeconds <= 0)
            yield return "cooldownSeconds必须大于0";
    }
}
```

### 2. 实现组件逻辑类

```csharp
using Verse;

public class CompYourFeature : ThingComp
{
    // 状态变量
    private int lastActivationTick;
    private bool isActive = false;
    
    // 属性访问器
    private CompProperties_YourFeature Props => (CompProperties_YourFeature)props;
    
    // 初始化方法
    public override void Initialize(CompProperties props)
    {
        base.Initialize(props);
        // 初始化逻辑
    }
    
    // 每帧更新（性能敏感）
    public override void CompTick()
    {
        base.CompTick();
        
        // 使用间隔检查避免性能问题
        if (Find.TickManager.TicksGame % 60 == 0) // 每秒检查一次
        {
            UpdateFeature();
        }
    }
    
    // 功能逻辑
    private void UpdateFeature()
    {
        if (CanActivate())
        {
            Activate();
        }
        else if (isActive && ShouldDeactivate())
        {
            Deactivate();
        }
    }
    
    // 激活条件检查
    private bool CanActivate()
    {
        if (!isActive && 
            Find.TickManager.TicksGame - lastActivationTick > Props.cooldownSeconds * 60)
        {
            return CheckAdditionalConditions();
        }
        return false;
    }
    
    // 附加条件检查（可重写）
    protected virtual bool CheckAdditionalConditions()
    {
        // 检查电源需求
        if (Props.requiresPower)
        {
            var powerComp = parent.GetComp<CompPowerTrader>();
            return powerComp?.PowerOn ?? false;
        }
        return true;
    }
    
    // 激活功能
    private void Activate()
    {
        isActive = true;
        lastActivationTick = Find.TickManager.TicksGame;
        
        // 播放声音效果
        if (Props.activationSound != null)
        {
            Props.activationSound.PlayOneShot(new TargetInfo(parent.Position, parent.Map));
        }
        
        // 显示视觉效果
        if (Props.visualEffects != null)
        {
            foreach (var moteDef in Props.visualEffects)
            {
                MoteMaker.MakeStaticMote(parent.Position, parent.Map, moteDef);
            }
        }
        
        OnActivated();
    }
    
    // 停用功能
    private void Deactivate()
    {
        isActive = false;
        OnDeactivated();
    }
    
    // 可重写的激活/停用事件
    protected virtual void OnActivated() { }
    protected virtual void OnDeactivated() { }
    
    // 检查是否应该停用
    private bool ShouldDeactivate()
    {
        // 实现停用条件逻辑
        return true;
    }
    
    // 保存/加载状态
    public override void PostExposeData()
    {
        base.PostExposeData();
        Scribe_Values.Look(ref lastActivationTick, "lastActivationTick", 0);
        Scribe_Values.Look(ref isActive, "isActive", false);
    }
}
```

## XML配置示例

### 在Defs/YourDefs/YourFeatureDef.xml中

```xml
<?xml version="1.0" encoding="utf-8"?>
<Defs>
  <YourNamespace.CompProperties_YourFeature>
    <defName>YourFeature_Standard</defName>
    <label>标准功能</label>
    <description>这是一个标准功能配置</description>
    
    <!-- 配置参数 -->
    <activationSound>Sound_Activation</activationSound>
    <cooldownSeconds>5.0</cooldownSeconds>
    <requiresPower>true</requiresPower>
    
    <visualEffects>
      <li>Mote_Sparkle</li>
      <li>Mote_Glow</li>
    </visualEffects>
  </YourNamespace.CompProperties_YourFeature>
</Defs>
```

### 在ThingDef中引用组件

```xml
<ThingDef ParentName="BuildingBase">
  <defName>YourBuilding</defName>
  <comps>
    <li Class="YourNamespace.CompProperties_YourFeature">
      <compClass>YourNamespace.CompYourFeature</compClass>
      <activationSound>Sound_YourActivation</activationSound>
      <cooldownSeconds>3.0</cooldownSeconds>
    </li>
  </comps>
</ThingDef>
```

## 最佳实践

### 性能优化
1. **避免每帧复杂计算**：使用间隔检查（如每60tick检查一次）
2. **缓存计算结果**：对于不频繁变化的数据进行缓存
3. **使用事件驱动**：只在状态变化时执行逻辑

### 错误处理
1. **配置验证**：重写ConfigErrors方法验证配置
2. **空值检查**：对所有可能为null的引用进行检查
3. **异常处理**：使用try-catch处理可能的异常

### 兼容性考虑
1. **版本兼容**：考虑不同RimWorld版本的API变化
2. **模组兼容**：避免与其他模组的组件冲突
3. **保存兼容**：确保保存/加载逻辑的向后兼容
