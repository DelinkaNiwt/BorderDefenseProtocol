# 模组工程

本目录存储进行中的模组项目。

## 项目结构参考

```
模组名/
├─ About/              # 模组基本信息（作者、描述等）
├─ docs/               # 工作流文档（9步阶段产出）
│  ├─ 草稿/            # 草稿和临时文件
│  ├─ 1_创意方案.md
│  ├─ 2_需求和系统设计.md
│  ├─ 3_知识获取清单.md
│  ├─ 4_设计文档_验证清单.md
│  ├─ 5_原型代码.md
│  ├─ 6_测试报告.md
│  ├─ 7_正式版.md
│  └─ 8_经验总结.md
├─ 1.5/               # RimWorld 1.5 版本目录
│  ├─ Assemblies/
│  ├─ Defs/
│  ├─ Languages/
│  └─ Textures/
├─ 1.6/               # RimWorld 1.6 版本目录
├─ content/           # 版本间共享内容
└─ LoadFolders.xml   # 加载配置
```

## 工作流指导

1. 复制"示例"目录，改名为实际项目名
2. 按 9 步工作流填充 docs 中的文档
3. 游戏资源按版本放在对应目录
4. 项目完成后，将第8步的经验总结提交到 `模组知识/经验教训/开发总结/`
