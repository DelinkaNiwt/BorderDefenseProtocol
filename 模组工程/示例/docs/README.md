# 项目文档

此目录存储项目全生命周期的工作流文档（对应 9 步工作流）。

## 文件说明

- `1_创意方案.md` - 创意总监的创意方案文档
- `2_需求和系统设计.md` - 需求规格和系统设计
- `3_知识获取清单.md` - 项目所需知识和参考
- `4_设计文档_验证清单.md` - 最终设计和 RiMCP 验证清单
- `5_原型代码.md` - 脏原型开发记录
- `6_测试报告.md` - 测试和完善阶段报告
- `7_正式版.md` - 正式版代码和文档
- `8_经验总结.md` - 项目经验总结（供知识库提炼）
- `草稿/` - 临时草稿和讨论

## 使用

按工作流阶段依次填充。每个文档都是项目当前阶段的产出。
