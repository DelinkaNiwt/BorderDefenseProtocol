# RimTrion 项目工作流

本目录包含RimTrion项目的工作流和所有Agent的工作成果。

## 项目信息

查看 `项目上下文/项目元数据.yaml` 了解项目基本信息。

查看 `项目上下文/工作流状态.md` 了解当前工作进度。

## 工作流阶段

这个项目通过7个Agent的协作完成开发：

1. **头脑风暴官** - 创意和概念设计
2. **需求分析官** - 需求提取和清单整理
3. **架构设计官** - 系统架构和技术方案
4. **原型验证官** - 概念验证和原型开发
5. **开发实现官** - 完整代码实现
6. **调试修复官** - 问题排查和修复
7. **更新维护官** - 版本管理和长期维护

详见：[Agent工作流指南](../../工作区指导规范/Agent工作流指南.md)

## 项目成果

### 项目上下文 (项目上下文/)

- `项目元数据.yaml` - 项目基本信息和元数据
- `工作流状态.md` - 各阶段进度状态
- `版本日志.md` - 版本更新历史

### Agent工作 (智能体工作/)

7个Agent目录，每个包含：
- `状态.yaml` - Agent的工作状态
- `输出.md` - Agent的主要输出
- `RimWorld查询/` - RiMCP查询记录（可选）

### 交付物 (交付物/)

- `模组包/` - 编译后的模组文件
- `发布说明.md` - 用户可读的发布说明
- `使用指南.md` - 用户使用指南

### 项目事件 (.claude/事件/)

项目中重要事件和决策的记录。

## 使用此目录

### 查看项目进度

```bash
# 查看当前阶段和进度
cat 项目上下文/工作流状态.md

# 查看版本历史
cat 项目上下文/版本日志.md
```

### 查看某个Agent的工作

```bash
# 例如，查看架构设计的输出
cat 智能体工作/03_架构设计Agent/输出.md

# 查看架构验证报告
cat 智能体工作/03_架构设计Agent/架构验证报告.md
```

### 跟踪项目历史

```bash
# 查看项目的重要事件
ls .claude/事件/
```
