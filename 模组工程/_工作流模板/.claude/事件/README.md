# Claude事件记录

本目录记录项目中由Claude AI系统（主Agent和各子Agent）发生的关键事件和决策。

## 目录用途

- 记录项目的关键节点和事件
- 保存Agent工作流的决策历史
- 用于项目回溯和经验总结

## 文件命名规范

**格式**：`YYYYMMDD_HHMM_事件描述.md`

**示例**：
- `20260123_1400_项目初始化.md`
- `20260123_1530_Agent1完成概念框架.md`
- `20260124_1000_用户反馈修改.md`

## 事件类型

### 项目事件
- 项目初始化
- 项目激活
- 项目完成

### 工作流事件
- Agent启动
- Agent完成
- 用户反馈

### 决策事件
- 关键决策
- 方案选择
- 风险识别

## 事件记录模板

```markdown
# [事件名称]

**时间**：[YYYY-MM-DD HH:MM]
**类型**：[项目/工作流/决策]
**涉及Agent**：[Agent名称或NA]

## 事件概述

[事件的简要描述]

## 关键内容

[详细的事件内容]

## 决策和影响

[做出的决策和影响]

## 下一步

[后续的行动]
```
