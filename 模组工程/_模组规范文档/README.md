## 🎯 使用建议

### 我是新手，应该读什么？
1. 先读【快速启动指南】了解基础（10分钟）
2. 查看【示例模组】理解实践（15分钟）
3. 需要详细说明时查询具体规范

### 我要创建新模组，应该怎么办？
1. 选择合适的模板（轻量/标准/完整）
2. 参考【模组目录结构标准规范】组织目录
3. 参考【模组命名规范】进行命名
4. 参考【About.xml详细配置指南】配置metadata
5. 参考【Defs分类规范】组织定义文件

### 我遇到问题，应该查哪个文档？
- ❓ "我不知道文件应该放哪里" → **目录结构标准规范**
- ❓ "我不知道如何命名" → **模组命名规范**
- ❓ "我不知道About.xml怎么写" → **About.xml详细配置指南**
- ❓ "我不知道Defs如何分类" → **Defs分类规范**
- ❓ "我想快速开始" → **快速启动指南**
