**调研结论**：

```xml

  <HintPath>C:\NiwtDatas\Projects\RimworldModStudio\模组工程\ProjectTrion_Framework\1.6\Assemblies\ProjectTrion.dll</HintPath>
</Reference>
```

**理由**：
- ✓ 符合业界实践（WeaponFitting → AncotLibrary）
