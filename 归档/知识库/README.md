## 基于分析的数据来源

本知识库基于对以下六个框架模组的深入分析：

1. **AutoBlink_自动折跃** - 组件化架构和配置驱动
2. **HumanoidAlienRaces_外星人框架** - 条件驱动图形系统
3. **VanillaExpandedFramework_原版扩展框架** - 模块化插件架构
4. **WeaponFitting** - 武器配件系统和依赖注入
5. **AdaptiveStorageFramework** - 自适应存储系统
6. **AncotLibrary** - 基础库和工具集
