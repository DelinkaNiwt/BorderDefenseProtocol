# 归档

本目录存储已完成、弃用或过时的项目和文件。

## 清理标准

文件进入此目录的情况：
- ❌ 项目已完成或被放弃
- ❌ 旧版本文档或过时内容
- ❌ 临时生成的中间文件（整理后）
- ❌ 待整理的历史文件

## 整理指南

此目录中的文件需要逐步整理：

1. **有价值的内容** → 转移到 `模组知识/` 或 `模组工程/`
2. **历史参考** → 保留在此，建立索引
3. **无用文件** → 删除

## 子目录建议

- `旧规范/` - 过时的工作流和规范文档
- `参考/` - 参考项目和资源
- `其他/` - 其他历史文件
