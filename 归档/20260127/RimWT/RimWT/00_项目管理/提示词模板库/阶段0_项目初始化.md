# 项目上下文

我正在开发一个基于《境界触发者》的RimWorld大型模组。现在处于**阶段0：项目基础建设**。

## 项目核心理念
1. **体验先行** - 先明确想要什么感受，再设计实现机制
2. **知识驱动** - 建立完整知识体系，确保AI理解一致
3. **渐进迭代** - 分阶段产出，每阶段有明确验收标准
4. **玩家视角** - 用非专业人士能理解的方式把控项目

## 项目愿景
- 引入一套全新的以境界触发者为灵感的全新游戏玩法和体系
- Trion为底层系统的多样游戏内容（能量/物品/能力/建筑/Trion兵机械体/特性/袭击/派系/AI逻辑/载具） 
- 感受Trion能量分配和触发器配置系统的策略深度（强防御=弱攻击，多全能=欠专精，高爆发=低续航，等多种对“资源分配调度”的权衡取舍）
- 体验生成战斗体后全新的战斗体系和战斗模式，感受战斗体损伤不带回真身的安全感
- 体验原作中多样触发器能力的沉浸感，设计属于自己原创的全新触发器的成就感
- 在RimWorld中体验如原作面对入侵的近界民强敌般带来的压迫感和挑战感
