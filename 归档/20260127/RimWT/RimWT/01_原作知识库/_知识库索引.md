# 知识库索引

## 使用说明

本文件是 `01_原作知识库/` 的导航和管理工具。

- **状态说明：**
  - ✅ 已完成 - 文档已创建并初始化
  - 🔄 进行中 - 正在编写或补充
  - ⭕ 待建立 - 计划中但还未开始
  - ❓ 待评估 - 不确定是否需要

- **优先级说明：**
  - 🔴 核心 - 必须包含，直接影响核心玩法
  - 🟠 重要 - 应该包含，影响设计决策
  - 🟡 次要 - 可选，补充背景知识
  - 🔵 扩展 - 后续版本可考虑

## 知识库完整性检查清单

**阶段1完成时应满足的条件：**

### 核心机制 - 必须包含的内容
- [ ] Trion的定义、产生、消耗、个体差异、20岁停止发育
- [ ] 战斗体的激活、损伤规则、与真身的关系
- [ ] 触发器的配置规则、槽位限制、主副手、组合技
- [ ] 副作用的分级（C/B/A/S）和具体描述
- [ ] Trion兵的分类（单独型/团队型/自律型）和各国代表
- [ ] BORDER的等级制度（S/A/B/C级）
- [ ] 战斗职位体系（攻击手/射手/枪手/狙击手/全能手）
- [ ] 门的作用、开启条件、战术应用

### 触发器图鉴 - 必须包含的触发器
- [ ] BORDER制式触发器：弧月、光魂、蝎子、猎犬、小行星、流星、鹭、朱鹮、闪电、护盾、护城盾、蓑衣虫、变色龙、蚱蜢等
- [ ] BORDER组合技：战斧、螺旋钻、火蜥蜴、眼镜蛇、黄蜂等
- [ ] 黑触发器：游真触发器（印系统）、风刃、Organon、Alektor、Vorvoros等
- [ ] 近界民触发器：Lampyris、Chelidon、Despinis、Servitora等

### 战斗体验档案 - 必须识别的体验
- [ ] 维扎般的"强敌压迫感"（实力差距+不可预测+持续威胁）
- [ ] 配置触发器的"策略深度"（有限资源+多样选择+明显trade-off）
- [ ] 游真的"爽快战斗"（连招+即时反馈）
- [ ] "安全感"体验（战斗体损伤不影响真身）
- [ ] 其他关键体验（根据用户反馈补充）

### 敌对势力档案 - 必须包含的信息
- [ ] Aftokrator的触发器、成员、战斗特征、压迫感来源
- [ ] Galopoula的触发器、成员、战斗特征、战术模式
- [ ] 其他势力的基本信息（如需要）
