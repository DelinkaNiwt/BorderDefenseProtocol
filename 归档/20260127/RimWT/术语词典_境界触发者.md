## 核心概念

| 中文名 | 英文名 | 别名 | 描述 |
|--------|--------|------|------|
| Trion | Trion | 三离子 / 触发力 / 触力能 | 生命体内生物能量，心脏附近Trion腺体产生，触发器能量源。可锻炼增强，20岁后停止发育 |
| 触发器 | Trigger | - | 释放Trion能量的装置，生成Trion体、变身并提供攻击能力。BORDER成员和近界民的核心战斗装备 |
| 战斗体 | Trion Body | Trion体 | Trion能量构成的人工身体，激活触发器生成。超人强度/速度/耐久力，被摧毁后物理身体不受伤害，无法修复，只能重新生成 |
| 副作用 | Side Effect | - | 高Trion量影响大脑/感官而显现的人类能力延伸。分为C级(增强感官)/B级(特殊体质)/A级(超人能力)/S级(超感知觉)四级 |
| Trion兵 | Trion Soldier | 三离子兵 / 触力兵 | 近界民为战斗制造的武器，Trion构成的自动战斗单位。有单独型/团队战术型，炮击/战斗/捕获等功能特化 |
| 近界 | Neighborhood | - | 异世界总称，由黑暗中沿固定轨道运行的惑星国家组成。触发器技术支撑近界文明 |
| 近界民 | Neighbor | - | 近界住民。地球世界称呼的"近界民"多指为战斗制造的Trion兵，实际近界民具有人类外观 |
| 门 | Gate | - | 连接地球与近界的传送门，开启异次元通道。近界民通过门入侵三门市 |

## 触发器分类

**触发器类型树形结构示意：**

```
触发器 (Trigger)
├── 普通触发器 (Normal Trigger)
│   ├── BORDER制式触发器 (BORDER Standard Trigger)
│   │   ├── 制式触发器 (Trigger)
│   │   └── 试验性触发器 (Experimental Trigger)
│   └── 近界民触发器 (Neighbor Trigger)
│       （近界民触发器可按国家/组织分类，如Aftokrator、Galopoula等）
└── 黑触发器 (Black Trigger)
```

**触发器分类详细说明：**

| 中文名 | 英文名 | 别名 | 描述 |
|--------|--------|------|------|
| 普通触发器 | Normal Trigger | - | 常规触发器类型，含BORDER制式触发器和近界民触发器(非黑触发器)。使用者Trion能量为动力源 |
| 黑触发器 | Black Trigger | - | 杰出Side Effect能力者将生命和能力倾注制作的触发器。性能远超普通触发器，强烈反映制作者人格，与制作者不投缘则无法启动 |
| BORDER制式触发器 | BORDER Standard Trigger | - | 普通触发器子类，BORDER成员的标准化触发器系统。握柄中的芯片决定Trion表现形式，战斗中可切换触发器 |
| BORDER制式触发器组件 | BORDER Trigger Component | - | BORDER制式触发器的可替换模块/芯片，本质为芯片非独立触发器。如弧月/猎犬/蚱蜢/变色龙/紧急脱离等 |
| 近界民触发器 | Neighbor Trigger | - | 普通触发器子类，近界民的触发器系统。非黑触发器则属普通触发器。可按国家/组织分类，如Aftokrator/Galopoula触发器 |
