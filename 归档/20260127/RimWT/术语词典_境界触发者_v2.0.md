### 【核心概念】

#### Trion
- **英文**: Trion
- **别名**: 三离子 / 触发力 / 触力能
- **分类**: 核心概念
- **定义**: 生命体内的生物能量,是触发器系统的能量源
- **说明**: 由心脏附近的Trion腺体产生。可通过锻炼增强,但20岁后停止发育。Trion量决定了触发器的性能上限,高Trion量者可能显现副作用能力。Trion兵、Trion体、触发器武器、防卫建筑设施、远征艇穿梭机等都以Trion为能量基础构建。在近界,Trion质量决定了战斗力的上限。
- **相关**: [[触发器]] [[Trion体]] [[副作用]] [[Trion腺体]] [[Trion兵]]
- **备注**: Border使用数值量化Trion能力。战斗员通常为4-9，已知最高值为雨取千佳的38，最低战斗员值为三云修的2。黑触发器可大幅提升Trion值，如迅悠一使用风刃后从7提升至37

#### 触发器
- **英文**: Trigger
- **别名**: -
- **分类**: 核心概念
- **定义**: 释放Trion能量的装置,生成Trion体并提供攻击能力
- **说明**: 触发器由握柄和内部芯片组成,芯片决定了Trion的表现形式(武器、盾、技能等)。BORDER的制式触发器可替换不同芯片(组件)组合,近界民的触发器多为整体式设计。黑触发器由杰出能力者倾注生命制作,性能远超普通触发器,但与使用者心性不合无法启动。触发器在战斗中至关重要,是现代战斗的基本工具。
- **相关**: [[Trion]] [[Trion体]] [[黑触发器]] [[BORDER制式触发器]]
- **备注**: 触发器激活后立即生成Trion体,物理身体则不受伤害,Trion体被摧毁后需重新激活才能再战

#### 战斗体
- **英文**: Trion Body
- **别名**: Trion体
- **分类**: 核心概念
- **定义**: Trion能量构成的人工身体,激活触发器生成
- **说明**: 需要装备并激活触发器才可生成。战斗体具有超人的强度、速度和耐久力,远超物理身体的极限。被摧毁后物理身体不受伤害,但无法修复,只能重新激活触发器生成新的。
- **相关**: [[触发器]] [[Trion]] [[物理身体]]
- **备注**: 战斗体可以持续作战只要Trion充足,但持续激活会消耗大量Trion

#### 副作用
- **英文**: Side Effect
- **别名**: 特殊能力 / 人类能力延伸
- **分类**: 核心概念
- **定义**: 高Trion量影响大脑/感官而显现的人类能力延伸
- **说明**: 副作用按稀有度分为四级（非强度）：C级（强化五感，如增强听力）、B级（特殊体质，如情感感知）、A级（超技能，如动物交流）、S级（超感觉，如预知未来、看穿谎言）。分类反映稀有程度而非能力强弱。副作用是人类能力，非触发器能力，无法通过触发器复制。S级副作用非常稀有,已知包括看穿谎言(空闲游真)、预知未来(迅悠一)、感知近界民(雨取千佳)。副作用的强弱与个人Trion量和体质有关,无法通过训练直接提升等级。
- **相关**: [[Trion]] [[S级成员]] [[空闲游真]] [[迅悠一]] [[雨取千佳]]
- **备注**: 副作用在战斗中往往能决定关键时刻的胜负,某些S级副作用甚至能改变战局

#### Trion兵
- **英文**: Trion Soldier
- **别名**: 三离子兵 / 触力兵
- **分类**: 核心概念
- **定义**: 近界民为战斗制造的自动战斗单位

### 【触发器分类】

#### 普通触发器
- **英文**: Normal Trigger
- **别名**: -
- **分类**: 触发器分类
- **定义**: 常规触发器类型,使用者Trion能量为动力源
- **说明**: 普通触发器包括BORDER制式触发器和近界民触发器(非黑触发器)。普通触发器的性能主要取决于使用者的Trion量和技巧。BORDER的制式触发器采用模块化设计,可根据需要组合不同的芯片(组件)。近界民的普通触发器多为整体式设计,功能相对固定但往往性能更强。相比黑触发器,普通触发器缺乏高度个性化,但更容易被大规模制造和使用。
- **相关**: [[黑触发器]] [[BORDER制式触发器]] [[近界民触发器]]
- **备注**: 绝大多数战斗员使用普通触发器,黑触发器因为极其稀有所以仅为少数精英专用

#### 黑触发器
- **英文**: Black Trigger
- **别名**: -
- **分类**: 触发器分类
- **定义**: 杰出能力者将生命倾注制作的触发器,性能远超普通触发器
- **说明**: 黑触发器由掌握高级触发器制作技术的杰出能力者花费大量时间和Trion倾注生命制作。性能和破坏力都远超普通触发器,有时甚至达到普通触发器无法抗衡的程度。黑触发器强烈反映制作者的个性和能力风格,与心性不投缘的使用者无法启动。黑触发器极其稀有且不可再生,一旦丢失无法补充。已知黑触发器包括风刃、游真的黑触发器、Aftokrator和其他势力的黑触发器等。黑触发器使用者自动获得S级身份。
- **相关**: [[普通触发器]] [[风刃]] [[游真的黑触发器]] [[S级成员]]
- **备注**: Aftokrator等近界国家已知拥有数个黑触发器,其数量代表国家战力的重要指标

#### BORDER制式触发器
- **英文**: BORDER Standard Trigger
- **别名**: 制式触发器
- **分类**: 触发器分类
- **定义**: BORDER成员的标准化可替换模块触发器系统
- **说明**: BORDER制式触发器由握柄和内部芯片组成,握柄中的芯片决定Trion表现形式。使用者可根据战术需要自由组合多个不同功能的芯片(组件)。主手Main Trigger最多4个芯片槽位,副手Sub Trigger也最多4个槽位。每侧同时最多使用1个芯片,两侧可同时使用,最大并用2个组件。战斗中可快速切换芯片调整战术。制式触发器是BORDER的核心装备,标准化和模块化设计使其易于制造、维护和升级。
- **相关**: [[BORDER]] [[触发器组件]] [[试验性触发器]]
- **备注**: 不同等级的BORDER成员触发器配置不同,B级及以上自动配备雷达、通讯、紧急脱离等系统级组件

#### 试验性触发器
- **英文**: Experimental Trigger
- **别名**: -
- **分类**: 触发器分类
- **定义**: 玉狛支部特制,爆发力强但稳定性较低
- **说明**: 试验性触发器由玉狛支部结合近界民技术为特定使用者定制。外形和配置理念与制式触发器相似,但具有更多芯片槽位以支持更多组件同时激活,从而获得更强的爆发力。降低稳定性和持久性为代价,使用者长期使用可能导致Trion体不稳定甚至自毁(如烏丸京介的情况)。试验性触发器是玉狛支部的特殊研发产品,体现了他们更激进的技术路线。
- **相关**: [[BORDER制式触发器]] [[玉狛支部]] [[烏丸京介]]

#### 近界民触发器
- **英文**: Neighbor Trigger
- **别名**: -
- **分类**: 触发器分类
- **定义**: 近界民的触发器系统,多为整体式设计
- **说明**: 近界民的触发器系统与BORDER制式触发器差异很大。近界民触发器多采用整体式设计,功能固定但性能往往更强。近界民触发器可按使用者所属国家分类,如Aftokrator、Galopoula、Leoforio等的触发器各有特色。Aftokrator的触发器往往需要配合植入的Trion接收器才能发挥最大性能。Galopoula的触发器采用楔形模块设计,战斗时消耗楔形芯片激活不同功能。近界民触发器的设计理念与BORDER制式触发器截然不同。
- **相关**: [[BORDER制式触发器]] [[Aftokrator触发器]] [[Galopoula触发器]] [[Trion接收器]]

#### 弧月
- **英文**: Kogetsu
- **别名**: -
- **分类**: 触发器组件 > 攻击手武器
- **定义**: 日本刀外观的平衡型全能近战武器
- **说明**: BORDER最早开发的攻击手触发器组件,采用日本刀外观。弧月的攻击力与耐久度均衡,是最全能的近战武器,对初学者和高手都适用。许多攻击手以弧月为主要武器。弧月可搭配旋空能力进行远距离斩击。结构简洁,易于掌握,是攻击手的入门武器,但也被顶级攻击手如立川维持装备。弧月的易用性和稳定性使其经久不衰。
- **相关**: [[攻击手]] [[蝎子]] [[光魂]] [[旋空]]
- **备注**: 弧月有两个可选触发器：旋空（Senkū，延伸刀身）和幻踊（Gen'yō，变形能力）。28+名Border成员使用弧月，是最受欢迎的全能刀刃

#### 蝎子
- **英文**: Scorpion
- **别名**: -
- **分类**: 触发器组件 > 攻击手武器
- **定义**: 轻量级可变形刀剑,多功能但易碎
- **说明**: 蝎子是由迅悠一和Michael Cronin开发的轻量级可变形攻击手武器。标准型为黄色发光单刃刀刃，几乎无重量，可从身体任何部位显现。主要变体：风间队模型（中空刀身设计，专注快速延伸攻击范围）、木虎定制型（双端双刃，可围绕握柄旋转如风车）。蝎子可自由变形为任何形状，灵活度极高，但耐久度低（D级），首次碰撞弧月即会破裂。适合需要灵活多变和奇袭战术的攻击手。16名Border成员使用蝎子。
- **相关**: [[攻击手]] [[弧月]] [[光魂]]
- **备注**: 蝎子的可变形特性是其核心优势,但也导致耐久度下降

#### 光魂
- **英文**: Raygust
- **别名**: -
- **分类**: 触发器组件 > 攻击手武器
- **定义**: 可变形武器,可切换刀刃/盾牌模式
- **说明**: 光魂由握柄和半透明发光Trion构成,由寺岛雷藏以盾和蝎子为基础开发。可切换刀刃模式和盾牌模式。刀刃模式:大型双端军刀外观,攻击力强;盾牌模式:刀身变高大微弯盾牌,大幅提升耐久度降低攻击力,可承受攻击手触发器持续攻击和多发枪手触发器子弹。光魂兼具攻防,是平衡型武器的代表。其可变性和高耐久度使其成为顶级攻击手的选择。光魂的推进器能力可将光魂投射进行远距离打击。
- **相关**: [[攻击手]] [[弧月]] [[蝎子]] [[推进器]] [[寺岛雷藏]]
- **备注**: 光魂是三大攻击手武器中最后开发的,代表了攻击手武器设计的新方向
