# 原作资料整理大纲

> **目的**：从网络上大量无序的原作信息中，有方向性、有轻重点地提取和整理对模组开发有帮助的设定

> **定位**：这是三份大纲的"前置阶段"，为"大纲A：机制概念库"和"大纲B：玩法体验设计"提供原材料

---

## 使用说明

### 资料整理的原则

1. **选择性提取**：只提取对模组开发有价值的信息，不是复制原作百科
2. **标注价值**：每条资料都标注对模组的价值（高/中/低）
3. **记录来源**：注明信息来源（漫画第X话、动画第X集、设定集等）
4. **区分确定性**：标注信息的确定程度（明确/推测/待确认）

### 填充流程

1. 收集原作资料（漫画、动画、设定集、访谈等）
2. 按照本大纲的章节逐一填充
3. 每填充一个小节，标注 [已填充] 或 [待填充]
4. 完成后，基于此资料库填充"大纲A"和"大纲B"
