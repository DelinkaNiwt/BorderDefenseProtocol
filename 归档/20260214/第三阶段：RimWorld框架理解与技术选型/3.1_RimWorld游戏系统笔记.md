## 第1章 Pawn的身体与健康 ★★★

> **模组需求**：战斗体伤害隔离(R16)、Trion接收器植入物(实体2)、弱点系统

### 1.1 健康状况的种类与玩家感知

Pawn身上会出现的健康状况（统称"Hediff"）可分为8大类：

| # | 类别 | 玩家感知 | 典型实例 | 来源DLC |
|---|------|---------|---------|---------|
| 1 | **物理伤害** | 战斗或事故造成的即时伤害，可治疗可自愈 | 切割伤、瘀伤、烧伤、枪伤 | 核心 |
| 2 | **缺失部位** | 部位被摧毁或手术移除，永久性，需假肢替换 | 失去左臂、失去左眼 | 核心 |
| 3 | **植入物/假肢** | 手术安装的人工部件，替换或增强身体功能 | 仿生眼、义肢、启灵连接、止痛器 | 核心 |
| 4 | **疾病/状态** | 随时间发展的病症，需治疗和免疫竞赛 | 流感、瘟疫、哮喘、白内障 | 核心 |
| 5 | **药物相关** | 使用药物后的兴奋/成瘾/宿醉三阶段 | 烟瘾、酒精兴奋、宿醉 | 核心 |
| 6 | **生育相关** | 怀孕和分娩过程 | 怀孕、分娩 | Biotech |
| 7 | **基因/机械相关** | 基因系统和机械师系统产生的特殊状态 | 死眠、化学依赖、频段连接 | Biotech |
| 8 | **异常相关** | 超自然异常状态 | 蹒跚者转化、金属恐怖寄生、拒绝死亡 | Anomaly |

**关键认知**：绝大多数游戏中的疾病、增益、减益都是通过同一套"Hediff"系统实现的——从刀伤到癌症、从仿生臂到心灵连接、从药物兴奋到怀孕，本质上都是附加在Pawn身上的Hediff条目，只是配置不同。

📖 `Hediff分类与继承体系`

### 1.2 植入物的游戏效果

植入物分为两大类，体验截然不同：

**替换型植入物**（假肢/仿生体/超凡体）：
- 直接替换整个身体部位，移除该部位上的所有伤病
- 效果由"部位效率"决定——义肢级(50%)低于自然、仿生级(125%)超越自然、超凡级(150%)远超自然
- 固体材质不会出血
- 替换后子部位全部消失（如替换手臂则手指也消失）

**附加型植入物**（增强器/辅助器）：
- 附加在现有部位上，不替换部位本身
- 通过属性修改提供效果，如：
  - 止痛器：完全消除疼痛感
  - 学习辅助器：提升20%全局学习速度
  - 胃部再处理器：降低25%饥饿速率，并免疫肠道寄生虫
  - 昼夜节律辅助器：降低20%休息需求
  - 快乐线：降低20%脑效率（代价），换取持续快乐

**品质影响**：植入物受品质系统影响，高品质植入物效果更好。

📖 `植入物效果机制`

### 1.3 严重程度与病程演变

每个健康状况都有一个"严重程度"数值，它是驱动一切动态变化的核心：

**严重程度的作用**：
- 驱动**阶段切换**——严重度越高，进入越严重的阶段（如流感从轻微→严重→极度严重）
- 降到0时**自动消失**（如药物效果消退）
- 达到致死阈值时**直接致死**（如失血严重度达到100%）

**7种变化方式**：

| 方式 | 规则 | 典型实例 |
|------|------|---------|
| 每日固定变化 | 以固定速率每日增减 | 兴奋剂效果：每日-0.75（约16小时消退） |
| 免疫竞赛 | 未获免疫时恶化，获得免疫后康复 | 流感：免疫力和病情赛跑 |
| 治疗加速 | 医疗治疗后加速康复，效果取决于治疗质量 | 治疗流感加速康复 |
| 生长模式 | 在生长/稳定/缓解三种模式间切换 | 癌症：可能生长也可能自行缓解 |
| 自愈 | 按固定间隔自动恢复 | 伤口自然愈合 |
| 倒计时消失 | 持续固定时间后直接消失 | 临时增益效果 |
| 外部系统控制 | 由温度、环境等外部系统直接控制 | 低温症/中暑：由环境温度驱动 |

**免疫竞赛**是最经典的玩家决策点：疾病严重度和免疫力同时增长，玩家必须通过治疗、休息、药物来确保免疫力先达到100%，否则Pawn可能死于疾病。

📖 `Severity严重程度机制`

### 1.4 身体部位与受伤规则

RimWorld的身体是一棵**树形结构**，以躯干为根节点向外延伸：

```
躯干(Torso)
├── 头部(Head) ─ 头骨(Skull) ─ 大脑(Brain)
│   ├── 左眼(Eye) / 右眼(Eye)
│   ├── 左耳(Ear) / 右耳(Ear)
│   ├── 鼻子(Nose) / 下颌(Jaw)
│   └── 颈部(Neck)
├── 左肩(Shoulder) ─ 左臂(Arm) ─ 左手(Hand) ─ 手指(Finger)×5
├── 右肩(Shoulder) ─ 右臂(Arm) ─ 右手(Hand) ─ 手指(Finger)×5
├── [内部] 心脏(Heart) / 左肺(Lung) / 右肺(Lung)
├── [内部] 肝脏(Liver) / 左肾(Kidney) / 右肾(Kidney) / 胃(Stomach)
├── [内部] 脊柱(Spine)
├── 左腿(Leg) ─ 左脚(Foot) ─ 脚趾(Toe)×5
└── 右腿(Leg) ─ 右脚(Foot) ─ 脚趾(Toe)×5
```

**三维定位系统**决定了什么武器能打到什么部位：

| 维度 | 值 | 含义 | 典型部位 |
|------|---|------|---------|
| 垂直高度 | 顶部 | 头颈区域 | 头、颈、眼、耳、脑 |
| | 中部 | 躯干区域（默认） | 躯干、肩、臂、手 |
| | 底部 | 下肢区域 | 腿、脚、脚趾 |
| 深度层级 | 外部 | 外部可见（默认） | 躯干、头、臂、腿 |
| | 内部 | 内部器官 | 心脏、肺、脑、肾、肝 |

**伤害分配规则**：
- 每个部位有一个"覆盖率"，决定被命中的概率权重
- 外部部位先被命中，内部器官受外层保护（必须先穿透外层）
- 不同武器类型的伤害传播方式不同：
  - **利器**（刀剑）：命中一个部位，伤害集中
  - **钝器**（锤棒）：伤害向内传播到子部位（如打头可能伤到脑）
  - **刺击**（矛刺）：可直接穿透到内部器官
  - **抓挠**：浅层伤害，不穿透
  - **火焰**：只伤害外部部位

**为什么内部器官更安全**：内部器官（心脏、脑等）被外层部位保护，只有钝器传播和刺击穿透能直接伤害它们。这是RimWorld战斗中"护甲保护核心器官"的基础。

📖 `身体部位与伤害分配系统`

### 1.5 死亡条件与保护机制

Pawn死亡有**5个判定条件**，满足任一即死亡：

| # | 条件 | 说明 | 典型场景 |
|---|------|------|---------|
| 1 | **致死严重度** | 某个健康状况的严重度达到致死阈值 | 失血达100%、疾病恶化到极限 |
| 2 | **致命能力归零** | 5种致命能力中任一降为0 | 意识、血液循环、呼吸、代谢、血液过滤 |
| 3 | **核心部位效率归零** | 躯干效率降为0 | 躯干被完全摧毁 |
| 4 | **累计伤害超阈值** | 所有伤口的严重度总和超过阈值 | 全身多处重伤 |
| 5 | **随机死亡** | 某些状态按概率每日判定死亡 | 极度衰老、某些严重疾病 |

**5种致命能力**（肉体Pawn）：意识、血液循环、呼吸、代谢、血液过滤。机械体只有3种：意识、移动、操作。

**死亡保护机制**：
- **死亡豁免**：某些特殊状态可以完全阻止死亡判定（如Anomaly DLC的"拒绝死亡"）
- **死眠替代**：拥有死眠基因的Pawn在本应死亡时进入死眠状态而非死亡（Biotech DLC）
- **倒地替代死亡**：大多数情况下Pawn会先倒地而非直接死亡

**倒地即死概率**：倒地的Pawn有一定概率直接死亡，概率因阵营不同：
- 玩家殖民者：通常不会倒地即死
- 敌对派系Pawn：有较高的倒地即死概率（减少玩家需要处理的俘虏数量）
- 具体概率受难度设置影响

📖 `Pawn死亡判定与致死方式`、`战斗极端情况与安全网机制`

### 1.6 健康状况的连锁反应

健康状况之间存在丰富的连锁反应，形成动态的健康生态：

**8种常见连锁模式**：

| 模式 | 触发条件 | 结果 | 玩家体验 |
|------|---------|------|---------|
| **出血→失血** | 任何出血伤口 | 持续累积失血严重度 | 必须尽快止血/治疗 |
| **疾病→并发症** | 疾病严重度达到特定阶段 | 触发新的健康状况 | 疾病恶化会雪崩 |
| **年龄→慢性病** | 随年龄增长概率触发 | 白内障、痴呆、关节炎等 | 老年Pawn逐渐衰退 |
| **低温→低温症→冻伤** | 环境温度过低 | 先低温症，严重时冻伤四肢 | 极端天气的生存挑战 |
| **高温→中暑→烧伤** | 环境温度过高 | 先中暑，极端时直接烧伤 | 热浪事件的威胁 |
| **药物→成瘾→戒断** | 反复使用药物 | 成瘾后停用触发戒断症状 | 药物管理的长期代价 |
| **器官衰竭→死亡** | 关键器官效率归零 | 致命能力降为0 | 保护关键器官的重要性 |
| **免疫竞赛** | 感染性疾病 | 免疫力vs病情的赛跑 | 治疗和护理的核心决策 |

**连锁反应的三个配置层级**：
1. **种族层级**：定义在种族上，影响该种族所有个体（如人类的年龄→慢性病）
2. **健康状况层级**：定义在特定健康状况上，该状况存在时触发（如出血→失血）
3. **阶段层级**：定义在特定严重度阶段上，达到该阶段时触发（如疾病恶化到特定阶段→并发症）

📖 `Hediff互相影响与触发机制`

## 第2章 需求与资源管理 ★★★

> **模组需求**：Trion能量(数据属性)、R1腺体产生Trion、R12触发体消耗Trion

### 2.1 Pawn的需求系统

每个Pawn都有一组"需求条"，代表其内在状态。需求分为两大类：

**持续下降型**——需要玩家主动满足：

| 需求 | 下降方式 | 满足方式 | 极端后果 | DLC |
|------|---------|---------|---------|-----|
| 饥饿 | 持续消耗，速率受代谢效率/健康状况/床铺影响 | 进食 | 营养不良→饿死 | 核心 |
| 休息 | 持续消耗，速率受类别因子影响 | 睡觉 | 非自愿入睡 | 核心 |
| 娱乐 | 持续下降，有"娱乐容忍度"机制 | 娱乐活动 | 心情惩罚 | 核心 |
| 户外 | 室内时下降 | 待在室外 | 幽闭恐惧思想 | 核心 |
| 化学品 | 成瘾后持续下降 | 使用对应药物 | 戒断症状 | 核心 |
| 机械能量 | 活跃时10/天，空闲时3/天 | 充电 | 自动关机 | Biotech |
| 死亡休息 | 持续缓慢下降 | 进入死眠 | 强制死眠 | Biotech |
| 杀戮渴望 | 持续下降 | 击杀敌人 | 精神崩溃 | Biotech |
| 压制 | 持续下降 | 看守/演讲 | 奴隶叛乱 | Ideology |

**环境追踪型**——自动追踪环境状态，无需主动满足：

| 需求 | 追踪目标 | 影响 | DLC |
|------|---------|------|-----|
| 心情 | 所有思想的综合效果 | 精神崩溃阈值 | 核心 |
| 美观 | 当前环境美观度 | 心情修正 | 核心 |
| 舒适 | 当前使用家具的舒适度 | 心情修正 | 核心 |
| 房间大小 | 当前房间空间感知 | 心情修正 | 核心 |
| 权威 | 派系领袖的统治力 | 低于阈值影响派系 | Royalty |

**需求的冻结条件**：需求在以下情况下暂停更新——Pawn被暂停（冬眠舱）、睡眠中（部分需求）、精神状态中（部分需求）、不可交互且不可见时。

📖 `Need需求系统总览`

### 2.2 需求极端值与精神崩溃

**精神崩溃系统**是需求极端值最重要的后果：

当心情持续低于阈值超过约33分钟游戏时间后，按概率触发精神崩溃：

| 级别 | 默认阈值 | 平均触发间隔 | 典型崩溃行为 |
|------|---------|------------|------------|
| 小型 | ≈25% | 4天 | 发呆、暴食、隐居 |
| 中型 | ≈14% | 0.8天 | 破坏物品、纵火、社交打架 |
| 大型 | ≈4% | 0.5天 | 杀人、自杀、狂暴 |

**触发机制**：
- 心情低于阈值后开始计时，超过约33分钟后按MTB概率触发
- 心情回到阈值以上则计时重置
- 崩溃恢复后有约6小时冷却期
- 触发时会选择一个负面思想作为"最后一根稻草"显示

**其他极端值后果**：
- 饥饿归零→营养不良健康状况→严重度持续增长→致死
- 休息极低→非自愿入睡（按概率随机倒地睡觉）
- 压制极低→奴隶叛乱（按概率发动暴动）
- 机械能量归零→机械体自动关机

📖 `Need极端值效果与精神崩溃系统`

### 2.3 非Need资源条

除了标准需求系统外，RimWorld还有几种不属于Need但功能类似的资源条：

| 资源 | 所属系统 | 补充方式 | 消耗方式 | DLC |
|------|---------|---------|---------|-----|
| **精神力** | 心灵能力 | 冥想 | 使用心灵能力 | Royalty |
| **心灵熵** | 心灵能力 | 自然冷却 | 使用心灵能力（过热导致倒地） | Royalty |
| **血源素** | 基因系统 | 吸血/进食特定食物 | 持续消耗（基因驱动） | Biotech |
| **带宽** | 机械师系统 | 植入物/装备/建筑提供 | 每个机械体占用固定带宽 | Biotech |

**带宽系统的特殊性**（对RimWT Trion系统有重要参考价值）：
- 带宽是**离散整数**资源，用方块网格可视化
- 多来源自动叠加：植入物、装备、建筑各自贡献带宽，互不耦合
- 预占用模型：培育中的机械体也占用带宽
- 超额处理：带宽不足时自动断开最后分配的机械体

📖 `Need需求系统总览`、`机械师系统与带宽控制体系`

## 第3章 基因系统 ★★★

> **模组需求**：Trion腺体(实体1)、R5 Trion触发副作用

### 3.1 基因的基本规则

基因系统（Biotech DLC）为Pawn提供可遗传、可组装、可植入的属性修改框架。

**两类基因**：

| 维度 | 异种基因(Xenogene) | 内源基因(Endogene) |
|------|-------------------|-------------------|
| 来源 | 异种基因组植入、基因组装器 | 出生时继承、种族默认 |
| 可遗传 | 通常不可（除非异种类型设置为可遗传） | 是 |
| 覆盖规则 | 优先覆盖同互斥标签的内源基因 | 被异种基因覆盖时失效但仍存在 |
| 移除方式 | 植入新异种基因组时清除全部旧异种基因 | 通常不可移除 |

**异种基因组植入的全替换规则**：植入新的异种基因组时，Pawn的所有旧异种基因被**完全清除**，替换为新基因组的全部基因。这是"全有或全无"的设计——不能只替换部分基因。

**互斥标签机制**：基因可以设置互斥标签，同一标签下只能有一个基因生效。当异种基因和内源基因有相同互斥标签时，异种基因优先，内源基因被标记为"已覆盖"（仍存在但不生效）。

**异种类型(Xenotype)**：预设的基因组合模板，如吸血鬼(Sanguophage)、骠骑兵(Hussar)等。定义了该种族包含的所有基因、是否可遗传、战斗力倍率等。

📖 `基因系统结构与继承体系`、`基因获得与移除途径`

### 3.2 基因能提供什么效果

基因可以提供16种以上的效果类型：

| # | 效果类型 | 说明 | 典型实例 |
|---|---------|------|---------|
| 1 | 属性修改 | 加值或倍率修改各种属性 | 移动速度+20%、近战伤害×1.5 |
| 2 | 能力赋予 | 赋予主动使用的能力 | 吸血、毒液攻击 |
| 3 | 能力值修改 | 修改视力、操作等基础能力 | 视力+50%、操作能力-30% |
| 4 | 特质控制 | 强制添加或压制特质 | 强制嗜血特质、压制和平主义 |
| 5 | 需求控制 | 启用或禁用特定需求 | 启用杀戮渴望、禁用娱乐需求 |
| 6 | 外观改变 | 改变皮肤/毛发/体型/毛皮 | 灰色皮肤、猫耳、矮小体型 |
| 7 | 免疫/抗性 | 免疫特定疾病或降低伤害 | 免疫毒气、火焰抗性×0.5 |
| 8 | 疼痛修改 | 增减疼痛感知 | 疼痛+0.1、疼痛×0.5 |
| 9 | 药物相关 | 修改成瘾/耐受/过量概率 | 成瘾概率×0、耐受度×0.5 |
| 10 | 行为限制 | 禁止特定工作或行为 | 禁止暴力工作、不育 |
| 11 | 技能天赋 | 修改技能天赋和热情 | 射击+4天赋、添加热情 |
| 12 | 社交/心理 | 修改社交和精神崩溃行为 | 社交打架×2、周期性精神崩溃 |
| 13 | 资源追踪 | 管理特殊资源（如血源素） | 血源素每日消耗、进食补充 |
| 14 | 特殊行为 | 需要运行时逻辑的特殊效果 | 死眠、凝血、自愈、吸血 |

**关键认知**：绝大多数基因效果通过配置实现，不需要编写代码。只有需要特殊运行时逻辑的效果（如资源追踪、周期性行为）才需要自定义代码。

📖 `基因效果分类与实现机制`

### 3.3 代谢效率的权衡

基因系统通过三个维度约束基因组合，形成"强力效果必有代价"的平衡设计：

**三维约束体系**：

| 维度 | 范围 | 正值含义 | 负值含义 | 影响 |
|------|------|---------|---------|------|
| 代谢效率 | -5 ~ +5 | 效率高→食物消耗少 | 效率低→食物消耗多 | 食物消耗速率 |
| 复杂度 | -5 ~ +5 | 占用配额 | 释放配额（罕见） | 基因总量限制+组装时间 |
| 远古科技 | 0+ | 需要稀有资源 | — | 组装时需投入Archite胶囊 |

**代谢效率-食物消耗曲线**（不对称设计）：

| 代谢效率 | -5 | -3 | -1 | 0(基准) | +1 | +3 | +5 |
|---------|----|----|----|---------|----|----|----|
| 食物消耗倍率 | 2.25× | 1.63× | 1.0× | 1.0× | 0.9× | 0.7× | 0.5× |

负代谢的惩罚（最高2.25×）远大于正代谢的收益（最低0.5×）——强力基因的代谢代价是显著的。

**复杂度限制**：基础最大复杂度为6，Pawn的基因总复杂度不能超过此值。复杂度越高，组装时间越长。

**设计启示**：这套三维约束体系是RimWorld平衡强力效果的核心手段——任何强力基因都必须在代谢效率或复杂度上付出代价，玩家需要在效果和代价之间做出权衡。

📖 `代谢效率与Biostat系统`

## 第4章 装备与物品 ★★★

### 4.1 装备分类与穿戴规则

RimWorld装备分为三大类：

| 大类 | 数量限制 | 核心特征 |
|------|---------|---------|
| **武器** | 仅1把主武器 | 近战/远程二分，心灵武器可绑定 |

| **腰带工具** | 多件 | 特殊层级，不与其他层冲突 |

**6层服装层级系统**：

| # | 层级 | 说明 | 典型实例 |
|---|------|------|---------|
| 1 | 贴身层 | 最内层 | T恤、裤子、内衣 |
| 2 | 中间层 | 中层 | 衬衫、防弹衣、连衣裙 |

**护甲逐层判定**：伤害到达Pawn时，按从外到内的顺序逐层检查服装护甲。每层服装都有机会减免伤害，外层服装先判定。

### 4.5 装备对Pawn属性的影响方式

| 方式 | 说明 | 典型实例 |
|------|------|---------|
| **穿戴加成** | 穿戴时直接加值 | 防弹衣：护甲+X |
| **倍率修正** | 穿戴时按倍率修改 | 某些装备：移动速度×0.9 |

## 第6章 战斗系统 ★★★

> **模组需求**：R11战斗体激活、R16伤害隔离、R18紧急脱离

### 6.1 一次攻击的完整过程

一次攻击从发起到最终效果，经历以下步骤（纯游戏规则，不涉及代码管线）：

```
攻击发起 → 命中判定 → 伤害创建 → 护盾拦截 → 护甲减免 → 伤害施加 → 后续效果
```

**详细流程**：

1. **攻击发起**：Pawn使用武器发起攻击（近战直接判定，远程生成投射物飞行）
2. **命中判定**：
   - 远程：基于射击精度、距离、掩体等计算命中概率
   - 近战：基于近战技能、闪避能力判定命中/闪避
3. **伤害创建**：根据武器属性创建伤害信息（伤害类型、伤害量、穿甲值）
4. **护盾拦截**：
   - 护盾腰带：拦截远程和爆炸伤害（近战穿透）
   - 投射物拦截器：拦截飞行中的投射物
5. **护甲减免**：按从外到内逐层检查服装护甲，每层有概率完全吸收或部分减免
6. **伤害施加**：
   - 选择命中部位（按覆盖率加权随机）
   - 根据伤害类型决定传播方式（利器集中/钝器传播/刺击穿透）
   - 创建伤口，检查即死保护
7. **后续效果**：
   - 出血开始
   - 检查死亡/倒地
   - 附加特殊效果（如火焰附加燃烧）
   - 通知各系统（仇恨、派系关系等）

📖 `战斗伤害管线完整流程`

### 6.2 伤害类型与效果差异

RimWorld的伤害分为四大类，体验差异显著：

| 大类 | 典型武器 | 伤害特征 | 护甲交互 |
|------|---------|---------|---------|
| **利器** | 刀剑、子弹 | 集中伤害，造成出血 | 利器护甲减免 |
| **钝器** | 锤棒、棍棒 | 向内传播，可伤内脏 | 钝器护甲减免 |
| **热能** | 火焰、激光 | 只伤外部，可点燃 | 热能护甲减免 |
| **无护甲** | 毒素、心灵 | 无视护甲直接生效 | 不可减免 |

**伤害传播方式差异**：
- 利器（切割/刺击）：命中一个部位，伤害集中在该部位
- 钝器：伤害从命中部位向内传播到子部位（如打头→伤脑）
- 抓挠：浅层伤害，不穿透到内部
- 火焰：只伤害外部部位，不传播到内部

📖 `伤害类型分类与DamageDef体系`

### 6.3 六种伤害减免手段

玩家可以通过六种手段减少Pawn受到的伤害：

| # | 手段 | 适用场景 | 战术意义 |
|---|------|---------|---------|
| 1 | **未命中** | 远程攻击 | 距离越远、精度越低，未命中概率越高 |
| 2 | **闪避** | 近战攻击 | 高近战技能的Pawn可以闪避近战攻击 |
| 3 | **掩体** | 远程攻击 | 站在掩体后方降低被命中概率 |
| 4 | **护盾** | 远程/爆炸 | 护盾腰带完全吸收远程伤害（近战穿透） |
| 5 | **伤害因子** | 所有伤害 | 基因/健康状况提供的伤害倍率减免 |
| 6 | **护甲** | 所有物理伤害 | 服装护甲逐层判定，可完全吸收或部分减免 |

**护盾的两种模式**：
- **护盾腰带**（原版核心）：拦截远程和爆炸伤害，近战可穿透，有能量条
- **投射物拦截器**：拦截飞行中的投射物，有拦截半径，可设置拦截条件

📖 `伤害减免机制总览`、`护盾机制与伤害吸收系统`

### 6.4 战斗行为与玩家命令

**征召系统**：Pawn需要被征召后才能执行战斗命令。征召状态下Pawn停止日常工作，服从玩家的战斗指令。

**可下达的命令**：
- 移动到指定位置
- 攻击指定目标
- 使用能力
- 进入/离开掩体

**自动射击规则**：征召状态下，Pawn会自动攻击射程内的敌对目标。AI根据距离、威胁程度、武器类型选择目标。

**AI决策逻辑**：非玩家控制的Pawn（敌人、动物）由AI控制战斗行为，包括目标选择、走位、撤退等。

📖 `战斗行为与玩家命令系统`

### 6.5 战斗安全网与极端情况

RimWorld设计了多重安全网防止战斗中的不合理死亡：

| 机制 | 说明 | 适用对象 |
|------|------|---------|
| **即死保护** | 单次伤害不会直接摧毁未受损的重要部位 | 所有Pawn |
| **倒地替代死亡** | 大多数情况下Pawn先倒地而非直接死亡 | 所有Pawn |
| **倒地即死概率** | 倒地时有概率直接死亡，概率因阵营不同 | 主要影响敌对Pawn |
| **死亡豁免** | 特殊状态完全阻止死亡判定 | 拥有特定状态的Pawn |
| **死眠替代** | 本应死亡时进入死眠 | 拥有死眠基因的Pawn |

📖 `战斗极端情况与安全网机制`

### 6.6 Pawn形态与状态切换先例

RimWorld中已有多个"Pawn形态/状态切换"的先例，对RimWT战斗体系统有直接参考价值：

| 先例 | DLC | 切换方式 | 可逆性 | 玩家体验 |
|------|-----|---------|--------|---------|
| **变异体转化** | Anomaly | 完整的形态转化系统 | 部分可逆 | Pawn外观/行为/能力全面改变 |
| **非人化(蹒跚者)** | Anomaly | 健康状况驱动 | 不可逆 | Pawn失去人性，变为敌对 |
| **死眠** | Biotech | 健康状况驱动 | 可逆（自动苏醒） | Pawn进入长时间休眠 |

**变异体转化**是最完整的先例——它有专门的转化管理器，处理外观切换、行为改变、能力赋予/移除等全套流程。

## 第7章 派系系统 ★★☆

> **模组需求**：BORDER(实体12)、敌对惑星国家派系(实体13)

### 7.1 派系类型与关系

RimWorld的派系系统定义了游戏世界中的各方势力：

**派系类型**：

| 类型 | 关系 | 特征 | 典型实例 |
|------|------|------|---------|
| 玩家派系 | — | 玩家控制的殖民地 | 殖民地 |
| 友好派系 | 正面 | 可贸易、可求援 | 友好部落、友好离群者 |
| 中立派系 | 中性 | 可贸易，关系可变 | 商人联盟 |
| 敌对派系 | 负面 | 发动袭击 | 海盗、虫族、机械族 |
| 特殊派系 | 固定 | 关系不可改变 | 远古人、虫族 |

**关系变化**：
- 关系值范围：-100（死敌）到 +100（盟友）
- 贸易、赠送礼物、释放俘虏可提升关系
- 攻击、杀害、拒绝请求会降低关系
- 关系达到阈值时改变派系态度（敌对↔中立↔友好）

**科技水平**：每个派系有科技水平设定（部落/工业/太空等），影响其成员的装备和能力。

📖 `敌对派系与玩家Pawn系统差异`

### 7.2 敌对派系Pawn与玩家Pawn的体验差异

敌对派系的Pawn在多个维度上与玩家Pawn有显著差异：

| 维度 | 玩家Pawn | 敌对派系Pawn |
|------|---------|-------------|
| **倒地即死** | 通常不会 | 较高概率直接死亡 |
| **装备掉落** | 死亡时100%掉落 | 按概率掉落（减少战利品通胀） |
| **AI行为** | 玩家控制 | AI自动决策（攻击/撤退/逃跑） |
| **俘虏规则** | 不适用 | 倒地后可俘虏、审讯、招募 |
| **需求系统** | 完整需求 | 简化需求（不在地图上时冻结） |
| **社交系统** | 完整社交 | 简化或无社交 |

📖 `敌对派系与玩家Pawn系统差异`

## 第8章 事件与袭击 ★★☆

> **模组需求**：门/Gate(实体14)、R33门生成袭击、R35死亡事件生成黑触发器

### 8.1 游戏事件的种类

RimWorld的事件系统是游戏节奏的核心驱动力：

| 大类 | 说明 | 典型实例 |
|------|------|---------|
| **威胁事件** | 对殖民地构成威胁 | 袭击、虫巢、机械集群、毒性尘埃 |
| **贸易事件** | 贸易机会 | 商队到访、轨道商船 |
| **疾病事件** | 疾病爆发 | 流感、瘟疫、肌肉寄生虫 |
| **任务事件** | 可选任务 | 救援请求、贸易请求、难民 |
| **特殊事件** | 其他 | 太阳耀斑、火山冬天、心灵船 |

**触发频率和节奏感**：事件不是随机触发的，而是由"叙事者"根据殖民地状态、时间节奏、难度设置来调度，确保游戏节奏有张有弛。

📖 `事件系统大类与触发架构`

### 8.2 袭击的完整流程

一次袭击从触发到结束的完整流程：

```
叙事者决定触发袭击
  → 选择袭击派系（根据关系和威胁点数）
  → 选择袭击策略（10种策略之一）
  → 选择到达方式（13种方式之一）
  → 生成敌人（根据威胁点数和派系科技水平）
  → 敌人到达地图
  → 战斗进行
  → 结束条件满足（敌人全灭/撤退/逃跑）
  → 战后处理（战利品、俘虏、关系变化）
```

📖 `袭击流程与结束条件`

### 8.3 袭击策略与到达方式

**10种袭击策略**（决定敌人的战术行为）：

| 策略 | 行为 | 体验 |
|------|------|------|
| 直接攻击 | 直奔殖民地 | 最常见，正面对抗 |
| 围攻 | 在远处建立营地，用迫击炮轰击 | 需要主动出击 |
| 突袭 | 快速冲锋 | 高速度低防御 |
| 偷袭 | 从多个方向同时进攻 | 需要全方位防御 |
| 等等... | | |

**13种到达方式**（决定敌人如何出现在地图上）：

| 方式 | 说明 | 体验 |
|------|------|------|
| 边缘步行 | 从地图边缘走入 | 最常见，有预警时间 |
| 空投 | 直接降落在殖民地内 | 无预警，需要内部防御 |
| 隧道 | 从地下挖掘进入 | 虫族特色 |
| 等等... | | |

📖 `袭击策略与到达模式`

### 8.4 叙事者与难度调节

**三大叙事者**的节奏差异：

| 叙事者 | 节奏风格 | 特点 |
|--------|---------|------|
| Cassandra | 稳步升级 | 威胁逐渐增强，节奏可预测 |
| Randy | 完全随机 | 可能连续灾难也可能长期平静 |
| Phoebe | 宽松间隔 | 事件间隔长，给玩家充足准备时间 |

**威胁点数增长**：随着殖民地财富和人口增长，袭击的威胁点数也会增长，生成更多/更强的敌人。

**适应系统**：游戏会根据殖民地的实际战斗力调整威胁强度，避免过于碾压或过于困难。

📖 `事件触发机制与叙事者系统`

### 8.5 死亡事件的后续效果

Pawn死亡后会触发一系列后续效果：

| 效果 | 说明 | 持续时间 |
|------|------|---------|
| **心情惩罚** | 殖民者死亡给其他殖民者带来悲伤 | 数天到数十天 |
| **关系影响** | 杀死某派系成员影响派系关系 | 永久 |

| **社交影响** | 亲属/朋友/恋人的特殊悲伤 | 更长持续时间 |

## 第10章 建筑与工作台 ★☆☆

> **模组需求**：制造设施(实体10)、母触发器(实体11)

### 10.1 工作台与配方系统

工作台是RimWorld生产系统的核心：

**工作流程**：
1. 玩家在工作台上添加生产订单（配方）
2. 配方定义所需材料、技能要求、工作量
3. 有合适技能的Pawn前往工作台执行
4. 消耗材料+工作时间→产出成品
5. 成品品质由Pawn技能和灵感决定

**配方系统要素**：

| 要素 | 说明 |
|------|------|
| 材料需求 | 指定材料类型和数量 |
| 技能需求 | 最低技能等级要求 |
| 工作量 | 完成所需的工作tick数 |
| 产出 | 成品物品（可能有品质） |
| 副产品 | 可能产生的副产品 |
| 解锁条件 | 可能需要科研解锁 |

### 10.2 建筑类型与供电

**主要建筑类型**：

| 类型 | 说明 | 典型实例 |
|------|------|---------|
| 生产建筑 | 工作台类 | 裁缝台、锻造台、药物合成台 |
| 存储建筑 | 存放物品 | 货架、冰箱 |
| 防御建筑 | 战斗用途 | 沙袋、炮塔、陷阱 |
| 基础设施 | 供电/温控 | 发电机、空调、电线 |
| 装饰建筑 | 美观/娱乐 | 雕塑、电视、台球桌 |

**供电规则**：大多数高级建筑需要电力供应。电力通过发电机产生，经电线传输到建筑。断电时建筑停止工作。
